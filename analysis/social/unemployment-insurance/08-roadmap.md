# Unemployment Insurance: Roadmap

## Strategic Overview

Reforming unemployment insurance requires action at both federal and state levels. The federal-state structure means progress can occur on parallel tracks: federal legislation to set minimum standards and state action to improve programs within existing authority. The pandemic exposed system failures, creating a window for reform that must be seized before memory fades.

## Phase 1: Foundation Building (Years 1-2)

### Federal Actions

**Administrative Improvements**:

- Increase DOL administrative grants to states
- Issue guidance on alternative base periods
- Strengthen conformity requirement enforcement
- Technical assistance for IT modernization

**Legislative Groundwork**:

- Introduce comprehensive UI modernization bill
- Hold hearings on pandemic failures
- Document racial disparities in access
- Build congressional support

**Regulatory Action**:

- Clarify worker classification standards
- Strengthen anti-fraud measures
- Improve data collection requirements
- Issue model state legislation guidance

### State Actions

**Progressive States Lead**:

- Increase benefit levels toward 50% replacement
- Eliminate waiting weeks
- Adopt alternative base periods
- Extend duration to 26+ weeks

**Administrative Improvements**:

- Begin IT system modernization
- Improve claims processing times
- Reduce improper denials
- Expand outreach to underserved populations

**Coverage Expansion**:

- Adopt ABC test for worker classification
- Expand partial UI for reduced hours
- Lower monetary eligibility thresholds
- Cover part-time workers seeking part-time work

### Advocacy and Organizing

**Build Coalition**:

- Unite labor, civil rights, and anti-poverty organizations
- Organize gig workers and precarious workers
- Document and publicize system failures
- Develop shared policy platform

**Research and Documentation**:

- Publish analysis of pandemic response
- Document racial disparities by state
- Compare international best practices
- Model reform impacts

## Phase 2: Major State Reforms (Years 2-4)

### Progressive State Achievements

**California, New York, Washington, New Jersey, Massachusetts**:

- Implement comprehensive reforms
- Demonstrate feasibility of higher standards
- Create models for other states
- Document positive outcomes

**Key Reforms to Achieve**:

| Reform | Target States |
|--------|---------------|
| 50% wage replacement | All progressive states |
| Gig worker coverage | States with ABC test |
| 26+ weeks duration | All states below standard |
| No waiting week | States still requiring |
| Alternative base period | All states |

### Swing State Progress

**Target States**: Pennsylvania, Michigan, Wisconsin, Nevada, Arizona

**Strategy**:

- Coalition building with labor and community groups
- Ballot initiatives where available
- Legislative advocacy
- Governor and agency pressure

### Documentation and Learning

**Evaluate Early Adopters**:

- Track recipiency rate changes
- Measure benefit adequacy impact
- Assess trust fund sustainability
- Document worker outcomes

**Share Best Practices**:

- Cross-state learning networks
- Technical assistance from successful states
- Policy briefs and reports
- Legislative drafting support

## Phase 3: Federal Legislation (Years 3-5)

### UI Modernization Act

**Core Provisions**:

- Minimum 50% wage replacement
- 26-week minimum duration
- Alternative base period required
- No waiting weeks
- Cover all workers with earnings
- FUTA wage base increase to $50,000

**Legislative Strategy**:

- Build on state successes
- Coalition pressure campaign
- Targeted advocacy to swing senators
- Incremental if necessary

### Complementary Legislation

**Worker Classification**:

- Federal ABC test
- Presumption of employee status
- Enforcement resources
- Portable benefits for exempted workers

**Automatic Stabilizers**:

- Extended Benefits trigger reform
- Automatic benefit supplements during recessions
- Federal reinsurance for catastrophic events

### Budget and Financing

**FUTA Reform**:

- Increase taxable wage base
- Index to wage growth
- Modest rate increase
- Enhanced administrative funding

**State Incentives**:

- Matching grants for benefit increases
- Bonus payments for coverage expansion
- Technical assistance funding
- IT modernization grants

## Phase 4: Implementation and Expansion (Years 5-7)

### Full Federal Implementation

**Regulatory Development**:

- Detailed guidance on new standards
- State plan approval process
- Enforcement mechanisms
- Technical assistance

**State Compliance**:

- Deadline for plan submissions
- Phase-in period for major changes
- Funding to support transitions
- Monitoring and enforcement

### Remaining State Reform

**Lagging States**:

- Federal standards force minimum improvements
- Continued advocacy for going beyond minimums
- Technical assistance for implementation
- Accountability for disparities

### System Improvements

**Technology Modernization**:

- All states with modern IT systems
- Real-time eligibility determination
- Improved fraud detection
- Better user experience

**Administrative Capacity**:

- Adequate staffing levels
- Training and professional development
- Quality assurance systems
- Performance management

## Phase 5: Consolidation and Enhancement (Years 7-10)

### Evaluate and Adjust

**Assess Reforms**:

- Measure recipiency rate improvements
- Evaluate benefit adequacy
- Assess racial equity gains
- Review trust fund sustainability

**Make Adjustments**:

- Address implementation problems
- Close loopholes
- Increase standards where warranted
- Improve financing

### Further Enhancements

**Coverage Expansion**:

- Self-employment coverage options
- Multi-state worker simplification
- Broader "good cause" definitions
- New entrant coverage improvements

**Benefit Improvements**:

- Increase replacement rate targets
- Extend maximum duration
- Add dependent allowances
- Improve partial UI

### Long-Term Sustainability

**Trust Fund Health**:

- Adequate forward funding
- Automatic adjustments
- Federal backstop for catastrophic events
- Sustainable tax base

**Political Durability**:

- Build public understanding and support
- Document positive outcomes
- Maintain coalition engagement
- Resist rollback attempts

## Obstacles and Contingencies

### Political Obstacles

**Congressional Gridlock**:

- *Contingency*: Focus on state action; administrative improvements
- *Alternative*: Partial legislation; budget reconciliation

**State Resistance**:

- *Contingency*: Federal standards with enforcement
- *Alternative*: Incentive-based approach

**Employer Opposition**:

- *Contingency*: Coalition with supportive businesses
- *Alternative*: Phased implementation; small business exemptions

### Implementation Obstacles

**IT System Challenges**:

- *Contingency*: Extended timelines; shared services
- *Alternative*: Federal IT support; off-the-shelf solutions

**Administrative Capacity**:

- *Contingency*: Increased federal funding; technical assistance
- *Alternative*: Phased implementation; contractor support

**Trust Fund Concerns**:

- *Contingency*: Federal support during transition
- *Alternative*: Longer phase-in; experience rating adjustments

### Economic Obstacles

**Recession During Implementation**:

- *Contingency*: Emergency federal funding
- *Alternative*: Accelerate reforms during crisis

**Employer Tax Concerns**:

- *Contingency*: Phased tax increases; small business relief
- *Alternative*: Alternative financing mechanisms
