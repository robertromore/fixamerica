# Supplemental Security Income (SSI): Actions

## Overview

This guide provides concrete actions for individuals, advocates, and organizations working to improve SSI. Actions range from personal steps SSI recipients can take to protect their benefits, to advocacy efforts that can drive systemic change.

## Actions for SSI Recipients

### Protect Your Benefits

**Report Changes Promptly**:

| What to Report | Deadline | How |
|----------------|----------|-----|
| Earnings | Within 10 days | Online, phone, in-person |
| Address change | Within 10 days | Online, phone, in-person |
| Living arrangement change | Within 10 days | Phone, in-person |
| Asset changes | Within 10 days | Phone, in-person |
| Marriage/divorce | Within 10 days | In-person recommended |

**Keep Records**:

- Save all SSA correspondence
- Document earnings (pay stubs)
- Track medical appointments
- Keep copies of everything you submit
- Note dates of phone calls (who, when, what said)

**Respond to SSA Requests**:

- Meet all deadlines
- Request extensions if needed (in writing)
- Get receipts for documents submitted
- Follow up if no response

### Navigate Work While on SSI

**Know Your Work Incentives**:

| Incentive | What It Does | How to Use |
|-----------|--------------|------------|
| Earned income exclusion | First $65/month not counted | Automatic |
| 1-for-2 reduction | Keep $1 for every $2 earned | Automatic |
| Section 1619(a) | Continue SSI while working | Automatic if eligible |
| Section 1619(b) | Keep Medicaid after SSI ends | Must meet threshold |
| PASS | Save for work goal | Apply through SSA |
| IRWE | Deduct disability work expenses | Document and report |

**Get Help**:

- Contact Work Incentives Planning and Assistance (WIPA): 1-866-968-7842
- Use Ticket to Work: choosework.ssa.gov
- Consult benefits counselor before major work changes

**Protect Medicaid**:

- Section 1619(b) allows Medicaid continuation
- Report earnings accurately to maintain protection
- Understand your state's threshold

### Appeal Denials

**Know the Timeline**:

| Stage | Deadline | Action |
|-------|----------|--------|
| Initial denial | 60 days | Request reconsideration |
| Reconsideration denial | 60 days | Request ALJ hearing |
| ALJ denial | 60 days | Request Appeals Council review |
| Appeals Council denial | 60 days | File federal court appeal |

**Get Representation**:

- Legal aid (free if income-eligible)
- NOSSCR attorney directory
- P&A agency in your state
- Representatives can be paid from past-due benefits

**Prepare Strong Appeals**:

- Gather additional medical evidence
- Get supporting statements from doctors
- Document functional limitations
- Keep treatment records current

### Use ABLE Accounts

**Benefits of ABLE Accounts**:

- Save up to $100,000 without affecting SSI eligibility
- Tax-advantaged growth
- Use for disability-related expenses
- Available in most states

**Eligible Expenses**:

| Category | Examples |
|----------|----------|
| Education | Tuition, books, supplies |
| Housing | Rent, mortgage, utilities |
| Transportation | Vehicle purchase, maintenance, transit |
| Healthcare | Insurance, therapies, equipment |
| Technology | Computers, assistive devices |
| Personal support | Personal care attendants |

**How to Open**:

- Visit your state's ABLE program website
- Can open in any state's program
- Compare fees and investment options
- ablenrc.org has state comparisons

## Actions for Family Members

### Support Application Process

**Gather Documentation**:

| Type | Examples |
|------|----------|
| Medical records | Hospitalizations, treatments |
| Doctor statements | Functional limitations |
| School records | IEPs, evaluations (for children) |
| Work history | For adult disability |
| Financial records | Income, assets |

**Attend Appointments**:

- Consultative exams
- Field office visits
- ALJ hearings (as witness)
- Provide third-party statements

### Become a Representative Payee

**Responsibilities**:

| Duty | Requirement |
|------|-------------|
| Use benefits for recipient | Housing, food, medical care |
| Save excess for recipient | Maintain records |
| Report changes to SSA | Timely reporting |
| Complete annual accounting | Form SSA-6230 |
| Keep records | Save for SSA review |

**How to Apply**:

1. Visit local SSA office
2. Complete Form SSA-11
3. Provide identification
4. Pass background check
5. Receive training materials

## Legislative Advocacy

### Contact Elected Officials

**Members of Congress**:

| Action | Method |
|--------|--------|
| Constituent meetings | Schedule through district office |
| Phone calls | Capitol switchboard: 202-224-3121 |
| Letters/emails | Via official websites |
| Town halls | Attend and ask questions |
| Social media | Public pressure |

**Key Messages**:

| Issue | Message |
|-------|---------|
| Asset limits | "Frozen since 1989, trapping people in poverty" |
| Benefits | "25% below poverty line is inadequate" |
| Marriage penalty | "Couples lose $5,600/year for marrying" |
| Processing delays | "10,000 people die waiting each year" |

**Specific Asks**:

1. Co-sponsor SSI Restoration Act
2. Support SSA administrative funding
3. Hold hearings on SSI reform
4. Include SSI in budget negotiations

### Support Legislation

**Current Bills to Support**:

| Bill | Key Provisions |
|------|----------------|
| SSI Restoration Act | Asset limits, benefits, marriage penalty |
| SSI Savings Penalty Elimination Act | Raise asset limits |
| RISE Act | Restore benefits for certain non-citizens |
| Benefits cliff elimination proposals | Work incentive improvements |

**Actions**:

- Write letters of support
- Organize sign-on letters
- Generate constituent calls
- Testify at hearings
- Provide expert analysis

### State-Level Action
