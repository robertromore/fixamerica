# Open Internet: Actions for Citizens

## Overview

Citizens can take meaningful action to protect the open internet through advocacy, consumer choices, community engagement, and political participation.

## Individual Actions

### Stay Informed

| Action | Resources |
|--------|-----------|
| Follow policy news | EFF, Public Knowledge |
| Understand your rights | Consumer guides |
| Track legislation | Congress.gov |
| Monitor FCC actions | FCC website |

### Test Your Connection

| Action | Tool |
|--------|------|
| Speed tests | Speedtest.net, M-Lab |
| Throttling detection | Wehe app |
| Outage reporting | Downdetector |
| ISP transparency | Check ISP disclosures |

### Document Issues

| Issue | Action |
|-------|--------|
| Throttling | Record evidence, file complaint |
| Blocking | Document, report |
| Speed discrepancy | Test, compare to advertised |
| Hidden fees | Save bills, complain |

## Consumer Choices

### Evaluate Providers

| Factor | Consider |
|--------|----------|
| Competition | Choose independent if available |
| Practices | Research ISP policies |
| Transparency | Prefer clear pricing |
| Community networks | Support municipal options |

### Service Decisions

| Decision | Consideration |
|----------|---------------|
| Bundling | Avoid if locks you in |
| Contracts | Prefer month-to-month |
| Data caps | Factor into choice |
| Equipment | Consider owning vs. renting |

### Privacy Protection

| Action | Benefit |
|--------|---------|
| VPN use | Protect traffic from ISP |
| HTTPS | Encrypted connections |
| DNS settings | Alternative DNS providers |
| Browser settings | Block tracking |
