# Open Internet: History

## Overview

The internet's evolution from a research network to essential infrastructure has driven ongoing debates about openness, access, and control. U.S. policy has oscillated between treating the internet as an unregulated information service and recognizing it as essential communications infrastructure requiring common carrier regulation.

## Key Legislative History

### Telecommunications Act of 1996

| Element | Impact |
|---------|--------|
| **Title I (Information services)** | Light regulation |
| **Title II (Common carriers)** | Heavy regulation |
| **Section 230** | Platform immunity |
| **Goal** | Promote competition |
| **Result** | Classification debates |

### Section 230

| Element | Description |
|---------|-------------|
| **Immunity** | Platforms not liable for user content |
| **Good Samaritan** | Protects content moderation |
| **Purpose** | Encourage moderation, enable internet growth |
| **Criticism** | Shields platforms from accountability |

## Regulatory History

### FCC Approaches

| Era | Approach |
|-----|----------|
| **1990s-2002** | Light touch, information services |
| **2002-2010** | Information service classification |
| **2010-2014** | Open Internet Order (weak authority) |
| **2015-2017** | Title II, strong rules |
| **2017-2024** | Repeal, information service |

## Major Court Decisions

### *Comcast v. FCC* (2010)

| Element | Description |
|---------|-------------|
| **Issue** | FCC authority over Comcast throttling |
| **Holding** | FCC lacked authority under Title I |
| **Impact** | Forced FCC to consider Title II |

### *Verizon v. FCC* (2014)

| Element | Description |
|---------|-------------|
| **Issue** | 2010 Open Internet Order validity |
| **Holding** | Can't impose common carrier rules on information services |
| **Impact** | Led to Title II reclassification |

### *USTA v. FCC* (2016)

| Element | Description |
|---------|-------------|
| **Issue** | 2015 Title II order validity |
| **Holding** | Upheld FCC authority |
| **Impact** | Net neutrality rules survived |

### *Mozilla v. FCC* (2019)

| Element | Description |
|---------|-------------|
| **Issue** | 2017 repeal validity |
| **Holding** | Upheld repeal, allowed state laws |
| **Impact** | Opened door for state action |

## Lessons from History

| Lesson | Description |
|--------|-------------|
| **Classification matters** | Title I vs. II drives authority |

| **Politics shift** | Partisan positions evolve |
