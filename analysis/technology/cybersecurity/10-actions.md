# Cybersecurity: Actions for Citizens

## Overview

Citizens can contribute to improved cybersecurity through personal security practices, civic engagement, career choices, and support for policy reform.

## Personal Security

### Basic Hygiene

| Action | Why It Matters |
|--------|----------------|
| Use password manager | Unique, strong passwords |
| Enable MFA | Prevents account takeover |
| Update software | Patches vulnerabilities |
| Backup data | Ransomware resilience |
| Be skeptical | Phishing prevention |

### Advanced Measures

| Action | Why It Matters |
|--------|----------------|
| Use encrypted messaging | Protect communications |
| Review privacy settings | Limit data exposure |
| Monitor accounts | Early detection |
| Secure home network | Protect all devices |
| Consider VPN | Protect traffic |

### Family Security

| Action | Why It Matters |
|--------|----------------|
| Educate children | Age-appropriate awareness |
| Protect elderly relatives | Prevent scams |
| Family security plan | Incident response |
| Secure shared devices | Prevent compromise |

## Report Security Issues

### When to Report

| Situation | Report To |
|-----------|-----------|
| Personal cybercrime | FBI IC3 |
| Data breach | FTC, state AG |
| Critical infrastructure | CISA |
| National security | FBI |
| Vulnerabilities | Vendor, CISA |

### How to Report

| Channel | Use For |
|---------|---------|
| IC3.gov | Internet crime |
| ReportFraud.ftc.gov | Fraud, scams |
| CISA.gov | Infrastructure |
| Local FBI | Major incidents |

## Quick-Start Actions

### This Week

1. Enable MFA on critical accounts
2. Install a password manager
3. Update all devices
4. Sign up for CISA alerts

### This Month

1. Review and improve passwords
2. Check privacy settings
3. Back up important data
4. Contact representative about cyber legislation

### This Year

1. Take a security course
2. Get involved in local advocacy
3. Support reform organizations
4. Consider security career path
