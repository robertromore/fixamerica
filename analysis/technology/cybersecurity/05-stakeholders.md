# Cybersecurity: Stakeholders

## Overview

Cybersecurity involves diverse stakeholders with varying interests, capabilities, and perspectives on appropriate policy responses.

## Government Stakeholders

### Federal Agencies

| Agency | Role | Resources |
|--------|------|-----------|
| **CISA** | Civilian coordinator | ~$2.9B budget |
| **NSA/Cyber Command** | Military/intel | Classified |
| **FBI** | Law enforcement | Significant |
| **Secret Service** | Financial crimes | Moderate |
| **DOE** | Energy sector | Growing |
| **HHS** | Healthcare | Limited |

### Congressional Actors

## Stakeholder Positions

### On Regulation

| Stakeholder | Position |
|-------------|----------|
| Government (CISA) | Supports increased requirements |
| Critical infrastructure | Generally opposes mandates |
| Security vendors | Supports (market opportunity) |
| Tech industry | Mixed, prefers flexibility |
| Advocates | Supports strong requirements |

### On Liability

| Stakeholder | Position |
|-------------|----------|
| Software vendors | Strongly oppose |
| Government | Supports some liability |
| Victims | Support liability |
| Insurance | Mixed |
| Advocates | Support liability |
