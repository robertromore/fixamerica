# Digital Divide: Current State

## Overview

Despite recent federal investments, significant gaps in broadband access, affordability, and adoption persist, with rural, low-income, and minority communities disproportionately affected.

## Access Gaps

### Geographic Disparities

| Area | Broadband Access (25/3 Mbps) |
|------|------------------------------|
| Urban | 98%+ |
| Suburban | 95%+ |
| Rural | 83% |
| Tribal | 67% |
| US Territories | Variable, often poor |

### Speed Tiers

| Speed | Availability |
|-------|--------------|
| 25/3 Mbps (FCC minimum) | 95%+ |
| 100/20 Mbps | 89% |
| 250/25 Mbps | 84% |
| 1 Gbps | 55% |
| Fiber-to-home | 43% |

### Technology Types

| Technology | Coverage | Limitations |
|------------|----------|-------------|
| Fiber | 43% | Best option, limited deployment |
| Cable | 89% | Asymmetric, data caps |
| DSL | Declining | Obsolete speeds |
| Fixed wireless | Growing | Weather-dependent |
| Satellite | Universal | Latency, data limits |
| 5G | Growing urban | Limited rural |
