# Data Privacy: Citizen Actions

## Overview

Individuals can take meaningful action to protect their own privacy and advocate for systemic change. This guide provides actionable steps for people at all engagement levels.

## Immediate Actions

### Protect Your Own Privacy

| Action | Time | Impact |
|--------|------|--------|
| Review app permissions | 15 min | Reduce collection |
| Adjust privacy settings | 30 min | Limit sharing |
| Use privacy browser | 5 min | Reduce tracking |
| Install tracker blocker | 5 min | Block surveillance |
| Review and delete accounts | 1 hour | Reduce exposure |

**Quick settings changes:**
- Disable location for apps that don't need it
- Turn off ad personalization
- Limit app background refresh
- Review connected accounts

### Opt Out of Data Collection

| Service | How |
|---------|-----|
| **Google** | myaccount.google.com/data-and-privacy |
| **Facebook/Meta** | Settings > Privacy |
| **Apple** | Settings > Privacy & Security |
| **Credit offers** | optoutprescreen.com |
| **Direct mail** | dmachoice.org |
| **Data brokers** | Individual broker sites |

### Exercise Your Rights

**If you're in a state with privacy law (CA, CO, VA, CT, etc.):**

| Right | How to Exercise |
|-------|-----------------|
| **Access** | Submit data access request |
| **Deletion** | Submit deletion request |
| **Opt-out of sale** | Use "Do Not Sell" links |
| **Correction** | Submit correction request |

**Process:**
1. Find company's privacy page
2. Locate rights submission form
3. Submit request with required information
4. Follow up if no response in 45 days
5. File complaint with AG if denied

## Advocacy Actions

### Contact Elected Officials

| Action | When | Template Topics |
|--------|------|-----------------|
| Call your Senators | Ongoing | Support privacy legislation |
| Email Representative | Before key votes | Specific bill support |
| Attend town halls | When available | Raise privacy concerns |
| Submit written comments | Rulemaking periods | Technical input |

**Sample call script:**
> "I'm calling to urge [Senator/Representative] to support comprehensive federal privacy legislation. Companies collect vast amounts of personal data about Americans with little oversight. I support legislation that gives individuals control over their data, requires companies to minimize collection, and provides strong enforcement. Please support the [specific bill] and oppose industry attempts to weaken privacy protections. This matters to me because [personal reason]."

### Comment on Regulations

| Agency | Topic | Where |
|--------|-------|-------|
| FTC | Commercial surveillance | regulations.gov |
| FTC | Data broker rules | regulations.gov |
| CFPB | Financial data | regulations.gov |
| State AGs | State implementation | State websites |

**Effective comment elements:**
1. Personal experience with privacy harms
2. Specific concerns about current practices
3. Support for specific provisions

| Criterion | What to Look For |
|-----------|------------------|

## Action by Issue

### If Concerned About Data Brokers

1. Opt out of major brokers
2. Support broker registration legislation
3. Contact legislators
4. Report violations

### If Concerned About Tracking

1. Install tracker blockers
2. Adjust app permissions
3. Support consent legislation
4. Use privacy alternatives

### If Concerned About Children

1. Review children's app privacy
2. Support expanded COPPA
3. Engage with schools on ed tech
4. Contact legislators

### If Concerned About Workplace Privacy

1. Review workplace policies
2. Advocate for limits
3. Support worker privacy legislation
4. Connect with labor organizations
