## Opposition Arguments

### Public Safety Claims

| Argument | Counter |
|----------|---------|
| **"Catches criminals"** | Documented errors, limited evidence |
| **"Solves crimes faster"** | Alternatives exist |
| **"Deters crime"** | No evidence of deterrence |
| **"Finds missing persons"** | Narrow use doesn't justify mass surveillance |
| **"Prevents terrorism"** | Overstated, other methods work |

### Necessity Arguments

| Argument | Counter |
|----------|---------|
| **"Essential tool"** | Policed without it for centuries |
| **"Modern criminals require"** | Other investigative methods |
| **"Can't go back"** | Choice, not inevitability |
| **"Other countries use"** | Authoritarian regimes |

### Technical Arguments

| Argument | Counter |
|----------|---------|
| **"Accuracy improving"** | Still discriminatory |
| **"Better than humans"** | Different, not better |
| **"Just a tool"** | Tools shape outcomes |
| **"Neutral technology"** | Bias documented |

### Legal Arguments

| Argument | Counter |
|----------|---------|
| **"No expectation of privacy in public"** | Aggregation changes analysis |
| **"Existing law sufficient"** | Gaps documented |
| **"First Amendment protects"** | Commercial use different |
| **"Fourth Amendment doesn't apply"** | *Carpenter* expanded protections |

### Practical Arguments

| Argument | Counter |
|----------|---------|
| **"Already deployed"** | Can ban, regulate |
| **"Investment wasted"** | Sunk cost fallacy |
| **"Criminals will benefit"** | Balance with rights |
| **"Unenforceable"** | Other regulations work |

## Weaknesses in Opposition Arguments

### Public Safety Claims

| Weakness | Evidence |
|----------|----------|
| Wrongful arrests | Multiple documented cases |
| Discriminatory accuracy | NIST studies |
| Limited crime impact | No controlled studies |
| Chilling effects | First Amendment harm |
| Community distrust | Undermines cooperation |

### Necessity Claims

| Weakness | Evidence |
|----------|----------|
| Traditional methods work | Case clearance rates |
| Over-surveillance costs | Community relations |
| Alternatives available | Other investigative tools |
| Proportionality | Mass surveillance disproportionate |

### Technical Claims

| Weakness | Evidence |
|----------|----------|
| Persistent bias | NIST, academic studies |
| Context dependency | Performance varies |
| Adversarial vulnerability | Can be spoofed |
| Integration problems | System errors compound |

### Legal Claims

| Weakness | Evidence |
|----------|----------|
| *Carpenter* changed analysis | Location data protected |
| Mosaic theory | Aggregation matters |
| Civil rights violations | Documented discrimination |
| International trends | EU restrictions |

| "Modern policing" | "Authoritarian policing" |
