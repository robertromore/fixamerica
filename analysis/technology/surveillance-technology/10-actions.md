# Surveillance Technology: Actions for Citizens

## Overview

Citizens can combat surveillance technology through personal protective measures, community organizing, civic engagement, and support for reform organizations.

## Personal Protection

### Digital Security

| Action | Purpose |
|--------|---------|
| Use encrypted messaging (Signal) | Protect communications |
| Enable device encryption | Protect data |
| Use VPN | Obscure location/activity |
| Review privacy settings | Limit data collection |
| Use privacy browsers | Reduce tracking |

### Location Protection

| Action | Purpose |
|--------|---------|
| Disable location services when unnecessary | Reduce tracking |
| Opt out of data broker lists | Limit data availability |
| Review app permissions | Control access |
| Use cash | Avoid financial tracking |
| Be aware of cameras | Know surveillance locations |

### Biometric Protection

| Action | Purpose |
|--------|---------|
| Limit social media photos | Reduce facial data |
| Request deletion from databases | Where available |
| Know your rights at borders | Understand requirements |
| Opt out where possible | Exercise choices |

## Quick-Start Actions

### This Week

1. Review phone privacy settings
2. Install Signal for encrypted messaging
3. Look up your city's surveillance technology
4. Sign up for advocacy organization alerts

### This Month

1. Attend a city council meeting on surveillance
2. File a public records request
3. Contact your representative about surveillance
4. Join a local advocacy group

### This Year

1. Campaign for local surveillance ordinance
2. Testify at a hearing
3. Support litigation if you can
4. Build a community coalition
