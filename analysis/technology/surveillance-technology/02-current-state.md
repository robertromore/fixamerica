# Surveillance Technology: Current State

## Overview

Surveillance technology has proliferated rapidly across law enforcement, government agencies, and the private sector, with minimal legal constraints and inadequate oversight.

## Facial Recognition

### Law Enforcement Use

| Agency Type | Status |
|-------------|--------|
| Federal (FBI, ICE, CBP) | Extensive use |
| State police | Widespread |
| Local police | 3,100+ agencies |
| Schools | Growing |
| Border | Pervasive |

### Technology Landscape

| Provider | Users |
|----------|-------|
| Clearview AI | 3,100+ agencies |
| NEC | Many state DMVs |
| Idemia | Federal, state |
| Amazon Rekognition | Various (moratorium) |
| Various vendors | Fragmented market |

### Accuracy Issues

| Issue | Data |
|-------|------|
| Overall accuracy | Variable, context-dependent |
| Error rate disparities | Higher for darker skin, women |
| False positives | Documented wrongful arrests |
| Lighting, angle sensitivity | Performance degradation |

### Civil Rights Impacts

| Impact | Evidence |
|--------|----------|
| Wrongful arrests | Multiple documented cases |
| Misidentification by race | Higher rates for Black individuals |
| Protest surveillance | Documented use |
| Immigration enforcement | CBP extensive use |
