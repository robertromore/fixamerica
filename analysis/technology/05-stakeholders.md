# Technology Policy: Stakeholders

## Overview

Technology policy involves a complex ecosystem of stakeholders with varying interests, resources, and power. Understanding these relationships is essential for building reform coalitions and anticipating opposition.

## Who Benefits from the Status Quo

### Big Tech Platforms

| Company | Stakes |
|---------|--------|
| **Google/Alphabet** | Search monopoly revenue, ad dominance |
| **Meta** | User data, social graph control |
| **Amazon** | E-commerce dominance, AWS lock-in |
| **Apple** | App Store fees, ecosystem control |
| **Microsoft** | Enterprise dominance, AI integration |

### Data Industry

| Actor | Interest |
|-------|----------|
| **Data brokers** | Unregulated data trade |
| **Ad tech** | Tracking, targeting capabilities |
| **Credit bureaus** | Data collection exemptions |
| **Analytics firms** | Corporate surveillance |
| **Identity services** | Personal data aggregation |

### Telecommunications

| Actor | Interest |
|-------|----------|
| **ISPs** | No net neutrality, pricing power |
| **Telecom equipment** | Infrastructure contracts |
| **Content providers** | Platform relationships |
| **Trade associations** | Industry-wide lobbying |

### Other Beneficiaries

| Actor | Interest |
|-------|----------|
| **Investors** | Platform valuations |
| **Executive compensation** | Stock-based pay |
| **Surveillance vendors** | Government/corporate contracts |
| **Political campaigns** | Microtargeting capabilities |
| **Foreign actors** | Influence operations via platforms |

## Who Is Harmed

### Consumers

| Harm | Population |
|------|------------|
| **Privacy violations** | All internet users |
| **Data breaches** | Billions of records exposed |
| **Manipulation** | Dark patterns, algorithmic targeting |
| **Higher prices** | Reduced competition |
| **Reduced choice** | Monopolized markets |

### Workers

| Group | Harm |
|-------|------|
| **Gig workers** | Misclassification, no benefits |
| **Content moderators** | Trauma, poor conditions |
| **Warehouse workers** | Surveillance, pace pressure |
| **Tech employees** | NDAs, non-competes, layoffs |
| **Displaced workers** | Automation, AI replacement |

### Small Businesses

| Harm | Mechanism |
|------|-----------|
| **Platform dependence** | Amazon, Google control visibility |
| **Self-preferencing** | Platforms favor own products |
| **Data disadvantage** | Can't compete with platform data |
| **Commission extraction** | App store fees, payment fees |
| **Rule changes** | Algorithm shifts destroy businesses |

### Vulnerable Populations

| Population | Specific Harm |
|------------|---------------|
| **Children** | Online safety, mental health |
| **Elderly** | Scams, digital exclusion |
| **Low-income** | Digital divide, predatory targeting |
| **Minorities** | Algorithmic discrimination |
| **Immigrants** | Surveillance, ICE data access |

### Society Broadly

| Harm | Scale |
|------|-------|
| **Democracy** | Misinformation, foreign interference |
| **Public discourse** | Polarization, radicalization |
| **Local journalism** | Ad revenue collapse |
| **Competition** | Innovation suppressed |
| **Security** | Critical infrastructure vulnerabilities |

## Key Relationships

### Industry-Government

| Relationship | Dynamic |
|--------------|---------|
| **Lobbying** | $70M+ annual Big Tech spending |
| **Revolving door** | Staff movement both directions |
| **Campaign finance** | Tech a major donor |
| **Research funding** | Industry-sponsored studies |
| **Advisory roles** | Industry on government panels |

### Big Tech-Startups

| Relationship | Dynamic |
|--------------|---------|
| **Acquisition** | Potential competitors bought |
| **Investment** | Venture arms create dependencies |
| **Platform access** | APIs, visibility control |
| **Talent competition** | Hiring away employees |

### Platforms-Media

| Relationship | Dynamic |
