# Algorithmic Accountability: Stakeholders

## Overview

Algorithmic accountability involves diverse stakeholders with varying degrees of power, interest, and exposure to algorithmic systems. Understanding these stakeholders is essential for building effective reform coalitions.

## Affected Populations

### Job Seekers and Workers

| Exposure | Harm |
|----------|------|
| Resume screening | Discriminatory filtering |
| Video interviews | Facial analysis bias |
| Performance monitoring | Unfair evaluation |
| Scheduling algorithms | Unpredictable hours |
| Termination algorithms | Arbitrary firing |

**Advocacy capacity**: Low to moderate (unions, worker advocates)

### Credit and Financial Services Users

| Exposure | Harm |
|----------|------|
| Credit scoring | Discriminatory scores |
| Loan underwriting | Denial, higher rates |
| Insurance pricing | Unfair premiums |
| Banking access | Account denial |
| Fraud detection | False flags, account freezes |

**Advocacy capacity**: Moderate (consumer groups)

### Housing Seekers

| Exposure | Harm |
|----------|------|
| Tenant screening | Discriminatory rejection |
| Criminal background | Disproportionate impact |
| Rent pricing | Algorithm-driven increases |
| Ad targeting | Discriminatory steering |
| Mortgage decisions | Fair housing violations |

**Advocacy capacity**: Moderate (housing advocates)

### Healthcare Patients

| Exposure | Harm |
|----------|------|
| Diagnostic algorithms | Biased recommendations |
| Prior authorization | Automated denials |
| Risk scoring | Resource allocation bias |
| Treatment recommendations | Data-driven disparities |
| Insurance decisions | Algorithmic denial |

**Advocacy capacity**: Moderate (patient advocates)

### Criminal Justice Involved

| Exposure | Harm |
|----------|------|
| Pretrial risk assessment | Discriminatory detention |
| Sentencing algorithms | Biased recommendations |
| Parole/probation | Unfair supervision |
| Facial recognition | Misidentification |
| Predictive policing | Over-surveillance |

**Advocacy capacity**: Low (civil rights groups)

### Benefits Recipients

| Exposure | Harm |
|----------|------|
| Eligibility algorithms | Wrongful denial |
| Fraud detection | False accusations |
| Benefit calculations | Errors |
| Child welfare | Family disruption |
| Disability determination | Inconsistent decisions |

**Advocacy capacity**: Low (legal aid, advocates)

|-------|----------|
| Algorithm developers | Protect IP, limit liability |

| AI/ML vendors | Market access |
| Consulting firms | Continued demand |
