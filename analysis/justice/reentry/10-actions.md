# Reentry Reform: Actions

## Overview

This document provides specific actions individuals and communities can take to support reentry reform. Actions are organized by role and capacity, from individual support to systemic advocacy.

## Individual Actions

### Educate Yourself

| Action | Resources |
|--------|-----------|
| Learn about collateral consequences | NICCC database, Collateral Consequences Resource Center |
| Understand local policies | State policy summaries, local advocacy groups |
| Read personal narratives | Marshall Project, directly impacted voices |
| Follow reform organizations | Social media, newsletters |
| Attend public forums | Town halls, community meetings |

### Change Your Perspective

| Mindset Shift | Practice |
|---------------|----------|
| See people beyond records | Use person-first language ("person with a conviction") |
| Recognize systemic factors | Understand barriers to success |
| Challenge stereotypes | Question media portrayals |
| Embrace second chances | Advocate in personal circles |

### Support Returning Citizens Directly

| Action | How |
|--------|-----|
| Mentorship | Volunteer with reentry programs |
| Employment references | Vouch for people you know |
| Housing support | Consider renting to people with records |
| Transportation | Help with rides to appointments |
| Practical assistance | Help with paperwork, job applications |
| Social connection | Include in community activities |

### Personal Economic Power

| Action | Impact |
|--------|--------|
| Support fair chance employers | Patronize companies that hire fairly |
| Ask about hiring policies | As customer or investor |
| Support reentry enterprises | Social enterprises employing returning citizens |
| Divest from prison profiteers | Private prisons, prison contractors |

## Employer Actions

### Implement Fair Chance Hiring

| Step | Implementation |
|------|----------------|
| Remove conviction questions from applications | Ban the box internally |
| Delay background checks | After conditional offer |
| Use individualized assessment | Consider circumstances, not blanket rules |
| Train HR staff | Fair chance hiring practices |
| Create written policy | Document commitment |

### Partner with Reentry Organizations

| Partnership Type | Examples |
|------------------|----------|
| Job fairs | Host or participate in reentry job fairs |
| Pipeline programs | Work with reentry programs for referrals |
| Internships | Offer positions to returning citizens |
| Mentorship | Connect employees with program participants |

### Advocate for Industry Change

| Action | Approach |
|--------|----------|
| Join business coalitions | Second Chance Business Coalition |
| Share success stories | Document positive outcomes |
| Influence peers | Speak to other employers |
| Testify for legislation | Support fair chance laws |

### Become a Model Employer

| Practice | Description |
|----------|-------------|
| Set hiring targets | Commit to fair chance hiring goals |
| Track outcomes | Measure retention, advancement |
| Provide support | Assign mentors, allow flexibility |
| Publicize commitment | Make fair chance part of brand |

## Landlord Actions

### Adopt Fair Chance Housing Practices

| Practice | Implementation |
|----------|----------------|
| Individualized assessment | Evaluate each applicant individually |
| Limit lookback periods | Focus on recent history |
| Consider circumstances | Type of offense, time elapsed, rehabilitation |
| Delay criminal history inquiry | After initial screening |

### Reduce Risk, Not Access

| Strategy | Description |
|----------|-------------|
| Use risk mitigation funds | Programs that cover landlord losses |
| Accept guarantees | Co-signers, organizational guarantees |
| Partner with programs | Work with transitional housing providers |
| Start small | Begin with supportive housing partnerships |

## Advocacy Actions

### Contact Elected Officials

| Level | Key Asks |
|-------|----------|
| Federal | Support Second Chance Act reauthorization, Clean Slate |
| State | Ban the box, clean slate, voting restoration |
| Local | PHA policy reform, local fair chance ordinances |

**How to Be Effective**:

- Personalize your message
- Share relevant personal stories
- Be specific about what you want
- Follow up persistently
- Build relationships over time
