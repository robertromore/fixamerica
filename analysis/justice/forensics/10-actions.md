# Forensic Science Reform: Actions

## What Individuals, Communities, and Organizations Can Do

Forensic science reform requires action at every level of society. This section provides concrete, actionable steps for individuals, communities, and organizations to advance reform.

## Individual Actions

### For All Citizens

#### Learn and Educate

| Action | Description | Resources |
|--------|-------------|-----------|
| Educate yourself | Learn about forensic science issues | NAS 2009 report; Innocence Project website |
| Share information | Discuss with friends and family | Social media; conversation |
| Attend presentations | Hear from exonerees and experts | Innocence Project events |
| Read investigative journalism | Follow forensic science coverage | Marshall Project; ProPublica |
| Watch documentaries | Visual education on issues | The Innocence Files; others |

#### Civic Engagement

| Action | Description | How To |
|--------|-------------|--------|
| Contact legislators | Express support for reform | Phone calls; emails; meetings |
| Attend town halls | Raise forensic issues | Bring prepared questions |
| Vote informed | Consider candidates' positions | Research criminal justice positions |
| Serve on juries | Apply critical thinking | Question forensic claims |
| Attend court | Observe forensic testimony | Public courtrooms |

#### Financial Support

| Action | Description | Options |
|--------|-------------|---------|
| Donate to reform organizations | Fund advocacy | Innocence Project; state organizations |
| Support exoneree assistance | Help wrongfully convicted | Witness to Innocence; state funds |
| Fund research | Support validation studies | University programs |

### For Legal Professionals

#### Defense Attorneys

| Action | Description | Implementation |
|--------|-------------|----------------|
| Challenge forensic evidence | Daubert/Frye motions | Every applicable case |
| Request error rate information | Discovery of examiner proficiency | Standard discovery |
| Hire defense experts | Independent forensic analysis | Budget requests; expert networks |
| Demand full disclosure | Lab protocols, notes, case information | Comprehensive discovery motions |
| Document systemic issues | Build record for reform | Case documentation; publication |
| Develop forensic competency | Training and education | CLE programs; specialized training |
| Network with other defenders | Share strategies | Professional associations |
| File bar complaints | Address prosecutorial misconduct | Documented cases |

#### Prosecutors

| Action | Description | Implementation |
|--------|-------------|----------------|
| Adopt conviction integrity practices | Review past cases | Create conviction review unit |
| Ensure full disclosure | Provide all forensic information | Open file discovery |
| Train on limitations | Understand forensic science limits | Ongoing education |
| Avoid overstatement | Accurate forensic testimony | Testimony protocols |
| Support lab independence | Advocate organizational separation | Internal advocacy |
| Welcome challenges | Treat defense scrutiny as healthy | Cultural change |
| Report misconduct | Address forensic analyst problems | Internal reporting |

#### Judges

| Action | Description | Implementation |
|--------|-------------|----------------|
| Apply Daubert rigorously | Genuine scientific evaluation | Pretrial hearings |
| Require error rate disclosure | Mandate testimony on limitations | Evidentiary rulings |
| Appoint defense experts | Provide resources for indigent defendants | Case orders |
| Update jury instructions | Accurate forensic evidence instructions | Pattern jury instructions |
| Seek scientific education | Learn forensic science issues | Judicial education programs |
| Document decisions | Create precedent | Written opinions |
