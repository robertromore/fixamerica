## Prison Conditions

### Violence

**Assault Rates**:

| Type | Rate per 1,000 Prisoners |
|------|-------------------------|
| Prisoner-on-prisoner assault | 28 |
| Staff-on-prisoner assault | 4 (likely underreported) |
| Sexual victimization (all) | 40 |
| Sexual victimization (staff) | 17 |

**Contributing Factors**:

- Overcrowding increases tension
- Understaffing reduces supervision
- Gang dynamics drive conflict
- Lack of programming creates idleness
- Mental illness without treatment
- Inadequate responses to reported threats

### Healthcare

**Medical Care**:

| Issue | Prevalence |
|-------|------------|
| Chronic conditions | 50%+ of population |
| Hepatitis C | 17-25% |
| HIV | 1.3% (4x general population) |
| Diabetes | 10%+ |
| Hypertension | 30%+ |

**Problems**:

| Issue | Description |
|-------|-------------|
| Delayed care | Weeks or months for non-emergency |
| Inadequate staffing | Many facilities lack full-time physicians |
| High co-pays | $2-$5 per visit (substantial when earning $0.12/hour) |
| Medication access | Delays, formulary restrictions |
| Specialty care | Often unavailable or severely delayed |
| Death from treatable conditions | Documented cases of preventable mortality |

**Mental Health**:

| Condition | Prevalence in Prison | General Population |
|-----------|---------------------|-------------------|
| Serious mental illness | 15-20% | 4-5% |
| Any mental health condition | 45-65% | 20% |
| History of trauma | 70%+ | varies |
| Intellectual disability | 10% | 2% |

**Mental Health Treatment**:

| Treatment | Access |
|-----------|--------|
| Medication management | Generally available |
| Individual therapy | Rare |
| Group therapy | Limited |
| Specialized units | Insufficient capacity |
| Crisis intervention | Often punitive (solitary) |

### Solitary Confinement

**Scale**:

| Population | Number |
|------------|--------|
| In solitary on any day | ~80,000 |
| Experience solitary annually | 300,000+ |
| Long-term (1+ year) | 25,000+ |

**Conditions**:

| Element | Typical |
|---------|---------|
| Cell size | 6x9 feet |
| Time out of cell | 1 hour/day or less |
| Human contact | Minimal |
| Programming | None |
| Duration limits | Often none |

**Populations Overrepresented**:

- People with mental illness
- LGBTQ+ individuals (often for "protection")
- Gang-affiliated or labeled as such
- Those who report abuse
- Those with disciplinary infractions

**Effects**:

| Effect | Documentation |
|--------|---------------|
| Psychological harm | Extensive clinical evidence |
| Psychosis | Developed in isolation |
| Self-harm | Elevated rates |
| Suicide | Much higher in solitary |
| Long-term cognitive effects | Memory, concentration impairment |

### Food and Nutrition

| Issue | Description |
|-------|-------------|
| Caloric intake | Often insufficient |
| Nutritional quality | Processed, high-sodium |
| Special diets | Religious often inadequate |
| Food as punishment | Restricted diets used |
| Commissary dependence | Essential supplements require money |

### Sanitation and Housing

| Issue | Prevalence |
|-------|------------|
| Overcrowding | Many facilities over capacity |
| Double-celling | Common practice |
| Ventilation | Often inadequate |
| Temperature extremes | Heat and cold documented |
| Pest infestation | Common in older facilities |
| Mold and deterioration | Widespread in aging infrastructure |

### Business Model

**Revenue Sources**:

| Source | Description |
|--------|-------------|
| Per diem payments | Daily rate per prisoner ($50-$100+) |

| Additional services | Phone, commissary, healthcare |

**Cost Structure**:

| Cost Driver | Approach |
|-------------|----------|
| Labor | Lower wages than public facilities |
| Staffing ratios | Often lower than public |
| Programming | Minimal investment |
| Healthcare | Often contracted to lowest bidder |

## Programming

### Educational Programs

| Program | Availability |
|---------|--------------|
| GED/high school | Most facilities |
| Vocational training | Limited |
| College courses | Limited (growing since Pell restoration) |
| Library access | Most facilities |

**Participation**:

- 30-40% participate in some education
- Only 5-10% in college programs
- Waiting lists common
- Funding often insufficient

### Rehabilitative Programs

| Program Type | Availability |
|--------------|--------------|
| Cognitive behavioral therapy | Growing but limited |
| Substance abuse treatment | 10-15% of those who need it |
| Anger management | Common but variable quality |
| Victim awareness | Some facilities |
| Parenting | Limited |

### Barriers to Programming

| Barrier | Impact |
|---------|--------|
| Funding cuts | Program elimination |
| Staffing shortages | Security over programming |
| Transfer/disruption | Interrupted participation |
| Wait lists | Years for some programs |
| Solitary placement | Excludes participation |
| Short sentences | Insufficient time to complete |
