# Prison Reform: Solutions

## Decarceration Strategies

### Sentencing Reform

**Reduce Sentence Lengths**:

| Reform | Mechanism |
|--------|-----------|
| Repeal mandatory minimums | Restore judicial discretion |
| Reduce statutory maximums | Lower ceiling on sentences |
| Presumptive sentencing | Shorter default sentences |
| Eliminate habitual offender enhancements | End sentence stacking |
| Cap sentences for non-violent offenses | Proportionality |

**Evidence Base**:

| Finding | Source |
|---------|--------|
| Marginal deterrence of longer sentences: minimal | Multiple studies |
| Incapacitation benefits decline over time | Age-crime curve research |
| Longer sentences increase recidivism | BJS, academic studies |
| Short sentences as effective as long | Randomized studies |

### Expand Release Mechanisms

**Parole Restoration**:

| Reform | Description |
|--------|-------------|
| Restore discretionary parole | Where abolished |
| Reform parole criteria | Focus on risk, not politics |
| Limit technical violations | Don't reincarcerate for rules |
| Expand good time | More reduction for good behavior |
| Create earned time | Credits for programming |

**Second Look/Resentencing**:

| Reform | Description |
|--------|-------------|
| Second look after 10-15 years | Presumption of release |
| Retroactive sentencing reform | Apply new laws retroactively |
| Universal resentencing | Review all pre-reform sentences |
| Judicial resentencing authority | Periodic review by courts |

**Compassionate Release**:

| Reform | Description |
|--------|-------------|
| Expand medical criteria | Terminal and serious illness |
| Add age-based eligibility | Elderly prisoners low risk |
| Streamline process | Remove bureaucratic barriers |
| Independent review | Not just corrections department |

## Cost-Benefit Considerations

### Costs of Reform

| Reform | Cost Estimate |
|--------|---------------|
| Healthcare improvement | $2-5B/year nationally |
| Programming expansion | $1-3B/year nationally |
| Staffing adequacy | $2-4B/year nationally |
| Facility improvement | Tens of billions (capital) |

### Savings from Reform

| Reform | Savings Estimate |
|--------|-----------------|
| Decarceration (per prisoner) | $35,000-$60,000/year |
| Recidivism reduction | Avoided incarceration costs |
| Reduced healthcare costs | Prevention cheaper than crisis |
| Economic productivity | Released people contribute |

### Net Impact

| Scenario | Net Effect |
|----------|------------|
| 10% population reduction | $4-6B/year savings |
| 25% population reduction | $10-15B/year savings |
| 50% population reduction | $20-30B/year savings |
