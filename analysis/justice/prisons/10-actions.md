# Prison Reform: Actions

## Individual Actions

### Education and Awareness

**Learn About the Issue**:

| Action | How |
|--------|-----|
| Read essential books | *The New Jim Crow*, *Just Mercy*, *Locked In* |
| Follow news coverage | Marshall Project, The Appeal |
| Listen to podcasts | Ear Hustle, Justice in America |
| Attend local events | Town halls, film screenings |
| Learn your state's data | Sentencing Project state pages |

**Share Information**:

| Action | Audience |
|--------|----------|
| Social media | General public |
| Personal conversations | Friends, family |
| Workplace discussions | Colleagues |
| Faith community | Congregation members |
| Book clubs | Deeper discussion |

## Advocacy Campaigns

### Legislative Campaigns

**State Level Priority Issues**:

| Issue | Campaign Focus |
|-------|----------------|
| Sentencing reform | Mandatory minimum repeal |
| Parole restoration | Where abolished |
| Solitary confinement | Restrictions or bans |
| Private prison ban | Phase-out legislation |
| Conditions standards | Healthcare, safety |

**Federal Level Priority Issues**:

| Issue | Campaign Focus |
|-------|----------------|
| First Step Act implementation | Full implementation |
| Further sentencing reform | More reductions |
| BOP reform | Oversight, conditions |
| Private prison ban | Permanent prohibition |

## Action Priority Matrix

### High Impact, Lower Effort

| Action | Why High Impact |
|--------|-----------------|
| Contact legislators | Direct influence |
| Vote in local elections | Prosecutors, judges matter |
| Employer policy change | Affects many |
| Share information | Multiplies awareness |

### High Impact, Higher Effort

| Action | Why Worth It |
|--------|-------------|
| Sustained organizing | Builds power |
| Running for office | Decision-making power |
| Major litigation | Systemic change |
| Coalition leadership | Coordinates action |

### By Available Time

| Time Available | Suggested Actions |
|----------------|-------------------|
| 1 hour/month | Calls to legislators, social media |
| 5 hours/month | Add volunteering, meetings |
| 10+ hours/month | Add organizing, leadership roles |
| Full-time | Career in reform work |
