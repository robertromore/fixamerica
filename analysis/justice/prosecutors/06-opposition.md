# Prosecutorial Reform: Opposition

## Prosecutors' Associations

### National District Attorneys Association

**Position on Key Reforms**:

| Reform | NDAA Position | Argument |
|--------|---------------|----------|
| Open-file discovery | Oppose | Witness safety, work product |
| Mandatory data collection | Oppose | Burden, misuse |
| External oversight | Oppose | Separation of powers |
| Limiting discretion | Strongly oppose | Core prosecutorial function |
| Qualified immunity reform | Oppose | Would chill prosecution |
| Reform prosecutors | Critical | Undermine public safety |

**Lobbying Activities**:

| Activity | Impact |
|----------|--------|
| State legislature lobbying | Block reform bills |
| Federal advocacy | Oppose DOJ oversight |
| Amicus briefs | Support prosecutorial power |
| Media campaigns | Counter reform narratives |
| Training programs | Promote traditional approaches |

### State Prosecutors' Associations

**Variation**:

| State Type | Association Stance |
|------------|-------------------|
| Traditional/conservative | Strong opposition to reform |
| Moderate | Selective opposition |
| Progressive-leaning | Some reform support |

**Common Positions**:

- Oppose any limitations on charging discretion
- Oppose open-file discovery mandates
- Oppose civilian oversight
- Support maintaining immunity
- Oppose data reporting requirements

## Arguments Against Reform

### Public Safety Arguments

| Argument | Claimed Mechanism |
|----------|------------------|
| Crime will increase | Deterrence reduced |
| Recidivism will rise | No consequences |
| Victims abandoned | Offenders not punished |
| Police demoralized | Won't do their jobs |

**Evidence Assessment**:

| Claim | Evidence |
|-------|----------|
| Reform causes crime | Mixed; no clear causal link |
| Incarceration deters | Marginal deterrent effect |
| Prosecution prevents recidivism | Prison increases recidivism |
| Tough prosecution = safety | Mass incarceration didn't reduce crime proportionally |

### Institutional Arguments

| Argument | Claimed Basis |
|----------|--------------|
| Prosecutors must have discretion | System would collapse |
| External oversight undermines independence | Separation of powers |
| Data collection is burden | Resources diverted |
| Discovery endangers witnesses | Witness safety |

### Slippery Slope Arguments

| Argument | Claim |
|----------|-------|
| Any reform leads to chaos | Moderation impossible |
| Reform is abolition in disguise | Hidden agenda |
| Can't stop at reform | Will end prosecution |

## Opposition Effectiveness

### Successful Tactics

| Tactic | Success Rate |
|--------|--------------|
| Recall campaigns | Mixed (SF yes, LA no) |
| State legislation | Significant in red states |
| Media campaigns | Effective at shifting narrative |
| Internal resistance | Slows implementation |
| Judicial opposition | Undermines some policies |

### Limitations

| Limitation | Effect |
|------------|--------|
| Voters elected reformers | Democratic mandate |
| Crime data mixed | Hard to prove causation |
| Reform prosecutors adapting | Learning from attacks |
| Victim alternatives emerging | Not monolithic victim voice |
| Evidence base growing | Research supports reform |

## Counter-Strategies

### Reform Response to Opposition

| Opposition Tactic | Counter-Strategy |
|------------------|------------------|
| Crime wave narrative | Comparative data, trend analysis |
| Victim exploitation | Amplify reform-minded victims |
| Recall campaigns | Voter mobilization |
| Internal resistance | Personnel changes, training |
| Media attacks | Proactive communications |
| State legislation | Coalition building, legal challenges |
