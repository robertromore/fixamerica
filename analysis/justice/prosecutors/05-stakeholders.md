# Prosecutorial Reform: Stakeholders

## Prosecutors

### District Attorneys

**Profile**:

| Characteristic | Description |
|----------------|-------------|
| Number | ~2,400 elected DAs nationally |
| Term | Typically 4 years |
| Selection | 95% elected; 5% appointed |
| Authority | Near-total over charging, pleas, policy |

**Interests**:

| Interest | Priority Level |
|----------|----------------|
| Maintain discretion | Highest |
| Political viability | High |
| Conviction rates | High (varies) |
| Public safety | Stated high |
| Office resources | High |
| Personal career | Varies |

**Divisions Within**:

| Group | Orientation |
|-------|-------------|
| Traditional prosecutors | Maintain status quo |
| Reform prosecutors | Transform system |
| Moderate reformers | Incremental change |

### Line Prosecutors

**Characteristics**:

| Factor | Description |
|--------|-------------|
| Number | ~30,000 state prosecutors |
| Demographics | ~75% white, ~65% male |
| Career path | Often to DA, judiciary, private practice |
| Autonomy | High within office guidelines |

**Concerns**:

| Concern | Perspective |
|---------|-------------|
| Caseload | Often overwhelming |
| Safety | Threats from defendants |
| Resources | Inadequate support |
| Professional autonomy | Want discretion |
| Career advancement | Conviction record matters |
| Policy changes | May resist reform |

### Prosecutors' Associations

**Major Organizations**:

| Organization | Role |
|--------------|------|
| National District Attorneys Association (NDAA) | Primary national association |
| State prosecutors' associations | State-level advocacy |
| Association of Prosecuting Attorneys (APA) | Progressive-leaning |

**NDAA Positions**:

| Issue | Position |
|-------|----------|
| Qualified immunity | Support maintaining |
| Discovery reform | Oppose open-file |
| Reform prosecutors | Oppose/criticize |
| Oversight | Oppose external |
| Data transparency | Resist mandates |

## Defendants and Affected Communities

### Criminal Defendants

**Demographics**:

| Characteristic | Overrepresentation |
|----------------|-------------------|
| Black Americans | 3x population share |
| Latino Americans | 1.5x population share |
| Low-income | Vast majority |
| Mental illness | ~40% of incarcerated |
| Prior convictions | Majority |

**Interests**:

| Interest | Need |
|----------|------|
| Fair charging | Proportionate to conduct |
| Full discovery | Know evidence before plea |
| Meaningful choice | Real option beyond plea |
| Proportionate sentences | Not trial penalty |
| Competent defense | Resourced representation |
| Post-conviction review | Remedy for errors |

### Wrongfully Convicted

**Exoneree Population**:

| Metric | Data |
|--------|------|
| Total exonerations | 3,400+ since 1989 |
| Years lost | 28,000+ person-years |
| Average time served | 8.7 years |
| Prosecutorial misconduct cases | ~30% of exonerations |

**Interests**:

| Interest | Need |
|----------|------|
| Release | Freedom from wrongful incarceration |
| Compensation | For years lost |
| Record clearing | Expungement |
| Accountability | Prosecutorial consequences |
| Systemic reform | Prevent future wrongful convictions |

### Families of Incarcerated

**Impact**:

| Effect | Scale |
|--------|-------|
| Children with incarcerated parent | 5+ million |
| Economic loss | Primary earner often |
| Housing instability | Loss of income |
| Family separation | Relationship damage |
| Collateral consequences | Shared stigma |

### Communities

**High-Prosecution Communities**:

| Characteristic | Impact |
|----------------|--------|
| Over-policed neighborhoods | More exposure to prosecution |
| Concentrated poverty | Less ability to fight charges |
| Black and Latino neighborhoods | Disproportionate prosecution |
| Intergenerational cycles | Conviction begets conviction |

**Interests**:

| Interest | Need |
|----------|------|
| Safety | From crime and from system |
| Fairness | Equal treatment |
| Resources | Investment over incarceration |
| Voice | Input into prosecution policy |
| Trust | In justice system |

## Crime Victims

### Victim Population

**Characteristics**:

| Factor | Reality |
|--------|---------|
| Overlap with defendant population | Significant |
| Race | Disproportionately Black and Latino |
| Economic status | Disproportionately low-income |
| Repeat victimization | Common |
| Reporting rates | Many crimes unreported |

### Victim Interests

**Diverse Needs**:

| Need | Variation |
|------|-----------|
| Accountability | Some want harsh; others want healing |
| Safety | Protection from perpetrator |
| Voice | Heard in process |
| Compensation | For losses |
| Closure | Process resolution |
| Information | About case status |

**Victim Perspectives on Reform**:

| Group | Tendency |
|-------|----------|
| Some victim advocates | Support tough prosecution |
| Crime survivor organizations | Mixed on reform |
| Victim-offender programs | Support alternatives |
| Families of murdered | Deeply divided |

### Victim Advocacy Organizations

|--------------|-------------|

| Mothers Against Drunk Driving | Punishment-focused |
| National Center for Victims of Crime | Mixed |
| State victim advocacy groups | Often prosecution-aligned |

**Reform-Oriented Groups**:

| Organization | Focus |
|--------------|-------|
| Crime Survivors for Safety and Justice | Restorative approaches |
| Alliance for Safety and Justice | Alternative accountability |
| Survived & Punished | Criminalized survivors |

## Judges

### Trial Judges

**Role in Prosecution Issues**:

| Function | Reality |
|----------|---------|
| Plea acceptance | Rubber stamp mostly |
| Brady enforcement | Varies widely |
| Sentencing | Bound by recommendations often |
| Discovery disputes | Defers to prosecutors |

**Interests**:

| Interest | Priority |
|----------|----------|
| Docket management | High (drives behavior) |
| Case disposition | Quick resolution valued |
| Appellate affirmance | Don't want reversals |
| Judicial independence | From criticism |
| Public safety perception | Electoral vulnerability |

### Appellate Judges

**Role**:

| Function | Approach |
|----------|----------|
| Brady review | Deferential |
| Misconduct review | Harmless error doctrine |
| Standard setting | Limited reform leadership |

## Law Enforcement

### Police

**Relationship with Prosecutors**:

| Aspect | Dynamic |
|--------|---------|
| Case development | Close partnership |
| Charging recommendations | Police influence varies |
| Witness coordination | Joint management |
| Decline pushback | Police object to declinations |
| Political alliance | Often aligned |

**Interests Regarding Prosecution**:

| Interest | Perspective |
|----------|-------------|
| Case acceptance | Want cases prosecuted |
| Cooperation | Supportive prosecutors |
| Credit for arrests | Convictions validate work |
| Reform prosecutors | Generally oppose |

### Police Unions

**Role in Prosecution Politics**:

| Activity | Impact |
|----------|--------|
| DA endorsements | Major in elections |
| Opposition to reform DAs | Funded recalls |
| Media campaigns | Against progressive prosecutors |
| Lobbying | For tough-on-crime policies |
