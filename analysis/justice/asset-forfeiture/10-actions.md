# Action Items for Civil Asset Forfeiture Reform

## Overview

This document provides concrete action items for individuals, communities, and organizations seeking to reform civil asset forfeiture. Actions are organized by stakeholder type and effort level.

## Individual Actions

### Immediate Actions (1-2 Hours)

| Action | Description | Impact |
|--------|-------------|--------|
| Educate yourself | Read overview and current state documents | Foundation |
| Share information | Post articles on social media | Awareness |
| Sign petitions | Support reform petitions in your state | Political signal |
| Contact legislators | Email or call state representatives | Direct pressure |
| Register to vote | Ensure you can vote in elections | Political power |

### Short-Term Actions (1-2 Days)

| Action | Description | Impact |
|--------|-------------|--------|
| Write letters to editor | Submit to local newspapers | Public awareness |
| Attend town halls | Ask questions about forfeiture | Accountability |
| Join organizations | Become member of reform groups | Movement building |
| Research local data | FOIA requests for local seizure data | Transparency |
| Identify local cases | Document forfeiture stories in community | Evidence |

### Ongoing Actions

| Action | Frequency | Impact |
|--------|-----------|--------|
| Follow reform news | Weekly | Stay informed |
| Contact legislators | During session | Sustained pressure |
| Vote on reform | Every election | Electoral accountability |
| Donate to reform organizations | As able | Resource support |
| Mentor others | Ongoing | Movement growth |

## Know Your Rights

### During a Traffic Stop

| Right | How to Exercise | Script |
|-------|-----------------|--------|
| Remain silent | Invoke clearly | "I am exercising my right to remain silent." |
| Refuse consent to search | State clearly | "I do not consent to any searches." |
| Ask if free to go | Clarify detention | "Am I free to leave?" |
| Record interaction | If legal in your state | Know state recording laws |
| Get badge number | Request politely | "May I have your badge number?" |

### If Property Is Seized

| Step | Timing | Action |
|------|--------|--------|
| Get documentation | Immediately | Request seizure receipt |
| Note details | Same day | Document everything |
| Check deadlines | Immediately | Review all paperwork for deadlines |
| Seek legal help | Within days | Contact legal aid or attorney |
| File claim | Before deadline | Submit formal claim to contest |

### Protecting Yourself

| Protection | Method | Reason |
|------------|--------|--------|
| Carry documentation | Bank withdrawal slips, receipts | Prove legitimate source |
| Avoid large cash | Use cards, checks when possible | Reduce seizure risk |
| Know your route | Highway interdiction zones | Awareness |
| Travel with witnesses | When carrying significant cash | Corroboration |
| Record interactions | Where legal | Documentation |

## Legislative Engagement

### During Legislative Session

| Activity | Timing | Method |
|----------|--------|--------|
| Track legislation | Continuous | Bill monitoring services |
| Contact sponsors | When bill introduced | Express support |
| Attend hearings | When scheduled | Provide testimony |
| Lobby legislators | Before key votes | Meetings, calls |
| Mobilize constituents | Before key votes | Action alerts |

### Effective Constituent Contact

| Method | Effectiveness | Tips |
|--------|---------------|------|
| In-person meeting | Highest | Schedule appointment, bring materials |
| Phone call | High | Be brief, identify as constituent |
| Personalized letter | High | Handwritten or typed, specific |
| Personalized email | Moderate | Specific, brief, include address |
| Form email/petition | Lower | Better than nothing |

### Testimony Tips

| Element | Guidance |
|---------|----------|
| Preparation | Know bill, opposition arguments |
| Personal story | Lead with experience if applicable |
| Data | Include key statistics |
| Ask | Make specific request |
| Time | Respect limits, be concise |
| Written record | Submit written testimony |
| Follow up | Thank committee, continue engagement |
