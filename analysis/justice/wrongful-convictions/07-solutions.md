# Wrongful Convictions: Solutions

## Eyewitness Identification Reform

### Evidence-Based Procedures

**Best Practices**:

| Practice | Purpose |
|----------|---------|
| Double-blind administration | Prevent unintentional cues |
| Sequential presentation | Reduce relative judgment |
| Unbiased instructions | Tell witness perpetrator may not be present |
| Confidence statement | Record at time of ID |
| Video recording | Document procedure |
| Filler selection | Fair lineup construction |

**Implementation**:

- Statewide mandates
- Police training
- Model policies
- Compliance monitoring

### Limiting Use of Eyewitness Evidence

**Court Reforms**:

- Expert testimony on memory
- Jury instructions on reliability
- Suppression of suggestive IDs
- Reliability assessment before admission

## Interrogation Reform

### Recording Requirements

**Comprehensive Recording**:

| Requirement | Specification |
|-------------|---------------|
| Scope | All custodial interrogations |
| Crimes covered | All felonies minimum |
| Recording | Complete, from Miranda |
| Exceptions | Narrow and specified |
| Consequences | Suppression if not recorded |

### Interrogation Method Reform

**Move Beyond Reid Technique**:

- PEACE method (UK model)
- Cognitive interviewing
- Information-gathering approach
- Reduce false confession risk
- Training reform

### Protections for Vulnerable Populations

**Youth**:

- Specialized training for youth interrogation
- Parental presence (real, not waived)
- Shorter time limits
- Age-appropriate language

**Intellectually Disabled**:

- Screen for disability
- Modified procedures
- Support person present
- Additional protections

## Forensic Science Reform

### Validate Existing Techniques

**Requirements**:

- Scientific validation of techniques
- Error rate determination
- Blind proficiency testing
- Ongoing quality assurance

### Reform Crime Labs

| Reform | Purpose |
|--------|---------|
| Independence from police | Reduce bias |
| Accreditation required | Ensure standards |
| Analyst certification | Competency |
| Quality management | Prevent errors |
| Defense access | Balance |

### Limit Testimony

**Testimony Standards**:

- Testify only to validated science
- No overstated conclusions
- Acknowledge limitations
- Error rates disclosed
- No "match" without basis

### Create National Standards

**Federal Action**:

- National forensic science commission
- Binding standards
- Research funding
- Oversight and enforcement

## Prosecutorial Reform

### Conviction Integrity Units

**Effective CIU Requirements**:

| Element | Purpose |
|---------|---------|
| Independence | Separate from trial prosecutors |
| Resources | Dedicated staff and budget |
| Access | Can investigate any case |
| Partnership | Work with innocence projects |
| Transparency | Public reporting |

### Brady Reform

**Strengthened Disclosure**:

- Open file discovery
- Early disclosure requirements
- Checklist compliance
- Consequences for violations
- Training requirements

### Accountability

**Prosecutorial Accountability**:

| Mechanism | Purpose |
|-----------|---------|
| Bar discipline | Professional consequences |
| Named in opinions | Public record |
| Pattern tracking | Identify repeat violators |
| Remedies for defendants | Vacatur, new trial |
| Decertification | Remove from office |

## Defense Quality

### Resource Parity

**Requirements**:

- Funding for investigation
- Expert access
- Forensic testing
- Adequate time per case
- Caseload limits

### Training

**Specialized Training**:

- Eyewitness science
- False confession recognition
- Forensic science challenges
- Innocence investigation

## Post-Conviction Access

### Expanded DNA Testing

**Model Provisions**:

| Element | Requirement |
|---------|-------------|
| Access | Any convicted person |
| Standard | Reasonable probability testing would prove innocence |
| Funding | State pays |
| Time limits | None |
| Consequences | New trial if exculpatory |

### Non-DNA Innocence Claims

**Access to Courts**:

- Freestanding innocence claims allowed
- New evidence claims without time limit
- Lower standard for review
- Actual innocence exception to procedural default

### Evidence Preservation

**Requirements**:

- Mandatory preservation of biological evidence
- Preservation for duration of sentence
- Notification before destruction
- Sanctions for destruction

## Compensation and Support

### Compensation Standards

**Adequate Compensation**:

| Element | Standard |
|---------|----------|
| Amount | $100,000/year minimum |
| Benefits | Health care, counseling |
| Barriers | No false confession bar |
| Timing | Automatic upon exoneration |
| Process | Administrative, not lawsuit |

### Reentry Support

**Services for Exonerees**:

- Immediate housing
- Health care (physical and mental)
- Job training and placement
- Education assistance
- Legal services
- Case management

## Prevention

### Training

**System-Wide Training**:

| Audience | Topics |
|----------|--------|
| Police | Eyewitness procedures, interrogation |
| Prosecutors | Brady, tunnel vision |
| Defense | Innocence investigation |
| Judges | Forensic science, memory |
| Forensic analysts | Cognitive bias |

### Culture Change

**Institutional Reform**:

- Accuracy over conviction rates
- Openness to error
- Continuous improvement
- Learning from exonerations
- Accountability without defensiveness
