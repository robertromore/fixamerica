# Victims' Rights and Services: Actions

## What Individuals, Communities, and Organizations Can Do

### Individual Actions

#### If You Are a Crime Victim

| Action | Description | Resources |
|--------|-------------|-----------|
| **Know your rights** | Learn victim rights in your jurisdiction | State AG website, NCVLI VictimLaw |
| **Report the crime** | File police report if safe to do so | Local police, online reporting options |
| **Seek medical care** | Get immediate health needs addressed | ER, urgent care, SANE nurses |
| **Contact victim services** | Reach out to local advocacy organization | VictimConnect: 1-855-484-2846 |
| **Apply for compensation** | File state compensation claim | State compensation board |
| **Document everything** | Keep records of expenses, impacts | Photos, receipts, journals |
| **Request notification** | Enroll in case status notification | VINE: 1-877-411-5588 |
| **Prepare impact statement** | Write about crime's effects | Victim advocate can assist |
| **Seek counseling** | Get mental health support | Victim services, insurance, community mental health |
| **Connect with support groups** | Find peer support | Local DV/SA programs, POMC |

#### If Someone You Know Is a Victim

| Action | Do | Don't |
|--------|------|-------|
| **Listen** | Believe them, be present | Interrogate, question decisions |
| **Validate** | Affirm their experience | Minimize or compare |
| **Provide information** | Share resources, hotlines | Pressure to act |
| **Respect choices** | Support their decisions | Make decisions for them |
| **Offer practical help** | Childcare, rides, meals | Overwhelm with help |
| **Be patient** | Healing takes time | Set timelines |
| **Maintain confidentiality** | Keep their story private | Share without permission |
| **Check in** | Stay connected over time | Disappear after initial crisis |
| **Take care of yourself** | Get your own support | Burn out trying to help |

### Getting Started: First Steps

#### For Individuals

1. Learn about victim rights in your state
2. Save local victim services hotline in phone
3. Learn bystander intervention techniques
4. Make one donation to local victim services
5. Tell one person about victim resources

#### For Organizations

1. Assess current victim-related policies
2. Identify one partnership opportunity
3. Schedule one training on trauma/victimization
4. Add victim resources to communications
5. Join one coalition or advocacy effort

#### For Communities

1. Map existing victim services
2. Identify underserved populations
3. Convene stakeholder meeting
4. Develop action plan
5. Implement one priority initiative
