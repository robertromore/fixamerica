# Victims' Rights and Services: Solutions

## Reform Proposals for Better Victim Services

### Framework for Solutions

Effective reforms must address multiple dimensions simultaneously:

| Dimension | Focus |
|-----------|-------|
| **Rights Enhancement** | Strengthening legal protections |
| **Implementation** | Making rights real in practice |
| **Funding** | Sustainable financial support |
| **Access** | Reaching underserved populations |
| **Quality** | Improving service delivery |
| **Prevention** | Reducing future victimization |
| **System Change** | Transforming institutional culture |

### Rights Enhancement Reforms

#### 1. Strengthen Enforcement Mechanisms

**Problem**: Rights without remedies are hollow.

| Reform | Description |
|--------|-------------|
| Standing to enforce | Allow victims to independently seek court enforcement |
| Mandamus relief | Expedited court review when rights violated |
| Damages remedy | Compensation when rights are denied |
| Attorney fee shifting | Legal costs covered when rights enforced |
| Ombudsman offices | Independent victim rights monitors |

**Model Approach**: Arizona and California have created dedicated victim rights enforcement mechanisms with independent ombudsman offices.

#### 2. Expand Notification Requirements

**Problem**: Victims frequently unaware of case developments.

| Current Gap | Reform |
|-------------|--------|
| Inconsistent notification | Automated, multi-modal systems |
| No opt-in default | Automatic enrollment with opt-out |
| Limited events covered | Comprehensive case event notification |
| No custody status | Real-time custody and release alerts |
| Language barriers | Multi-language notification |

**Technology Solution**: Expand VINE (Victim Information and Notification Everyday) to universal coverage with text, email, phone, and app options.

#### 3. Enhance Participation Rights

| Participation Area | Reform |
|-------------------|--------|
| Plea negotiations | Meaningful consultation before plea offers |
| Bail/detention hearings | Right to be heard on safety |
| Sentencing | Protected victim impact statement time |
| Parole hearings | Video/written statement options |
| Case disposition | Explanation of outcomes |

#### 4. Address Digital-Age Challenges

| Challenge | Reform |
|-----------|--------|
| Online harassment | Expand protective orders to digital spaces |
| Image-based abuse | Expedited takedown procedures |
| Stalking apps | Criminalize stalkerware |
| Digital evidence | Victim access to own data |
| Privacy breaches | Enhanced data protection |

### Funding Reforms

#### 1. Stabilize and Grow VOCA

**Problem**: VOCA funding has collapsed from $4.4 billion to $1.5 billion.

| Reform | Impact |
|--------|--------|
| Capture all corporate penalties | Deferred/non-prosecution agreements into fund |
| Minimum floor guarantee | Baseline funding regardless of deposits |
| Inflation adjustments | Automatic cost-of-living increases |
| Multi-year grants | Organizational stability |
| Reduced administrative burden | More funds to services |

**VOCA Fix 2.0 Components**:
1. All monetary penalties from corporate crimes into fund
2. $3 billion annual floor guarantee
3. 5-year grant cycles for stable organizations
4. 15% administrative cap
5. Automatic COLA adjustments

#### 2. Reform State Compensation Programs

| Current Problem | Reform |
|-----------------|--------|
| Low caps ($25,000 typical) | Raise to $100,000 minimum |
| Narrow eligibility | Include property crimes, expand covered expenses |
| Documentation barriers | Alternative verification methods |
| Slow processing | 30-day processing mandate |
| Reporting requirements | Remove police report mandates |
| Contributory conduct exclusions | Eliminate or narrow substantially |

**Model Program Elements**:
- Universal eligibility for violent crime victims
- No contributory conduct disqualification
- Emergency awards within 72 hours
- Multi-language application assistance
- No cooperation requirements beyond reasonable participation

#### 3. Improve Restitution Collection

**Problem**: Only 8-15% of ordered restitution is ever collected.

| Reform | Mechanism |
|--------|-----------|
| Wage garnishment authority | Automatic withholding |
| Tax refund intercept | IRS/state coordination |
| State collection assistance | Professional collectors |
| Payment plans | Ability-to-pay based |
| Centralized collection | State-level coordination |
| Insurance offsets | Coordinate with other sources |
| Victim notification | Regular payment updates |

#### 4. Create Sustainable Funding Streams

| New Funding Source | Mechanism |
|-------------------|-----------|
| Opioid settlement funds | Dedicated victim services allocation |
| Cannabis revenue | State earmarks for victim programs |
| Civil asset forfeiture | Portion to victim services |
| Insurance surcharges | Small policy add-ons |
| Court fee restructuring | Victim services priority |

### Access Improvement Reforms

#### 1. Reach Underserved Populations

**Immigrant Victims**:

| Barrier | Solution |
|---------|----------|
| Deportation fear | Strengthened confidentiality, U-visa expansion |
| U-visa backlog | Increase visa cap, expedite processing |
| Language barriers | Certified interpreters, multilingual materials |
| System unfamiliarity | Community navigator programs |
| Public charge concerns | Clear exemptions for victim services |

**LGBTQ+ Victims**:

| Barrier | Solution |
|---------|----------|
| Provider competency | Mandatory cultural competency training |
| Shelter access | Affirming placement policies |
| Service gaps | Dedicated LGBTQ+ victim programs |
| IPV recognition | Gender-neutral DV frameworks |
| Data collection | SOGI (Sexual Orientation and Gender Identity) data |

**Homeless Victims**:

| Barrier | Solution |
|---------|----------|
| Address requirements | Waive residency requirements |
| Documentation | Alternative ID verification |
| Competing needs | Integrated services (housing + victim services) |
| Distrust | Street outreach programs |
| Follow-up | Persistent outreach methods |

**Rural Victims**:

| Barrier | Solution |
|---------|----------|
| Distance to services | Mobile advocacy, telehealth |
| Limited providers | Regional service networks |
| Anonymity challenges | Confidential services |
| Transportation | Transportation assistance |
| Technology gaps | Multi-modal service delivery |

#### 2. Remove Reporting Requirements

| Current Requirement | Reform |
|--------------------|--------|
| Police report for compensation | Remove or provide alternatives |
| Cooperation requirements | Limit to reasonable participation |
| Crime verification | Alternative evidence accepted |
| Mandatory reporting conflicts | Address in eligibility rules |

#### 3. Expand Eligibility

| Current Exclusion | Reform |
|-------------------|--------|
| Property crime victims | Trauma services available |
| Fraud/identity theft | Financial recovery assistance |
| "Contributory" victims | Eliminate disqualifying factors |
| Undocumented immigrants | Equal access regardless of status |
| Victims with criminal records | No exclusion based on own history |

### Quality Improvement Reforms

#### 1. Implement Trauma-Informed Practices

| System | Trauma-Informed Reform |
|--------|------------------------|
| Law enforcement | Universal trauma training, interview protocols |
| Prosecution | Victim-centered practices, reduced re-interviewing |
| Courts | Accommodations, support persons, scheduling flexibility |
| Service providers | Evidence-based trauma treatment |
| Corrections | Notification procedures, safety planning |

**Core Principles**:
1. Safety (physical and emotional)
2. Trustworthiness and transparency
3. Peer support
4. Collaboration and mutuality
5. Empowerment, voice, and choice
6. Cultural, historical, and gender responsiveness

#### 2. Establish Service Standards

| Standard Area | Requirement |
|---------------|-------------|
| Response times | 24-hour crisis response |
| Accessibility | Physical, language, disability access |
| Confidentiality | Clear policies, training |
| Cultural competency | Demonstrated capacity |
| Evidence-based practices | Required for funded programs |
| Outcome measurement | Standardized metrics |

#### 3. Improve Victim Advocacy

| Current Gap | Reform |
|-------------|--------|
| Inconsistent qualifications | Certification standards |
| Limited training | Standardized curricula |
| Poor compensation | Living wage requirements |
| Burnout | Caseload limits, self-care support |
| No attorney access | Right to victim's attorney |

#### 4. Expand Restorative Justice Options

| Reform Component | Description |
|------------------|-------------|
| Victim-offender dialogue | Available in all jurisdictions |
| Victim choice | Always voluntary |
| Trained facilitators | Professional standards |
| Safety protocols | Thorough preparation |
| Post-process support | Ongoing services available |
| Funding parity | Equal to traditional options |

| Healthcare-justice links | SANE program expansion |
| Immigration-victim services | Certified liaison programs |
| Child welfare-victim services | Joint protocols |
| Housing-victim services | Coordinated entry |

#### 3. Technology Modernization

| Technology Need | Solution |
|-----------------|----------|
| Case management | Standardized platforms |
| Notification systems | Universal, modern VINE |

### Specific Reform Proposals by Crime Type

#### Domestic Violence

| Reform | Implementation |
|--------|----------------|
| Firearm restrictions | Universal relinquishment upon order |
| Housing protections | Early lease termination, anti-discrimination |
| Economic security | Emergency funds, job protections |
| Lethality assessment | Universal screening |
| Coercive control laws | Criminalize non-physical abuse patterns |

#### Sexual Assault

| Reform | Implementation |
|--------|----------------|
| Rape kit testing | Mandatory testing, tracking |
| Statute of limitations | Eliminate for sexual assault |
| Shield law strengthening | Broader evidence exclusions |
| Affirmative consent | Legal standard update |
| Campus coordination | Improved Title IX processes |

#### Human Trafficking

| Reform | Implementation |
|--------|----------------|
| Vacatur laws | Expunge trafficking-related records |
| Safe harbor laws | Minors immune from prostitution charges |
| Comprehensive services | Housing, healthcare, legal, employment |
| Demand reduction | John schools, buyer accountability |
| Labor trafficking focus | Equal attention with sex trafficking |

#### Elder Abuse

| Reform | Implementation |
|--------|----------------|
| Financial exploitation | Enhanced banking protections |
| APS resources | Increased adult protective services |
| Guardianship reform | Abuse prevention safeguards |
| Healthcare training | Universal abuse screening |
| Penalties | Enhanced sanctions for elder crimes |

### Summary: Comprehensive Reform Agenda

| Priority Level | Reform Area |
|---------------|-------------|
| **Immediate** | VOCA stabilization, notification systems, compensation reform |
| **Short-term** | Enforcement mechanisms, underserved population access, trauma training |
| **Medium-term** | Restorative justice expansion, prevention investment, technology modernization |
| **Long-term** | System transformation, survivor-centered redesign, sustainable funding |
