# Roadmap: Implementation Timeline for Mental Health Crisis Response Reform

## Overview

Transforming mental health crisis response requires phased implementation over multiple years. This roadmap outlines a realistic timeline for communities to build the infrastructure, workforce, and systems needed for non-police crisis response and justice system diversion.

## Phases of Implementation

| Phase | Timeline | Focus |
|-------|----------|-------|
| Phase 1 | Months 1-6 | Assessment, planning, coalition-building |
| Phase 2 | Months 6-18 | Foundational infrastructure; pilot programs |
| Phase 3 | Years 2-3 | Expansion and integration |
| Phase 4 | Years 3-5 | Sustainability and continuous improvement |
| Phase 5 | Years 5+ | Full system transformation |

## Phase 1: Assessment and Planning (Months 1-6)

### Objective
Understand current system, build coalitions, and develop implementation plan.

### Activities

#### Month 1-2: Assessment

| Task | Deliverable |
|------|-------------|
| Map current crisis response system | System map document |
| Analyze 911 call data | Call volume; types; outcomes |
| Inventory existing resources | Services; capacity; gaps |
| Identify key stakeholders | Stakeholder list; initial engagement |
| Review legal and policy landscape | Legal memo on requirements/barriers |

#### Month 2-4: Coalition Building

| Task | Deliverable |
|------|-------------|
| Convene stakeholder group | Working group established |
| Engage police department | Police participation secured |
| Engage mental health providers | Provider commitment |
| Engage people with lived experience | Advisory group formed |
| Meet with elected officials | Political support assessed |

#### Month 4-6: Planning

| Task | Deliverable |
|------|-------------|
| Define model(s) to implement | Model selection document |
| Develop implementation plan | Detailed plan with timeline |
| Identify funding sources | Funding strategy |
| Address legal/liability issues | Legal framework established |
| Design evaluation framework | Evaluation plan |

### Key Milestones

| Milestone | Target |
|-----------|--------|
| System mapping complete | Month 2 |
| Stakeholder coalition formed | Month 3 |
| Implementation plan adopted | Month 6 |
| Funding identified | Month 6 |

## Phase 2: Foundation and Pilots (Months 6-18)

### Objective
Build foundational infrastructure and launch pilot programs.

### Crisis Response Pilot

#### Months 6-9: Preparation

| Task | Deliverable |
|------|-------------|
| Hire crisis response leadership | Director; supervisors |
| Develop dispatch protocols | Call screening criteria |
| Train 911 dispatchers | Training completed |
| Procure vehicles and equipment | Fleet ready |
| Develop clinical protocols | Response protocols |
| Negotiate police backup agreements | MOU signed |

#### Months 9-12: Staffing and Training

| Task | Deliverable |
|------|-------------|
| Recruit crisis responders | Staff hired |
| Develop training curriculum | Curriculum complete |
| Train crisis teams | Initial cohort trained |
| Conduct ride-alongs with existing services | Field experience |
| Shadow police mental health calls | Understanding of current response |

#### Months 12-15: Pilot Launch

| Task | Deliverable |
|------|-------------|
| Launch pilot in defined area | Pilot operational |
| Limited hours initially | Coverage hours established |
| Parallel response with police initially | Safety protocols in place |
| Real-time data collection | Data system operational |
| Weekly team debriefs | Continuous learning process |

#### Months 15-18: Pilot Evaluation and Adjustment

| Task | Deliverable |
|------|-------------|
| Analyze pilot data | Evaluation report |
| Adjust protocols based on experience | Revised protocols |
| Expand hours/area if warranted | Expansion plan |
| Develop sustainability plan | Long-term funding secured |

## Phase 5: Full System Transformation (Years 5+)

### Objective
Achieve fundamental system transformation where mental health crises receive appropriate mental health response.

### System Vision

| Outcome | Target |
|---------|--------|
| Majority of mental health calls to non-police | 70%+ |
| Police as backup, not primary | Standard practice |
| Zero deaths during mental health crisis response | Goal |
| Jails not primary mental health facilities | Dramatic reduction in SMI in jails |
| Community services meet need | Adequate capacity |

### Continuous Evolution

| Activity | Ongoing |
|----------|---------|
| Adapt to emerging needs | Continuous |
| Incorporate new evidence | Continuous |
| Expand upstream prevention | Continuous |
| Address root causes | Continuous |

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| Political opposition | Early stakeholder engagement; data; framing |
| Workforce shortage | Training pipelines; competitive compensation |
| Police union opposition | Engage early; address concerns; find common ground |
| Funding uncertainty | Diverse funding streams; sustainability planning |
| Implementation challenges | Phased approach; flexibility; learning culture |
| High-profile incident | Clear protocols; quality assurance; rapid response plans |
