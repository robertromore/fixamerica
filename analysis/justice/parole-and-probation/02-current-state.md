# Parole and Probation Reform: Current State Analysis

## Overview of Community Supervision in America

The United States supervises more people in the community than any other nation, with approximately 4.5 million individuals under probation or parole supervision on any given day. This population exceeds the combined prison and jail population and represents a vast system of state control operating largely outside public view.

| Problem | Description |
|---------|-------------|
| Reduced contact | Meaningful supervision impossible |

| Revocation default | Jail as easiest response |
