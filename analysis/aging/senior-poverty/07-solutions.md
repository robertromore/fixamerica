# Senior Poverty and Economic Security: Solutions

## Reform Framework

### Guiding Principles

| Principle | Application |
|-----------|-------------|
| **Dignity in aging** | No one should face poverty after a lifetime of work |
| **Universal foundation** | Strengthen Social Security as primary retirement security |
| **Targeted enhancement** | Additional support for most vulnerable populations |
| **Prevention** | Build retirement security before people reach old age |
| **Equity** | Address disparities by gender, race, and geography |
| **Simplicity** | Reduce barriers to accessing benefits |
| **Adequacy** | Benefits should meet actual cost of living |
| **Sustainability** | Ensure long-term program solvency |

### Strategic Framework

| Strategy | Components |
|----------|------------|
| **Strengthen Social Security** | Benefit enhancements; solvency; equity improvements |
| **Modernize SSI** | Raise benefits; update rules; improve access |
| **Reduce healthcare costs** | Medicare improvements; drug pricing; coverage gaps |
| **Address housing** | Affordable housing expansion; rent assistance |
| **Prevent future poverty** | Universal retirement savings; pension protection |
| **Target disparities** | Gender gap fixes; racial equity; rural access |

## Proposed Solutions

### Solution 1: Strengthen Social Security Benefits

**Description**: Increase Social Security benefits for low-income beneficiaries, improve the COLA formula, and establish a minimum benefit floor that ensures no one who worked a full career lives in poverty.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Minimum benefit** | $1,370/month (125% FPL) for 30+ years of work |
| **Low earner boost** | Increase PIA for bottom two bend points |
| **Improved COLA** | Use CPI-E (elderly-specific) for adjustments |
| **Benefit enhancement** | Increase benefits for current beneficiaries |

**Evidence:**

| Finding | Source |
|---------|--------|
| Minimum benefit would cut elderly poverty 20%+ | CBPP analysis |
| CPI-E averages 0.2% higher than CPI-W | BLS data |
| Benefits adequate for basic needs in only 3 states | Elder Index |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Cost | Pair with revenue increases (payroll cap lift) |
| Complexity | Phase in over 10 years |
| Political opposition | Strong public support for Social Security |

### Solution 2: Social Security Equity Improvements

**Description**: Modify Social Security benefit formulas to address historical inequities affecting women, caregivers, and low-wage workers.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Caregiver credits** | Count up to 5 years of caregiving in benefit calculation |
| **Benefit calculation period** | Reduce from 35 to 30 highest-earning years |
| **Widow(er) benefits** | Increase survivor benefits to 75% of combined couple benefit |
| **Combined benefit rule** | Improve treatment of two-earner couples |

**Evidence:**

| Finding | Source |
|---------|--------|
| Women's benefits 77% of men's | SSA data |
| Caregivers lose $300,000+ lifetime | AARP research |
| Widow poverty rate 14.5% vs. 4.5% married | Census |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Cost | Phased implementation; revenue increase |
| Targeting | Caregiver credits require verification |
| Precedent | Similar credits exist internationally |

### Solution 3: Modernize Supplemental Security Income (SSI)

**Description**: Comprehensively reform SSI to provide adequate income support for the poorest elderly, including raising benefit levels, updating asset limits, and simplifying enrollment.

**Key Components:**

| Component | Current | Proposed |
|-----------|---------|----------|
| **Federal benefit level** | $943/month (75% FPL) | $1,255/month (100% FPL) |
| **Asset limit (individual)** | $2,000 (since 1989) | $10,000 (indexed) |
| **Asset limit (couple)** | $3,000 | $20,000 (indexed) |
| **In-kind support rules** | Complex deductions | Simplified; food/shelter excluded |
| **Application** | 16-page form | Streamlined; presumptive eligibility |
| **Resource exclusions** | Limited | Include all retirement accounts; vehicles |

**Evidence:**

| Finding | Source |
|---------|--------|
| SSI participation rate only 55% | CBPP |
| Inflation-adjusted asset limit would be $10,000+ | CPI calculation |
| 1 million elderly eligible but not enrolled | HHS estimate |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Budget cost | $10-15B annually for full reform |
| SSA administration | Simplification reduces workload |
| Political framing | Emphasis on modernization, not expansion |

### Solution 4: Reduce Medicare Out-of-Pocket Costs

**Description**: Cap Medicare beneficiary out-of-pocket costs, add dental/vision/hearing coverage, and reduce prescription drug costs to prevent medical expenses from driving seniors into poverty.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Out-of-pocket cap** | $2,500/year maximum for Part B/D |
| **Dental benefit** | Basic preventive and restorative coverage |
| **Vision benefit** | Exams and glasses coverage |
| **Hearing benefit** | Exams and hearing aids coverage |
| **Part D improvements** | Expand drug price negotiation; eliminate donut hole |

**Evidence:**

| Finding | Source |
|---------|--------|
| Average Medicare OOP $6,800/year | CMS data |
| 70% of seniors lack dental coverage | KFF |
| Dental/vision/hearing would add $700+/beneficiary | CBO estimates |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Cost | Offset with drug negotiation savings |
| Provider network | Phase in benefits; build capacity |
| Industry opposition | Strong public support |

### Solution 5: Housing Affordability for Seniors

**Description**: Expand affordable housing options for low-income seniors through increased Section 8 vouchers, senior housing development, and property tax relief programs.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Section 8 expansion** | 500,000 new vouchers targeted to seniors |
| **Senior housing development** | $5B annual investment in affordable senior housing |
| **Property tax relief** | Federal matching for state circuit breakers |
| **Home modification** | Grants for aging-in-place modifications |
| **Mobile home protection** | Rent stabilization; anti-displacement |

**Evidence:**

| Finding | Source |
|---------|--------|
| 54% of senior renters cost-burdened | Harvard JCHS |
| 2-10 year waitlists for Section 8 | HUD data |
| Every $1 housing assistance reduces Medicaid $0.63 | Enterprise research |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Federal budget | Link to healthcare savings |
| Housing supply | Combine with development incentives |
| Local opposition | Community engagement; senior focus |

### Solution 6: Universal Retirement Savings

**Description**: Establish universal access to retirement savings through auto-IRA programs, expanded 401(k) coverage requirements, and federal matching contributions for low-income workers.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Auto-IRA requirement** | All employers without plans must offer payroll-deduction IRA |
| **Federal match** | 50% match up to $1,000/year for workers under 150% FPL |
| **401(k) expansion** | Require all firms with 10+ employees to offer plan |
| **Auto-enrollment** | Default enrollment at 6% with escalation |
| **Leakage reduction** | Limit early withdrawals; emergency savings tier |

**Evidence:**

| Finding | Source |
|---------|--------|
| 55 million workers lack plan access | EBRI |
| Auto-enrollment increases participation 50+ percentage points | Research |
| State auto-IRAs (CA, OR) showing success | State reports |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Employer burden | Phase in; exempt smallest businesses |
| Match cost | Target to lowest earners |
| Doesn't help current seniors | Complement other reforms |

### Solution 7: Improve Food Security

**Description**: Increase SNAP benefits and participation for seniors through benefit increases, application simplification, and targeted outreach.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Benefit increase** | Raise minimum benefit to $50/month |
| **Medical deduction expansion** | Allow full medical expense deduction |
| **Simplified application** | Combined Social Security/SNAP application |
| **Eliminate asset tests** | Remove asset limit for seniors |
| **Outreach funding** | $100M for senior SNAP outreach |

**Evidence:**

| Finding | Source |
|---------|--------|
| Only 48% of eligible seniors participate | USDA |
| 5.5 million seniors food insecure | Feeding America |
| SNAP reduces food insecurity 30%+ | Research |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Budget cost | Modest relative to other programs |
| Stigma | Outreach; automatic enrollment |
| State variation | Federal minimum standards |

### Solution 8: Medicare Savings Program Expansion

**Description**: Expand and simplify Medicare Savings Programs to help more low-income seniors afford Medicare premiums and cost-sharing.

**Key Components:**

| Component | Current | Proposed |
|-----------|---------|----------|
| **QMB income limit** | 100% FPL | 150% FPL |
| **Asset limit** | $9,090 | Eliminated |
| **Application** | Separate from Medicare | Auto-enrollment with LIS |
| **Coordination** | Complex | Streamlined with Medicaid |

**Evidence:**

| Finding | Source |
|---------|--------|
| 2 million more eligible than enrolled | CMS |
| Low-income seniors pay higher share of income for Medicare | KFF |
| Auto-enrollment increases participation 20+ percentage points | Research |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Federal-state coordination | Federal minimum; state flexibility |
| Data sharing | SSA/CMS coordination |
| Cost | Moderate increase |

### Solution 9: Age Discrimination Enforcement

**Description**: Strengthen enforcement of age discrimination laws to protect older workers' earnings and employment opportunities.

**Key Components:**

| Component | Details |
|-----------|---------|
| **ADEA strengthening** | Restore disparate impact liability |
| **EEOC funding** | Double age discrimination investigation funding |
| **Burden of proof** | Clarify plaintiff-friendly standards |
| **Damages** | Allow compensatory and punitive damages |
| **Hiring protections** | Prohibit age inquiries in hiring |

**Evidence:**

| Finding | Source |
|---------|--------|

| Average unemployment duration for 55+ double younger workers | BLS |
| Age discrimination costs older workers $850B+ annually | Estimate |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Business opposition | Frame as productivity issue |
| Enforcement capacity | Phased EEOC expansion |
| Proof difficulties | Burden shifting provisions |

### Solution 10: Pension Protection and Enhancement

**Description**: Strengthen pension protections and create options to convert 401(k) balances into lifetime income.

**Key Components:**

| Component | Details |
|-----------|---------|
| **PBGC strengthening** | Increase premium; improve solvency |
| **Multiemployer plan rescue** | Continued funding for troubled plans |
| **Annuity options** | Require 401(k) plans to offer lifetime income option |
| **Fiduciary rules** | Strengthen protection against conflicts |
| **Lifetime income illustration** | Require monthly income estimates on statements |

**Evidence:**

| Finding | Source |
|---------|--------|
| 1.3 million workers affected by multiemployer plan failures | PBGC |
| Only 6% of 401(k) participants annuitize | Research |
| Pension income reduces poverty risk 75% vs. savings alone | Research |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Annuity complexity | Default options; safe harbor |
| Industry resistance | Consumer protection framing |
| Multiemployer cost | Already enacted (ARP 2021) |

## International Models

### Netherlands: Quasi-Mandatory Pensions

| Feature | Description |
|---------|-------------|
| **Basic pension** | Universal flat-rate pension at minimum wage level |
| **Occupational pensions** | 90%+ coverage through sectoral agreements |
| **DC with pooling** | Risk-sharing within generation |
| **Result** | 3% elderly poverty rate |

### Denmark: Multipillar System

| Feature | Description |
|---------|-------------|
| **Public pension** | Universal basic + means-tested supplement |
| **Occupational** | Mandatory for most workers |
| **Private** | Tax-advantaged savings |
| **Result** | 4% elderly poverty rate |

### Canada: Targeted Enhancement

| Feature | Description |
|---------|-------------|
| **OAS** | Universal pension for all residents |
| **GIS** | Means-tested top-up for low income |
| **CPP** | Earnings-based social insurance |
| **Recent reform** | CPP expansion (gradual) |
| **Result** | 12% elderly poverty (improving) |

### Key Lessons

| Lesson | Application to US |
|--------|-------------------|
| Universal foundation essential | Strengthen Social Security |
| Automatic enrollment works | Apply to retirement savings |
| Risk pooling improves outcomes | Move toward collective DC models |
| Means-tested supplements help poorest | Modernize SSI |
| Multiple pillars provide security | Don't rely on single leg |

## Sequencing

### Immediate Priorities (2025-2026)

| Reform | Rationale |
|--------|-----------|
| **SSI modernization** | Most vulnerable; administrative fix largely |
| **Medicare Savings Program expansion** | High-impact; relatively simple |
| **Social Security solvency** | Must address trust fund; opportunity for improvements |
| **SNAP senior participation** | Existing program; outreach |

### Near-Term (2027-2029)

| Reform | Rationale |
|--------|-----------|
| **Social Security minimum benefit** | Builds on solvency legislation |
| **Medicare dental/vision/hearing** | ACA precedent; high demand |
| **Housing voucher expansion** | Budget priority |
| **Universal auto-IRA** | State momentum |

### Medium-Term (2030-2035)

| Reform | Rationale |
|--------|-----------|
| **Comprehensive Social Security equity** | Requires extended implementation |
| **Medicare out-of-pocket cap** | Budget implications |
| **Universal retirement coverage** | Employer adjustment needed |
| **Age discrimination reform** | Legal complexity |
