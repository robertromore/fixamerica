# Senior Poverty and Economic Security: Implementation Roadmap

## Strategic Approach

Reducing senior poverty requires sustained action across multiple fronts: strengthening the Social Security foundation, modernizing safety net programs, reducing healthcare costs, and preventing future poverty through improved retirement savings. The strategy recognizes that the Social Security trust fund situation creates both urgency and opportunityâ€”solvency legislation must pass, and that legislation can include benefit improvements. The approach sequences reforms to achieve early wins (SSI modernization, SNAP outreach) while building toward comprehensive reform.

### Guiding Principles

| Principle | Application |
|-----------|-------------|
| **Solvency as opportunity** | Use trust fund deadline to include benefit improvements |
| **Start with most vulnerable** | SSI modernization helps poorest; politically viable |
| **Administrative before legislative** | Maximize executive action; reduce barriers |
| **Build coalition** | Connect senior poverty to broader economic security agenda |
| **Demonstrate impact** | Evaluate and publicize results to build momentum |

### Political Context

| Factor | Assessment |
|--------|------------|
| **Public support** | Strong (70%+ support strengthening Social Security) |
| **Trust fund deadline** | Creates pressure for action by ~2033 |
| **Partisan landscape** | Social Security historically bipartisan; currently polarized |
| **Budget constraints** | Deficit concerns complicate new spending |
| **Advocacy capacity** | Strong (AARP, unions, advocacy groups) |
| **Window of opportunity** | Solvency deadline creates action-forcing event |

## Phase 2: Social Security Solvency and Improvement (2027-2030)

### Social Security Legislation Package

**Must Pass by ~2033**: The trust fund deadline creates action-forcing event. The strategy is to include benefit improvements in solvency legislation.

| Component | Details |
|-----------|---------|
| **Revenue**: Lift payroll cap | Apply tax to earnings over $400K |
| **Revenue**: Investment income contribution | Apply 12.4% to investment income over $200K |
| **Benefit**: Minimum benefit | 125% FPL for 30+ years work |
| **Benefit**: CPI-E adoption | Better COLA for seniors |
| **Benefit**: Widow(er) enhancement | 75% of combined benefit |
| **Equity**: Caregiver credits | Up to 5 years credited |
| **Equity**: Calculation period | 35 years to 30 years |

## Risk Mitigation

### Political Risks

| Risk | Likelihood | Mitigation |
|------|------------|------------|
| **Social Security cut-only approach** | Medium | Coalition advocacy; public opinion |
| **Administration change** | Medium | Embed in statute; state action |
| **Budget sequestration** | Medium | Exempt programs; coalition advocacy |
| **Industry lobbying success** | Medium | Public awareness; transparency |
| **Partisan gridlock** | High | Focus on bipartisan elements; state action |

### Implementation Risks

| Risk | Likelihood | Mitigation |
|------|------------|------------|
| **SSA administrative capacity** | Medium | Phase implementation; increase funding |
| **Outreach challenges** | Medium | Partner with community organizations |
| **Benefit formula complexity** | Low | Clear implementation guidelines |
| **Healthcare provider networks** | Medium | Phase in Medicare benefits |

### External Risks

| Risk | Likelihood | Mitigation |
|------|------------|------------|
| **Recession** | Medium | Automatic stabilizers; contingency planning |
| **Inflation** | Medium | COLA protections |
| **Healthcare cost growth** | High | Continued cost containment |
| **Demographic acceleration** | Certain | Plan for growth |

## Contingency Plans

### If Social Security Legislation Stalls

| Scenario | Response |
|----------|----------|
| Solvency-only passes | Pursue benefit improvements separately |
| No legislation before deadline | Across-the-board cuts create crisis/opportunity |
| Revenue-only blocked | Benefit restructuring with progressive cuts |

### If Administrative Actions Blocked

| Scenario | Response |
|----------|----------|
| New administration reverses | Seek legislative codification |
| Agency capacity insufficient | Prioritize highest-impact actions |
| Litigation challenges | Defend aggressively; legislative backup |

### If State Action Leads

| Scenario | Opportunity |
|----------|-------------|
| State SSI supplements grow | Document impact; build federal case |
| State auto-IRAs succeed | Push federal action |
| State age discrimination laws | Create momentum |
