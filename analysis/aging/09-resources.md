# Aging Policy: Resources

## Overview

This section compiles essential resources for understanding aging policy, including government data, academic research, advocacy organizations, and current journalism.

### Federal Agencies

| Agency | Key Resources |
|--------|---------------|
| **Centers for Medicare & Medicaid Services (CMS)** | Medicare data, Medicaid statistics, nursing home compare |
| **Social Security Administration (SSA)** | Benefit data, actuarial reports, program statistics |

| **Census Bureau** | Population aging data, income statistics |
