# Elder Abuse and Neglect: Citizen Actions

## For Individuals Concerned About an Elder

### Recognizing Warning Signs

| Category | Warning Signs |
|----------|---------------|
| **Physical abuse** | Unexplained injuries, bruises, burns; fear of caregiver; reluctance to talk openly |
| **Emotional abuse** | Withdrawal, depression, anxiety; caregiver berates or threatens elder |
| **Financial exploitation** | Sudden changes in finances; new "friends" controlling money; unexplained transfers |
| **Neglect** | Poor hygiene, malnutrition, dehydration; untreated medical conditions; unsafe living conditions |
| **Self-neglect** | Inability to manage daily tasks; hoarding; refusing needed care |

### If You Suspect Abuse

| Step | Action | Details |
|------|--------|---------|
| 1 | **Document concerns** | Note dates, observations, statements |
| 2 | **Talk privately** | If safe, speak with elder away from potential abuser |
| 3 | **Report to APS** | Call state Adult Protective Services or 1-800-677-1116 |
| 4 | **Call 911** | If immediate danger, call emergency services |
| 5 | **Consult professionals** | Elder law attorney, geriatric care manager, or ombudsman |

### Reporting Resources

| Resource | Contact | When to Use |
|----------|---------|-------------|
| **Eldercare Locator** | 1-800-677-1116 | Connects to local APS |
| **Long-term care ombudsman** | Via Eldercare Locator | Nursing home concerns |
| **Local police** | Non-emergency line | To file report |
| **CFPB complaint** | consumerfinance.gov | Financial exploitation |
| **State attorney general** | State-specific | Consumer fraud |

## For Family Members and Caregivers

### Prevention Strategies

| Strategy | Implementation |
|----------|----------------|
| **Stay connected** | Regular visits, calls, video chats |
| **Know the finances** | Understand accounts, income, expenses |
| **Establish trusted contacts** | Designate someone for financial institutions |
| **Create legal documents** | Powers of attorney, advance directives while elder has capacity |
| **Build support network** | Involve multiple family members, friends |
| **Use technology** | Monitoring with consent, video calls |

### Managing Caregiver Stress

| Action | Resources |
|--------|-----------|
| **Seek respite** | Adult day programs, in-home respite, short-term facility stays |
| **Join support groups** | Caregiver support groups, online communities |
| **Get training** | Caregiver education programs, disease-specific training |
| **Address your health** | Doctor visits, mental health support, self-care |
| **Ask for help** | Family, friends, community resources |

### Financial Protection Steps

| Step | Action |
|------|--------|
| 1 | Review accounts regularly with elder's permission |
| 2 | Set up account alerts for unusual activity |
| 3 | Establish trusted contact with financial institutions |
| 4 | Consider daily withdrawal limits |
| 5 | Register with do-not-call lists |
| 6 | Monitor mail for scam indicators |
| 7 | Review statements and bills together |

## For Healthcare Providers

### Clinical Actions

| Action | Details |
|--------|---------|
| **Screen routinely** | Use validated tools (EASI, H-S/EAST) |
| **Document thoroughly** | Injuries, statements, observations |
| **Report suspected abuse** | Follow mandatory reporting laws |
| **Treat the patient** | Address immediate medical needs |
| **Connect to services** | APS, social work, community resources |
| **Follow up** | Schedule return visits, check on safety |

### Screening Best Practices

| Practice | Implementation |
|----------|----------------|
| **Private interview** | Speak with patient alone |
| **Ask direct questions** | "Has anyone hurt you?" "Are you afraid?" |
| **Look for patterns** | Multiple ER visits, inconsistent histories |
| **Assess capacity** | Cognitive status affects intervention options |
| **Document carefully** | Body maps, photographs with consent |

## For Financial Professionals

### Detection and Response

| Action | Implementation |
|--------|----------------|
| **Know warning signs** | Unusual transactions, new "helpers," confusion |
| **Use trusted contacts** | Contact designated person when concerned |
| **Consider transaction delays** | Hold suspicious transactions when permitted |
| **Report to APS** | Mandatory or permitted depending on state |
| **File SARs** | Suspicious Activity Reports when required |
| **Train staff** | Recognition of exploitation, response protocols |

### Customer Protection

| Protection | How to Implement |
|------------|------------------|
| **Trusted contact designation** | Offer to all elder customers |
| **Account alerts** | Help customers set up notifications |
| **Large withdrawal protocols** | Cooling-off periods, verification |
| **Scam awareness education** | Workshops, materials |
| **Simplified statements** | Easier for elders to review |

## For Legal Professionals

### Client Protection

| Action | Details |
|--------|---------|
| **Assess capacity** | Determine if client can make informed decisions |
| **Meet privately** | Separate client from potential influencers |
| **Watch for undue influence** | New "friends," sudden changes in documents |
| **Report abuse** | Ethical obligations vary by state |
| **Connect to resources** | APS, law enforcement, social services |

### Proactive Planning

| Document | Purpose |
|----------|---------|
| **Durable power of attorney** | Financial decision-making |
| **Healthcare proxy** | Medical decision-making |
| **Living will** | End-of-life preferences |
| **Trusts** | Asset protection |
| **Beneficiary designations** | Review and update regularly |
