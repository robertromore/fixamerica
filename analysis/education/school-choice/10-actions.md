# School Choice: Actions

## Overview

This document provides actionable guidance for citizens who want to engage constructively with school choice issues in their communities. Whether you support, oppose, or want to improve school choice policies, effective civic action requires information, strategy, and sustained engagement.

## Understand Your Local Context

### Research Your Community

Before taking action, understand how school choice operates where you live.

**Questions to Answer**

| Question | Where to Find Answer |
|----------|---------------------|
| What choice options exist? | State education agency website |
| How many students use choice? | District enrollment data |
| What is the fiscal impact? | District budget documents |
| Who authorizes charters? | State charter law |
| What voucher programs exist? | State education agency |
| What are integration levels? | CRDC data, district reports |
| What is charter performance? | State report cards |

**Information Sources**

| Source | Content |
|--------|---------|
| State education agency | Program information, data |
| Local school district | Enrollment, budget, facilities |
| School board meetings | Policy decisions, public comment |
| Local journalism | Coverage of local issues |
| Parent networks | Ground-level experiences |

### Assess the Situation

| Factor | Indicators |
|--------|------------|
| Choice intensity | % of students in choice schools |
| Quality variation | Performance differences across sectors |
| Segregation patterns | Demographic data by school |
| Fiscal stress | District budget trends |
| Accountability gaps | Reporting requirements by sector |
| Access equity | Who is using choice |

## Issue-Specific Actions

### Promoting Accountability

**Goals**

- Equal standards across all publicly funded schools
- Transparent reporting of outcomes
- Consequences for poor performance

**Actions**

| Action | Target |
|--------|--------|
| Support accountability legislation | State legislature |
| Testify for testing requirements | Legislative hearings |
| Request data from all schools | School boards, state agencies |
| Publicize performance differences | Media, community forums |
| Support quality authorizers | Authorizer elections/appointments |

**Talking Points**

- "All schools receiving public funds should meet public standards"
- "Parents deserve comparable information across all options"
- "Accountability protects children and taxpayers"

### Promoting Integration

**Goals**

- Diverse schools across all sectors
- Access supports for underserved families
- Transportation equity

**Actions**

| Action | Target |
|--------|--------|
| Advocate for controlled choice | School boards, districts |
| Support diversity lottery preferences | Charter authorizers |
| Demand transportation equity | Districts, state legislatures |
| Highlight segregation data | Media, community forums |
| Support integration incentives | State and federal policy |

**Talking Points**

- "Diverse schools benefit all students"
- "Choice without access supports privileged families"
- "Integration requires intentional policy"

### Protecting Public Schools

**Goals**

- Adequate funding for traditional public schools
- Fair competition between sectors
- Community school investment

**Actions**

| Action | Target |
|--------|--------|
| Advocate for hold-harmless funding | State legislature |
| Support community school programs | School boards, state agencies |
| Highlight fiscal impacts | Media, community forums |
| Oppose unfunded mandates | Legislative advocacy |
| Elect supportive school board members | Local elections |

**Talking Points**

- "All students deserve excellent schools, including those in neighborhood schools"
- "Funding should follow actual costs, not just student counts"
- "Community schools address whole child needs"

### Improving Choice Quality

**Goals**

- Close low-performing choice schools
- Strengthen authorization
- Support high-quality options

**Actions**

| Action | Target |
|--------|--------|
| Support authorizer quality standards | State legislature, authorizers |
| Advocate for closure of failing schools | Authorizers, state agencies |
| Highlight quality variation | Media, policy forums |
| Support replication of successful models | Policy advocacy |
| Demand performance data | Choice operators, authorizers |

**Talking Points**

- "Choice only works if options are good"
- "Low-performing schools harm children regardless of sector"
- "Quality authorization protects families"

## Electoral Engagement

### School Board Elections

| Action | Timing |
|--------|--------|
| Research candidates | Before filing deadline |
| Attend candidate forums | Campaign period |
| Ask about choice positions | Direct questions |
| Support aligned candidates | Donations, volunteering |
| Vote | Election day |
| Mobilize others | GOTV efforts |

**Questions for Candidates**

- What is your position on charter school authorization?
- How should voucher/ESA programs be regulated?
- What should happen when schools underperform?
- How should we address segregation in school choice?
- How can we ensure equitable access to choice?

### State Elections

| Action | Focus |
|--------|-------|
| Identify key races | Governor, legislature, state board |
| Research candidate positions | Voting records, statements |
| Engage with campaigns | Ask questions, provide input |
| Support aligned candidates | Donations, volunteering |
| Vote | Primary and general elections |

### Federal Elections

| Action | Focus |
|--------|-------|
| Understand federal role | Limited but influential |
| Research candidate positions | Charter grants, voucher support |
| Engage during primaries | When positions are formed |
| Vote | Every election matters |

## Building Coalitions

### Finding Common Ground

| Issue | Potential Allies |
|-------|------------------|
| Accountability | Parents, taxpayer groups, quality operators |
| Transparency | Media, researchers, parent groups |
| Access equity | Civil rights groups, poverty advocates |
| Integration | Civil rights groups, researchers |
| Quality | All stakeholders should agree |

### Coalition Strategies

| Strategy | Application |
|----------|-------------|
| Focus on shared goals | Quality, equity, accountability |
| Avoid ideological framing | Evidence-based arguments |
| Include diverse voices | Credibility across perspectives |
| Build personal relationships | Trust before action |
| Celebrate small wins | Maintain momentum |

## Effective Communication

### Writing Letters and Emails

| Element | Guidance |
|---------|----------|
| Be specific | Name the policy or decision |
| Be personal | Share your story |
| Be factual | Use evidence |
| Be respectful | Even in disagreement |
| Request action | Clear ask |
| Follow up | Persistence matters |

### Public Testimony

| Stage | Guidance |
|-------|----------|
| Preparation | Research the issue, know your audience |
| Content | Personal story + policy point |
| Delivery | Calm, clear, within time limits |
| Follow-up | Written version, continued engagement |

### Media Engagement

| Activity | Guidance |
|----------|----------|
| Letters to editor | Short, timely, personal |
| Op-eds | Expertise, clear argument, new angle |
| Interviews | Prepare key messages, stay on point |
| Social media | Consistent, evidence-based, engaging |
