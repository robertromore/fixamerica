# Education Equity: Actions

## Overview

Citizens can take meaningful action to advance education equity at multiple levels. This guide provides concrete steps for individuals and groups based on their circumstances, resources, and community contexts. Actions range from personal choices to community organizing to political engagement.

## Individual Actions

### As a Parent

**School Choice Decisions**

| Action | Description | Impact |
|--------|-------------|--------|
| Choose diverse schools | Select integrated schools when available | Directly supports integration |
| Stay in public schools | Resist private school exit | Maintains investment in public education |
| Support magnets/choice | Participate in voluntary integration programs | Creates demand for integration |
| Avoid opportunity hoarding | Don't exploit system for unfair advantage | Reduces stratification |

**Within Your Child's School**

| Action | Description |
|--------|-------------|
| Support inclusive policies | Advocate for welcoming all students |
| Question tracking decisions | Challenge biased placement |
| Share resources | Contribute to school-wide benefit, not just own child |
| Build cross-group relationships | Model inclusive community |

**PTA/Parent Organizations**

| Action | Impact |
|--------|--------|
| Raise equity concerns | Put issues on agenda |
| Redirect fundraising | Support school-wide rather than exclusive programs |
| Challenge exclusive practices | Oppose policies that advantage some students |
| Build diverse leadership | Ensure parent groups represent whole community |

### As an Educator

**Classroom Practices**

| Action | Description |
|--------|-------------|
| Maintain high expectations | Believe all students can succeed |
| Examine biases | Reflect on differential treatment |
| Culturally responsive teaching | Connect curriculum to student experiences |
| Fair discipline | Use restorative rather than exclusionary approaches |

**School-Level Advocacy**

| Action | Description |
|--------|-------------|
| Challenge tracking | Question ability grouping |
| Support detracking | Advocate for heterogeneous classes |
| Improve identification | Push for equitable gifted/AP access |
| Reform discipline | Support restorative practices |

**Professional Engagement**

| Action | Description |
|--------|-------------|
| Union advocacy | Push union toward equity positions |
| Professional development | Seek and share equity training |
| Mentoring | Support teachers in high-need schools |
| Career choices | Consider teaching in high-need schools |
