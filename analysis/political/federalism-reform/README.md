# Federalism Reform: Overview

Federalism reform encompasses efforts to clarify, balance, and improve the relationship between federal and state governments. This includes addressing federal preemption of state laws, unfunded mandates, grant conditions, interstate cooperation, and the appropriate division of governmental responsibilities in a modern context.
