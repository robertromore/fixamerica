# Media and Information: Actions

## For Individuals

### Support Quality Journalism

**Direct Financial Support**

| Action | Impact |
|--------|--------|
| Subscribe to local newspaper | Revenue for local journalism |
| Donate to nonprofit news | Support independent reporting |
| Support public radio/TV | Strengthen public media |
| Pay for quality national news | Fund investigative journalism |

**What to Support**

| Priority | Example |
|----------|---------|
| Local first | Your community's news sources |
| Nonprofit news | ProPublica, local nonprofits |
| Investigative | Organizations doing accountability journalism |
| Public media | Local NPR/PBS stations |

### Practice Good Information Hygiene

**Consumption Habits**

| Practice | How |
|----------|-----|
| Diversify sources | Read across perspectives |
| Check before sharing | Verify before posting |
| Seek local news | Prioritize community information |
| Recognize emotion manipulation | Pause when content makes you angry |

**Verification Steps**

| Step | Action |
|------|--------|
| Check the source | Is it credible? |
| Read beyond headlines | Full context |
| Check dates | Is it current? |
| Find original sources | Primary documents |
| Consult fact-checkers | PolitiFact, Snopes |

### Reduce Platform Manipulation

**Social Media Practices**

| Action | Effect |
|--------|--------|
| Adjust algorithm settings | Control what you see |
| Follow quality sources | Shape your feed |
| Don't amplify misinformation | Even to criticize |
| Report false content | Use platform tools |
| Take breaks | Reduce manipulation exposure |

## Getting Started

### Individual First Steps

**Today**

1. Subscribe to your local news source
2. Verify before sharing on social media
3. Adjust social media settings for control

**This Month**

1. Donate to a nonprofit newsroom
2. Contact representative about local news legislation
3. Learn media literacy basics

### Community First Steps

**This Quarter**

1. Assess local news coverage gaps
2. Connect with journalism support organizations
3. Start community media literacy program

### Organizational First Steps

**This Year**

1. Develop media support policy
2. Commit resources to local journalism
3. Join advocacy coalition
