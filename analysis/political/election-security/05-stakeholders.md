# Election Security: Stakeholders

## Who Benefits from Strong Election Security

### All Voters

**Fundamental Interests**

| Benefit | Description |
|---------|-------------|
| Vote counted as cast | Confidence in ballot recording |
| No manipulation | Results reflect actual preferences |
| Verified outcomes | Ability to audit and confirm |
| Informed participation | Protection from misinformation |

**Specific Voter Groups**

| Group | Security Interest |
|-------|-------------------|
| First-time voters | Trust in system legitimacy |
| Minority voters | Protection from targeted suppression |
| Overseas/military | Secure transmission of ballots |
| Elderly voters | Protection from fraud schemes |
| All voters | Confidence in democratic process |

### Candidates and Campaigns

**Legitimate Interest in Fair Elections**

| Stakeholder | Interest |
|-------------|----------|
| Winning candidates | Mandate legitimacy |
| Losing candidates | Fair shot, accurate count |
| All candidates | Level playing field |
| Campaign staff | Cybersecurity for operations |

### Election Officials

**Professional Stakes**

| Interest | Description |
|----------|-------------|
| Successful elections | Professional reputation |
| Defensible processes | Protection from false claims |
| Resources for job | Equipment, staff, training |
| Personal safety | Protection from threats |

### Government Institutions

**Systemic Interests**

| Institution | Stake |
|-------------|-------|
| Federal government | National security, legitimacy |
| State governments | Election administration authority |
| Local governments | Community trust |
| Courts | Orderly dispute resolution |

### Democracy Itself

**Foundational Requirements**

| Need | Connection to Security |
|------|----------------------|
| Legitimate government | Secure elections confer legitimacy |
| Peaceful transitions | Trust prevents unrest |
| Representation | Accurate results enable representation |
| Public confidence | Security maintains trust |

## Who Is Harmed by Security Failures

### Voters in Compromised Jurisdictions

**Direct Harms**

| Harm | Impact |
|------|--------|
| Manipulated votes | Will of voters overridden |
| Database breaches | Personal data exposed |
| Registration manipulation | Disenfranchisement |
| Misinformation targeting | Deceived voters |

### Election Workers

**Professional and Personal Costs**

| Harm | Impact |
|------|--------|
| Blamed for failures | Reputation damage |
| Security incidents | Career consequences |
| Harassment campaigns | Personal safety threats |
| Burnout | Overwhelming demands |

### Candidates Who Lose Unfairly

**Campaign Impacts**

| Harm | Description |
|------|-------------|
| Stolen elections | Legitimate winner loses |
| Compromised campaigns | Operations disrupted |
| Reputation damage | False claims of cheating |
| Resource waste | Investment in fraudulent contest |

### Public Trust

**Societal Harms**

| Harm | Effect |
|------|--------|
| Delegitimized winners | Governing authority undermined |
| Civic disengagement | Why vote if it's rigged? |
| Political violence | January 6 as example |
| International standing | Democracy model damaged |

## Who Has Power to Change Things

### Federal Government

**CISA (Cybersecurity and Infrastructure Security Agency)**

| Power | Limitation |
|-------|------------|
| Free security services | Cannot mandate use |
| Threat intelligence | Sharing only |
| Incident response | Requires invitation |
| Standards guidance | Voluntary compliance |

**Election Assistance Commission (EAC)**

| Power | Limitation |
|-------|------------|
| Voting system certification | Voluntary |
| Best practices | Advisory only |
| Grant administration | Limited funding |
| Testing standards | States choose whether to use |

**Congress**

| Power | Limitation |
|-------|------------|
| Funding appropriations | Political disagreements |
| Federal election standards | Constitutional limits |
| Oversight | Partisan dynamics |
| CISA/EAC authorization | Subject to politics |

**Department of Justice**

| Power | Limitation |
|-------|------------|
| Investigate interference | Resource constraints |
| Prosecute violations | Jurisdictional limits |
| Foreign actor indictments | Limited enforcement ability |
| Civil rights enforcement | Political interference |

### State Governments

**State Election Officials**

| Power | Variation |
|-------|-----------|
| Voting equipment certification | Standards differ by state |
| Election procedures | Wide latitude |
| County oversight | Varies by structure |
| Security mandates | Can require standards |

**State Legislatures**

| Power | Consideration |
|-------|---------------|
| Election law | Full authority |
| Funding appropriation | Budget constraints |
| Standards setting | Technical expertise limits |
| Local preemption | Some states restrict local innovation |

### Local Officials

**County/Municipal Election Administrators**

| Power | Constraint |
|-------|------------|
| Day-to-day operations | State law compliance |
| Equipment selection | From certified options |
| Security implementation | Budget limits |
| Worker training | Time and resources |

### Voting Equipment Vendors

**Industry Power**

| Power | Issue |
|-------|-------|
| Technology design | Proprietary control |
| Pricing | Limited competition |
| Maintenance | Dependency relationship |
| Lobbying | Influence over standards |

**Major Vendors**

| Company | Market Share | Issues |
|---------|--------------|--------|
| ES&S | ~50% | Market dominance |
| Dominion | ~30% | Target of conspiracy theories |
| Hart InterCivic | ~15% | Regional presence |
| Others | ~5% | Limited market access |

| CISA | Federal security services |

## Stakeholder Engagement Strategy

### Building Support

| Stakeholder | Engagement Approach |
|-------------|---------------------|
| Election officials | Professional development, resources |
| State legislators | Education, model legislation |
| Congress | Bipartisan framing, constituent pressure |
| Public | Simplified messaging, transparency |
| Media | Story support, expert access |

### Neutralizing Opposition

| Opponent | Strategy |
|----------|----------|
| Partisan resisters | Depoliticize, bipartisan validators |
| Vendor resistance | Transparency requirements, competition |
| Cost objectors | Federal funding solutions |
| Election deniers | Isolate, don't legitimize |
