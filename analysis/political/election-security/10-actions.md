# Election Security: Actions

## What You Can Do

Election security requires action at every level - from individual citizens to national policymakers. This guide provides specific, actionable steps for different audiences and engagement levels.

## Individual Actions

### Learn About Your Local Election Systems

**Know Your Equipment**

| Action | How To |
|--------|--------|
| Check what voting equipment is used | Visit verifiedvoting.org/verifier |
| Learn how votes are counted | Contact local election office |
| Understand audit procedures | Check state election website |
| Know ballot security measures | Review chain of custody policies |

**Verify Your Registration**

| Action | How To |
|--------|--------|
| Check registration status | Use state voter lookup tool |
| Update information regularly | Before registration deadlines |
| Sign up for alerts | Some states offer notifications |
| Keep records | Document any changes or issues |

### Sample Messages

**For Federal Legislators**

```text
Subject: Support Election Security Legislation

Dear [Senator/Representative],

I am writing to urge you to support strong election security measures,
including:

1. Mandatory paper ballots for all federal elections
2. Required risk-limiting audits before certification
3. Increased federal funding for election security
4. Enhanced protection for election officials

Our democracy depends on secure elections that voters can trust. Please
prioritize election security as a matter of national security.

Sincerely,
[Your name]
[Address]
```

**For State Officials**

```text
Subject: Strengthen [State] Election Security

Dear [Official],

I am a [State] voter concerned about election security. I urge you to
support:

1. Paper ballot requirements for all elections
2. Risk-limiting audits conducted before certification
3. Adequate funding for county election offices
4. Training and protection for election workers

Thank you for your attention to this critical issue.

Sincerely,
[Your name]
[Address]
```
