| Barrier | Effect |

### Primary System

**Primary Turnout**

| Type | Typical Turnout |
|------|-----------------|
| Presidential primary | 15-30% of party |
| Congressional primary | 10-20% of party |
| Primary voters | More extreme than general electorate |

**Primary Types by State**

| Type | States | Effect |
|------|--------|--------|
| Closed | ~15 | Only party members vote |
| Open | ~20 | Any voter can participate |
| Semi-open | ~10 | Various rules |
| Top-two/nonpartisan | ~5 | All candidates on one ballot |

## Impact

### On Representation

**Voter Alienation**

| Group | Concern |
|-------|---------|
| Moderates | Neither party represents them |
| Issue voters | Forced to accept package deal |
| Independents | Limited influence on nominations |
| Young voters | Disconnection from party system |

**Policy Misrepresentation**

| Issue | Public View | Party Positions |
|-------|-------------|-----------------|
| Immigration | Mixed, nuanced | Binary extremes |
| Gun policy | Moderate consensus | Polarized positions |
| Climate | Majority support action | Partisan divide |
| Healthcare | Universal coverage popular | Partisan approaches |

### On Governance

**Gridlock**

| Phenomenon | Effect |
|------------|--------|
| Divided government | Common, produces stalemate |
| Must-pass bills | Only legislation that moves |
| Long-term problems | Unaddressed |
| Policy swings | Whiplash with party changes |

**Incentive Structure**

| Incentive | Behavior |
|-----------|----------|
| Primary threat | Move to extremes |
| Base mobilization | Demonize opposition |
| Negative partisanship | Attack > propose |
| Media incentives | Conflict over substance |

### On Society

**Partisan Identity**

| Phenomenon | Measurement |
|------------|-------------|
| Affective polarization | Dislike of other party at historic highs |
| Social distance | Less willing to interact across party |
| Family/friend conflict | Politics strains relationships |
| Geographic clustering | Self-sorting into partisan communities |

## Current Regulations

### Ballot Access Laws

**State Requirements**

| Requirement | Range |
|-------------|-------|
| Signature collection | 10,000 - 200,000+ |
| Filing fees | $0 - $10,000+ |
| Deadlines | Months before election |
| Maintenance | Ongoing vote thresholds |

**Major vs. Third Party**

| Treatment | Major Party | Third Party |
|-----------|-------------|-------------|
| Automatic ballot access | Yes | No |
| Primary funding | Yes | Rarely |
| General election debates | Yes | Excluded |
| Media coverage | Extensive | Minimal |

| Convention control | Party leadership |
