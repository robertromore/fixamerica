# Party Systems: Root Causes

## Structural Causes

### Electoral System Design

**Duverger's Law**

| System | Tendency |
|--------|----------|
| Single-member plurality | Two parties |
| Proportional representation | Multiple parties |
| American system | Strong two-party tendency |

**Winner-Take-All Effects**

| Feature | Consequence |
|---------|-------------|
| One winner per district | Third parties shut out |
| No proportional allocation | Votes "wasted" for losers |
| Strategic voting | Vote for "lesser evil" |
| Spoiler effect | Third party hurts closest ally |

### Electoral College

**Presidential Selection Impact**

| Feature | Effect on Parties |
|---------|-------------------|
| State-by-state winner-take-all | Third parties can't win states |
| 270 electoral votes needed | Only two viable coalitions |
| Faithless elector risk | Discourages third parties |
| No runoff | Spoiler concerns dominate |

### Constitutional Structure

**Separation of Powers**

| Feature | Effect |
|---------|--------|
| Separate executive | No coalition governments |
| Fixed terms | Can't call new elections |
| Federalism | 50 different ballot systems |
| Bicameralism | Multiple barriers to entry |

## Legal Causes

### Ballot Access Laws

**State-Created Barriers**

| Barrier | Effect |
|---------|--------|
| Signature requirements | Cost and organization required |
| Filing deadlines | Early commitment required |
| Vote thresholds | Must maintain status |
| Sore loser laws | Can't run as independent after primary |

**Major Party Advantage**

| Benefit | Major Parties | Third Parties |
|---------|---------------|---------------|
| Automatic ballot access | Yes | Must qualify |
| Primary funding | Yes | Self-funded |
| Debate inclusion | Automatic | 15% threshold |
| Media coverage | Default | Exception |

### Campaign Finance

**Money Advantages**

| Factor | Effect |
|--------|--------|
| Donor networks | Established for major parties |
| Corporate giving | Favors viable winners |
| Public financing | Tied to prior performance |
| Super PAC support | Flows to competitive races |

### Primary System

**Party Control Over Nominations**

| Element | Effect |
|---------|--------|
| Party primaries | Control candidate selection |
| Delegate rules | Party sets criteria |
| State laws | Support party primaries |
| Access to resources | Party controls |

## Political Causes

### Partisan Incentives

**Why Parties Resist Change**

| Interest | Opposition |
|----------|------------|
| Elected officials | Protected from competition |
| Party organizations | Maintain power |
| Donors | Access to winners |
| Consultants | Profitable duopoly |

**Primary Dynamics**

| Factor | Effect |
|--------|--------|
| Low turnout | Extreme voters overrepresented |
| Base mobilization | Rewards ideological purity |
| Primary threats | Fear of challenge |
| Safe seats | General election doesn't moderate |

### Media Ecosystem

**Two-Party Reinforcement**

| Factor | Effect |
|--------|--------|
| Horse-race coverage | Focus on two frontrunners |
| Debate rules | Exclude third parties |
| "Spoiler" narrative | Discourages third party votes |
| Partisan media | Reinforces binary |

### Network Effects

**Self-Reinforcing Dominance**

| Effect | Mechanism |
|--------|-----------|
| Voter coordination | Vote for likely winners |
| Donor coordination | Fund viable candidates |
| Talent recruitment | Join winning team |
| Media attention | Focus on major parties |

## Cultural Causes

### Binary Thinking

**American Political Culture**

| Tendency | Manifestation |
|----------|---------------|
| Two sides to every issue | Binary framing |
| Us vs. them | Tribal identity |
| Winner-take-all mentality | Competition over compromise |
| Sports metaphors | Team loyalty |

### Partisan Identity

**Identity Politics**

| Development | Effect |
|-------------|--------|
| Party as identity | Beyond policy preferences |
| Affective polarization | Emotional attachment |
| Negative partisanship | Define self against other |
| Social sorting | Lifestyle alignment |

### Path Dependence

**Historical Lock-in**

| Factor | Effect |
|--------|--------|
| Existing party organizations | Institutional inertia |
| Voter habits | Familiar choices |
| Media infrastructure | Built around two parties |
| Legal framework | Supports existing parties |

## Economic Causes

### Concentration of Resources

**Campaign Economics**

| Factor | Effect |
|--------|--------|
| High cost of campaigns | Favors established parties |
| Donor networks | Built over decades |
| Media buys | Economies of scale |
| Professional campaigns | Require resources |

### Employment Interests

**Political Industry**

| Actor | Interest |
|-------|----------|
| Consultants | Two-party system is profitable |
| Pollsters | Expertise in binary |
| Media | Business model around conflict |
| Lobbyists | Access through parties |

## Interconnections

### The Self-Reinforcing System

```text

Electoral Rules (winner-take-all)
        ↓
Two-Party Dominance
        ↓
Parties Control Rule-Making
        ↓
Rules Reinforce Two Parties
        ↓
Electoral Rules (cycle)

```text

### Feedback Loops

**Polarization Cycle**

| Step | Dynamic |
|------|---------|
| Primaries select extremes | Low turnout, activist dominance |
| General election is binary | No moderate alternative |
| Winners are partisan | Congress more polarized |
| Polarization increases | Primary voters more extreme |
| Cycle continues | Accelerating polarization |

### Why Change Is Hard

**Multiple Barriers**

| Barrier | Challenge |
|---------|-----------|
| Constitutional structure | Requires fundamental change |
| State laws | 50 different systems |
| Party control | Write their own rules |
| Voter habits | Familiarity bias |
| Network effects | Coordination problems |
| Media reinforcement | Binary narrative |
