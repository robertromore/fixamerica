# Party Systems: Stakeholders

## Who Benefits from the Status Quo

### Major Party Organizations

**Democratic Party**

| Benefit | Value |
|---------|-------|
| Guaranteed ballot access | 50-state presence |
| Primary process control | Candidate selection |
| Donor networks | Billions in funding |
| Media coverage | Default attention |

**Republican Party**

| Benefit | Value |
|---------|-------|
| Guaranteed ballot access | 50-state presence |
| Primary process control | Candidate selection |
| Donor networks | Billions in funding |
| Media coverage | Default attention |

### Elected Officials

**Incumbents**

| Benefit | Mechanism |
|---------|-----------|
| Reduced competition | Third parties excluded |
| Safe seats | Primary only challenge |

| Campaign infrastructure | Party support |

**Party Leadership**

| Benefit | Power |
|---------|-------|

## Who Is Harmed

### The General Public

**Voter Harm**

| Impact | Effect |
|--------|--------|
| Limited choice | Binary options only |
| Forced bundling | Accept entire platform |
| Negative voting | Against rather than for |
| Alienation | Don't feel represented |

**Specific Groups Harmed**

| Group | Harm |
|-------|------|
| Moderates | No moderate option |
| Independents | Limited primary access |
| Ideological minorities | No representation |
| Non-voters | System designed without them |

### Third Parties

**Structural Disadvantages**

| Challenge | Effect |
|-----------|--------|
| Ballot access | Enormous resource drain |
| Media exclusion | "Spoiler" narrative |
| Debate exclusion | Can't reach voters |
| Fundraising | Can't compete |

**Specific Third Parties**

| Party | Challenge |
|-------|-----------|
| Libertarian | Best organized, still marginal |
| Green | Progressive votes split |
| Forward | Centrist space crowded |
| Constitution | Conservative competition |

### Issue Advocates

**Cross-Partisan Issues**

| Issue | Problem |
|-------|---------|
| Criminal justice reform | Spans party lines |
| Drug policy | Neither party owns |
| Civil liberties | Split coalitions |
| Term limits | Both parties oppose |

### Future Generations

**Long-Term Harm**

| Issue | Effect |
|-------|--------|
| Climate change | Partisan gridlock |
| Debt | No accountability |
| Infrastructure | Delayed investment |
| Education | Policy whiplash |
