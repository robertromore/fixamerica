# Infrastructure Resilience: Opposition

## Overview

Efforts to strengthen infrastructure resilience face opposition from multiple directions: those who question climate science, fiscal conservatives concerned about costs, industries resistant to regulation, and property rights advocates opposing land use restrictions. Understanding these opposition dynamics is essential for building effective coalitions.

## Climate Skepticism

### Arguments Against Climate Adaptation

| Argument | Description |
|----------|-------------|
| **Climate denial** | Reject underlying science |
| **Uncertainty** | "Too uncertain to act" |
| **Natural variation** | Attribute to cycles |
| **Cost-benefit** | Benefits don't justify costs |

### Sources of Opposition

| Source | Interest |
|--------|----------|
| **Fossil fuel industry** | Implicitly acknowledges problem |
| **Ideological groups** | Reject government action |
| **Some politicians** | Constituent base |
| **Media outlets** | Skeptical framing |

### Responding to Climate Arguments

| Response | Description |
|----------|-------------|
| **Risk management** | Resilience valuable regardless |
| **No-regrets** | Benefits even without climate change |
| **Consensus** | Scientific agreement |
| **Observed changes** | Already occurring |
