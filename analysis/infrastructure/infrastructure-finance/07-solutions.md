# Infrastructure Finance: Solutions

## Overview

Sustainable infrastructure finance requires a portfolio approach: reforming existing user fees, developing new revenue sources, leveraging federal investment, and ensuring equitable distribution. No single mechanism will close the funding gap; multiple complementary approaches are needed.

## User Fee Reform

### Gas Tax Reform

| Reform | Description |
|--------|-------------|
| **Increase** | Restore purchasing power |
| **Index to inflation** | Automatic adjustment |
| **Index to construction costs** | Match actual needs |
| **Diesel parity** | Adjust commercial rates |

### VMT Fee Transition

| Phase | Approach |
|-------|----------|
| **Phase 1** | Voluntary pilot programs |
| **Phase 2** | EV mandatory, hybrid optional |
| **Phase 3** | Universal, replace gas tax |
| **Phase 4** | Variable pricing (time, location) |
