# Ports and Waterways: Opposition

## Overview

Opposition to maritime infrastructure investment comes from fiscal conservatives questioning public spending, environmental groups concerned about ecosystem impacts, communities resisting port expansion, and competing interests within the industry itself. Understanding these opposition forces is essential for building effective coalitions.

## Fiscal Opposition

### Against Increased Spending

| Argument | Response |
|----------|----------|
| **Government shouldn't subsidize shipping** | Infrastructure is public good |
| **User fees should cover costs** | Current fees inadequate |
| **Inefficient government projects** | Private sector benefits |
| **Debt concerns** | Investment generates returns |

### Against Trust Fund Use

| Argument | Response |
|----------|----------|
| **Deficit reduction** | Collected for maritime purpose |
| **Other priorities** | Infrastructure essential |
| **Already adequate spending** | Backlog proves otherwise |

|----------|----------|

### Against Inland Waterway Projects
