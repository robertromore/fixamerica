# Ports and Waterways: Root Causes

## Overview

Maritime infrastructure challenges stem from complex funding mechanisms that fail to match investment to needs, lengthy federal project processes, split responsibility between federal and local authorities, decades of deferred maintenance, and inadequate attention to climate resilience. Understanding these root causes is essential for effective reform.

## Funding Structure Problems

### Harbor Maintenance Tax Issues

| Issue | Impact |
|-------|--------|
| **Historical underspending** | Trust fund accumulation |
| **Cargo value basis** | Not tied to infrastructure use |
| **Distribution formula** | Mismatch with needs |
| **Small port disadvantage** | Limited access to funds |

| Issue | Impact |
