# Broadband Infrastructure: Opposition

## Overview

Opposition to broadband reform comes primarily from incumbent ISPs protecting market power, ideological opponents of government intervention, and state legislators influenced by industry lobbying. Understanding opposition arguments, tactics, and funding is essential for advancing universal, affordable broadband access.

## Primary Opposition Forces

### Large ISPs

| Actor | Primary Interest |
|-------|------------------|
| **Comcast** | Protect cable monopoly |
| **Charter** | Maintain market position |
| **AT&T** | Protect legacy, capture subsidy |
| **Verizon** | Wireless priority |
| **Industry associations** | Coordinated opposition |

### Anti-Regulation Groups

| Organization | Position |
|--------------|----------|
| **Heritage Foundation** | Market-based solutions |
| **Cato Institute** | Deregulation |
| **ALEC** | Model anti-municipal legislation |
| **Americans for Prosperity** | Limited government |

## Opposition Arguments

### Against Public Investment

| Argument | Response |
|----------|----------|
| **Government shouldn't pick winners** | Market failed to deliver |
| **Private investment sufficient** | Decades of gaps prove otherwise |
| **Wasteful spending** | Subsidy to incumbents more wasteful |
| **Crowds out private** | Private wouldn't serve these areas |

### Against Municipal Broadband

| Argument | Response |
|----------|----------|
| **Government shouldn't compete** | It's infrastructure, like roads |
| **Taxpayer risk** | Many successful examples |
| **Unfair advantage** | Incumbents get subsidies too |
| **Will fail** | Evidence shows success |

### Against Regulation

| Argument | Response |
|----------|----------|
| **Stifles innovation** | Monopoly stifles more |
| **Reduces investment** | Investment continues under regulation |
| **Unnecessary** | Market has failed |
| **Government overreach** | Public interest requires oversight |

### Against Affordability Programs

| Argument | Response |
|----------|----------|
| **Subsidizes demand not supply** | Addresses real barrier |
| **Creates dependency** | Essential service like electricity |
| **Expensive** | Digital exclusion costs more |
| **ISPs will just raise prices** | Requires competition/regulation |

## Opposition Tactics

### Legislative Tactics

| Tactic | Example |
|--------|---------|
| **Municipal broadband bans** | ~20 states restrict |
| **Preemption laws** | Block local authority |
| **Subsidy capture** | Shape program rules |
| **Definition manipulation** | Weak speed standards |

### Regulatory Tactics

| Tactic | Example |
|--------|---------|
| **Comment campaigns** | Flood FCC proceedings |
| **Legal challenges** | Sue to block rules |
| **Delay tactics** | Slow implementation |
| **Mapping manipulation** | Overstate coverage |
