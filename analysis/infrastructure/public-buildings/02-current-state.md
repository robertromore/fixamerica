# Public Buildings: Current State

## Overview

America's public building portfolio encompasses millions of structures serving government functions, education, and community services. The current state varies significantly by sector and jurisdiction, but common themes emerge: aging infrastructure, deferred maintenance, energy inefficiency, and accessibility gaps.
