# Public Buildings: Opposition

## Overview

Efforts to increase public building investment face opposition from fiscal conservatives concerned about spending, federalism advocates opposing federal involvement in schools, and various interests benefiting from current arrangements. Understanding opposition arguments is essential for effective advocacy.

## Fiscal Opposition

### Cost Concerns

| Argument | Description |
|----------|-------------|
| **Spending magnitude** | Hundreds of billions needed |
| **Budget competition** | Other priorities |
| **Deficit impact** | Adds to federal debt |
| **Operating costs** | New buildings need ongoing funding |

### Fiscal Conservative Positions

| Position | Rationale |
|----------|-----------|
| **Limit federal role** | State and local responsibility |
| **Privatize** | Private sector more efficient |
| **Lease vs. own** | Avoid capital costs |
| **Reduce footprint** | Smaller government |

### Opposition to School Facilities Funding

| Argument | Description |
|----------|-------------|
| **Local responsibility** | Education is state/local |
| **Federal overreach** | Strings attached |
| **Redistribution** | Takes from successful districts |
| **Spending ineffective** | More money doesn't improve outcomes |
