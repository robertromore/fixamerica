# Retirement Security: Actions

## Individual Actions

### For Workers With Employer Plans

**Maximize Participation**

| Action | Impact |
|--------|--------|
| Enroll immediately | Time is most valuable asset |
| Contribute enough for full match | Free money from employer |
| Increase contribution 1%/year | Barely noticeable, compounds |
| Use auto-escalation | Automatic increases |

**Contribution Targets by Age**

| Age | Minimum Target | Ideal Target |
|-----|----------------|--------------|
| 20s | 10% (including match) | 15% |
| 30s | 15% | 20% |
| 40s | 15-20% | 25%+ |
| 50s | Max out with catch-up | Max contribution |
| 60s | Max out with catch-up | Max contribution |

**Investment Decisions**

| Action | Guidance |
|--------|----------|
| Choose low-cost options | Index funds or target-date |
| Avoid company stock | Limit to 10% maximum |
| Use target-date fund | If unsure about allocation |
| Rebalance annually | Or use target-date for automatic |
| Review fees | Switch to lower-cost if available |

**Fee Impact Example**

| Starting Balance | 0.25% Fee | 1.0% Fee | Difference |
|------------------|-----------|----------|------------|
| $50,000 over 30 years | $380,000 | $304,000 | $76,000 |

**Avoid Common Mistakes**

| Mistake | Solution |
|---------|----------|
| Not enrolling | Enroll on day one |
| Not getting full match | Contribute at least to match |
| Cashing out when changing jobs | Roll over to IRA or new plan |
| Borrowing from 401(k) | Build emergency fund instead |
| Investing too conservatively | Age-appropriate allocation |
| Ignoring fees | Compare fund expense ratios |

### For Workers Without Employer Plans

**Open Individual Accounts**

| Account Type | Best For | 2024 Limit |
|--------------|----------|------------|
| Roth IRA | Lower earners, want tax-free growth | $7,000 |
| Traditional IRA | Higher earners, want deduction now | $7,000 |
| SEP-IRA | Self-employed | 25% of income, up to $69,000 |
| Solo 401(k) | Self-employed, want higher limits | $69,000 total |
| SIMPLE IRA | Small business owner | $16,000 |

**Low-Cost Providers**

| Provider | Minimum | Notes |
|----------|---------|-------|
| Vanguard | $0 (for target-date) | Industry pioneer in low costs |
| Fidelity | $0 | No minimums, zero-fee funds |
| Schwab | $0 | No minimums |
| Betterment | $0 | Automated investing |

**State Auto-IRA Programs**

If available in your state and your employer participates:

| Step | Action |
|------|--------|
| 1 | Check if you're automatically enrolled |
| 2 | Verify contribution rate (increase if possible) |
| 3 | Review investment selection |
| 4 | Keep account when changing jobs |

**Gig Worker Options**

| Situation | Best Action |
|-----------|-------------|
| Full-time gig | Open SEP-IRA or Solo 401(k) |
| Part-time gig | Open Roth IRA |
| Multiple gigs | Consolidate into one IRA |
| Irregular income | Contribute when able |

### For All Workers

**Know Your Social Security**

| Action | Why It Matters |
|--------|----------------|
| Create My Social Security account | Track earnings record |
| Check annual statement | Verify earnings credited |
| Correct errors promptly | Limited time to fix |
| Understand claiming options | Significant benefit impact |

**Social Security Claiming Strategies**

| Claiming Age | Benefit Impact |
|--------------|----------------|
| 62 | 70% of full benefit |
| 67 (Full Retirement Age) | 100% of full benefit |
| 70 | 124% of full benefit |

*Delaying from 62 to 70 increases benefits by 77%*

**Build Emergency Savings**

| Step | Target |
|------|--------|
| Start | $1,000 initial emergency fund |
| Build | 3-6 months expenses |
| Maintain | Keep accessible (savings account) |
| Separate | Don't mix with retirement |

*Emergency savings prevents retirement withdrawals.*

**Reduce Debt**

| Priority | Action |
|----------|--------|
| High-interest | Pay off credit cards first |
| Student loans | Consider income-driven repayment |
| Mortgage | Aim to pay off before retirement |
| Car loans | Avoid financing longer than 4 years |

**Increase Financial Literacy**

| Topic | Why Important |
|-------|---------------|
| Compound interest | Understand growth |
| Asset allocation | Age-appropriate investing |
| Fees and costs | Protect returns |
| Social Security | Maximize benefits |
| Medicare | Healthcare in retirement |

### Age-Specific Actions

**In Your 20s**

| Action | Rationale |
|--------|-----------|
| Start saving anything | Time is your greatest asset |
| Get full employer match | Free money |
| Build credit score | Future financial flexibility |
| Live below means | Create savings capacity |
| Learn about investing | Build knowledge base |

**In Your 30s**

| Action | Rationale |
|--------|-----------|
| Increase savings rate | To 15%+ |
| Avoid lifestyle inflation | Raise savings, not spending |
| Review insurance | Protect against risks |
| Consider disability insurance | Protect earning power |
| Balance competing goals | Retirement vs. house, kids |

**In Your 40s**

| Action | Rationale |
|--------|-----------|
| Maximize contributions | Approaching catch-up eligibility |
| Pay down mortgage | Target payoff before retirement |
| Review asset allocation | Adjust for timeline |
| Estimate retirement needs | Reality check |
| Consider long-term care | Start planning |

**In Your 50s**

| Action | Rationale |
|--------|-----------|
| Use catch-up contributions | Extra $7,500 in 401(k), $1,000 in IRA |
| Model retirement scenarios | Detailed planning |
| Review Social Security strategy | Claiming decisions ahead |
| Consider healthcare gap | Medicare starts at 65 |
| Reduce portfolio risk | Sequence-of-returns matters |

**In Your 60s**

| Action | Rationale |
|--------|-----------|
| Finalize Social Security strategy | Major permanent decision |
| Understand Medicare | Enrollment, options |
| Plan withdrawal strategy | Tax-efficient drawdown |
| Consider part-time work | Bridge and purpose |
| Review estate plans | Beneficiaries, documents |

## Employer Actions

### For Employers With Plans

**Improve Plan Design**

| Action | Impact |
|--------|--------|
| Implement auto-enrollment | 90%+ participation |
| Use auto-escalation | Higher savings rates |
| Set higher defaults | 6%+ contribution rate |
| Offer generous match | 50%+ of 6%+ |
| Provide low-cost options | Index funds, low-fee TDFs |

**Support Financial Wellness**

| Program | Benefit |
|---------|---------|
| Financial education | Better decisions |
| Emergency savings | Reduce 401(k) loans |
| Student loan matching | SECURE 2.0 allows |
| Retirement planning services | Better outcomes |

### For Employers Without Plans

**Options by Size**

| Size | Best Option |
|------|-------------|
| 1-10 employees | State auto-IRA (if available), SEP-IRA |
| 10-50 employees | State auto-IRA, SIMPLE IRA, pooled plan |
| 50-100 employees | 401(k) or pooled employer plan |
| 100+ employees | 401(k) with auto-enrollment |

**Getting Started**

| Step | Action |
|------|--------|
| 1 | Check if state program mandates participation |
| 2 | Research options (state program, MEP, own plan) |
| 3 | Calculate costs and tax credits |
| 4 | Choose provider and plan design |
| 5 | Implement with employee communication |

**Tax Credits Available**

| Credit | Amount |
|--------|--------|
| Startup credit (SECURE 2.0) | Up to $5,000/year for 3 years |
| Auto-enrollment credit | $500/year for 3 years |
| Small employer contribution credit | Up to $1,000 per employee |

## Quick Reference

### Priority Actions by Situation

**If Starting from Zero**

1. Enroll in employer plan (or open IRA)
2. Contribute at least to get full match
3. Build $1,000 emergency fund
4. Increase contribution 1% every year

**If Behind on Savings**

1. Maximize contributions
2. Use catch-up contributions if 50+
3. Reduce expenses to free up savings
4. Consider working longer
5. Delay Social Security claiming

**If Approaching Retirement**

1. Model realistic scenarios
2. Develop Social Security strategy
3. Plan healthcare coverage
4. Create withdrawal strategy
5. Review estate documents
