# Corporate Governance Reform: Actions

## Overview

Corporate governance reform requires action at multiple levels: as individuals, as workers, as investors, as citizens, and as community members. This document provides practical actions for different stakeholders.

## For Workers

### In the Workplace

**Organize and Advocate**

| Action | How |
|--------|-----|
| Form or join a union | Contact union organizers in your industry |
| Discuss wages and conditions | This is your legal right under NLRA |
| Request information | Ask about company finances, executive pay |
| Advocate for worker voice | Propose employee committees, feedback channels |

**Seek Board Representation**

| Action | How |
|--------|-----|
| Research company governance | Read proxy statements |
| Propose worker advisory boards | Start with informal proposals |
| Connect with advocacy groups | Partner with organizations pushing codetermination |
| Document successes | Build case for formal representation |

### Worker Ownership

| Action | How |
|--------|-----|
| Learn about ESOPs | Research employee stock ownership plans |
| Propose profit sharing | Make the business case |
| Consider worker cooperatives | Explore cooperative ownership models |
| Support employee ownership transitions | For business succession |

### Collective Action

| Action | How |
|--------|-----|
| Support striking workers | Solidarity actions |
| Join labor advocacy organizations | See Resources document |
| Contact legislators | Support pro-worker legislation |
| Vote for pro-worker candidates | Research candidate positions |

## For Investors

### Individual Investors

**Align Investments with Values**

| Action | How |
|--------|-----|
| Screen investments | Use ESG criteria |
| Invest in B Corps | Certified benefit corporations |
| Choose responsible funds | ESG and impact funds |
| Divest from worst offenders | Companies with poor governance |

**Exercise Shareholder Rights**

| Action | How |
|--------|-----|
| Vote your shares | Don't ignore proxy ballots |
| Support shareholder proposals | Governance and ESG resolutions |
| Attend shareholder meetings | Ask questions, vote in person |
| File shareholder proposals | If you meet ownership thresholds |

### Retirement Savers

**Engage Your Pension Fund**

| Action | How |
|--------|-----|
| Learn your fund's policies | Request proxy voting guidelines |
| Advocate for ESG integration | Contact fund trustees |
| Support fiduciary engagement | Encourage active ownership |
| Vote for reform-minded trustees | Where trustees are elected |

**401(k) Participants**

| Action | How |
|--------|-----|
| Request ESG options | Ask plan administrator |
| Choose available ESG funds | If offered |
| Advocate for better options | Push employer for responsible choices |

### Institutional Investor Engagement

| Action | How |
|--------|-----|
| Join investor coalitions | Ceres, As You Sow, ICCR |
| Coordinate with other investors | Collective engagement |
| File shareholder resolutions | On governance issues |
| Engage with companies directly | Dialogue and letters |

|--------|-----|

| Action | How |
|--------|-----|

| Report workforce metrics | Compensation ratios, diversity |
| Share information with workers | Financial transparency |
