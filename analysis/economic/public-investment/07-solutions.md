# Public Investment: Solutions

## Guiding Principles

### Investment Criteria

**Effective public investment should be**:

| Principle | Application |
|-----------|-------------|
| Economically productive | Generate returns exceeding cost |
| Addressing market failures | Where private sector won't invest |
| Equity-enhancing | Reduce disparities |
| Climate-aligned | Support decarbonization |
| Strategically important | National competitiveness and security |
| Well-executed | Strong project management |

### Criteria for Evaluating Solutions

| Criterion | Question |
|-----------|----------|
| Economic impact | What are the returns? |
| Equity impact | Who benefits? |
| Political feasibility | Can it pass? |
| Implementation capacity | Can government execute? |
| Sustainability | Is funding stable long-term? |

## Infrastructure Investment

### Transportation

**Surface Transportation**

| Investment | Priority | Funding Need |
|------------|----------|--------------|
| Bridge repair and replacement | Urgent | $125 billion |
| Road maintenance | Urgent | $400 billion |
| Transit state of good repair | Urgent | $176 billion |
| Transit expansion | High | $200+ billion |
| High-speed rail corridors | Medium | $100+ billion |
| Complete streets and bike infrastructure | High | $50 billion |

**Approach**:

1. **Fix it first**: Prioritize maintenance over expansion
2. **Multi-modal investment**: Balance highways with transit, rail, active transport
3. **Climate alignment**: Invest in modes that reduce emissions
4. **Equity focus**: Ensure underserved communities benefit

## Financing Mechanisms

### Direct Appropriations

**Advantages**:

- Most straightforward
- Full public accountability
- Can target equity goals

**Disadvantages**:

- Annual budget competition
- Vulnerable to cuts

## Public-Private Partnerships

### When PPPs Can Work

| Condition | Requirement |
|-----------|-------------|
| Revenue-generating | User fees can support |
| Risk-sharing appropriate | Risks allocated to party best able to manage |
| Competition exists | Competitive bidding |
| Public oversight | Transparency, regulation |
| Long-term value | Not just short-term budget gimmick |

### PPP Structures

| Structure | Application |
|-----------|-------------|
| Design-build | Faster construction, fixed price |
| Design-build-operate | Life-cycle incentives |
| Availability payments | Government pays for service |
| Concessions | Private operation of public asset |

### PPP Guardrails

| Protection | Purpose |
|------------|---------|
| Public ownership | Retain asset ownership |
| Rate regulation | Prevent price gouging |
| Performance standards | Ensure service quality |
| Return limits | Prevent excess profits |
| Termination provisions | Enable public takeover |

### When to Avoid PPPs

- Core public goods (basic water, roads)
- Where competition impossible
- When privatization is disguised asset sale
- When primarily motivated by budget gimmicks

## Implementation Priorities

### Immediate (Years 1-2)

1. Execute current legislation (IIJA, CHIPS, IRA)
2. Strengthen project management capacity
3. Streamline permitting (with environmental protection)
4. Launch workforce development programs
5. Establish infrastructure bank or expand existing programs

### Medium-Term (Years 3-5)

1. Index gas tax to inflation
2. Expand broadband to universal access
3. Achieve lead pipe replacement targets
4. Establish regional technology hubs
5. Scale apprenticeship programs

### Long-Term (Years 5-10)

1. Establish long-term infrastructure funding
2. Achieve climate infrastructure goals
3. Close regional infrastructure gaps
4. Achieve R&D investment targets
5. Build world-class infrastructure systems
