# Wages: Roadmap

## Implementation Strategy

### Principles

| Principle | Application |
|-----------|-------------|
| Phase implementation | Allow adjustment time |
| Index to prevent erosion | Automatic updates |
| Multiple pathways | Federal, state, local, private |
| Build momentum | Early wins build support |
| Address opposition | Economic arguments, coalition building |

### Political Reality Assessment

| Challenge | Strategy |
|-----------|----------|
| Senate filibuster | Budget reconciliation, state action |
| Business opposition | Focus on broad public support |
| Regional variation | Floor not ceiling approach |
| Gig economy lobbying | Coalition building, ballot measures |

## Pathway Analysis

### Federal Pathway

**Budget Reconciliation**

| Requirement | Challenge |
|-------------|-----------|
| 51 votes | Manchin/moderate Democrats |
| Byrd Rule | Parliamentarian ruling |
| Revenue/spending | Must have budget impact |
| Limited scope | Only certain provisions qualify |

**Regular Order**

| Requirement | Challenge |
|-------------|-----------|
| 60 votes for cloture | Need Republican support |
| Committee passage | Committee chair support |
| Floor time | Competing priorities |
| Conference | House-Senate reconciliation |

### State Pathway

**Ballot Initiatives**

| Advantage | Challenge |
|-----------|-----------|
| Bypass legislature | Signature collection |
| Direct democracy | Industry spending |
| Popular support | Preemption concerns |
| Momentum building | State-by-state |

**Legislative Action**

| Target States | Prospect |
|---------------|----------|
| Democratic trifectas | High likelihood |
| Split government | Possible with pressure |
| Republican states | Unlikely without ballot initiative |

### Local Pathway

**Municipal Action**

| Strategy | Limitation |
|----------|------------|
| City councils | State preemption |
| County ordinances | Legal challenges |
| Living wage laws | Government contractors |
| Wage theft enforcement | Local enforcement capacity |

### Private Sector Pathway

**Employer Actions**

| Strategy | Examples |
|----------|----------|
| Major employer raises | Amazon $15, Target $24 |
| Shareholder advocacy | Investor pressure on pay |
| Consumer pressure | Boycotts, campaigns |
| Union contracts | Pattern bargaining |

## Sequencing Strategy

### Priority Ordering

**Tier 1: Foundation Building**

| Priority | Rationale |
|----------|-----------|
| State minimum wage increases | Build momentum, prove concept |
| Federal contractor minimum | Executive authority, large impact |
| Overtime threshold | Regulatory pathway exists |
| Enforcement funding | Bipartisan potential |

**Tier 2: Federal Framework**

| Priority | Rationale |
|----------|-----------|
| Raise the Wage Act | Core reform |
| Tipped minimum elimination | Part of minimum wage package |
| Paycheck Fairness Act | Gender equity bipartisan appeal |
| WAGE Act | Enforcement framework |

**Tier 3: Structural Reform**

| Priority | Rationale |
|----------|-----------|
| ABC test universalization | End misclassification |
| Sectoral bargaining | Transform wage-setting |
| Pay ratio requirements | Address top-end excess |
| Automatic indexing | Prevent future erosion |

### Dependencies

```
State momentum -> Federal pressure
          |
          v
Executive actions -> Regulatory framework
          |
          v
Coalition building -> Legislative majority
          |
          v
Federal legislation -> Long-term indexing
```

## Risk Mitigation

### Political Risks

| Risk | Mitigation |
|------|------------|
| Administration change | Embed in statute, not regulation |
| Court challenges | Careful drafting, severability |
| State preemption | Legal challenges, federal floor |
| Business backlash | Broad coalition, economic arguments |

### Implementation Risks

| Risk | Mitigation |
|------|------------|
| Employer non-compliance | Strong enforcement, private action |
| Loophole exploitation | Anticipate and close |
| Inadequate funding | Mandatory appropriations |
| Agency capture | Independent oversight |

### Economic Risks

| Risk | Mitigation |
|------|------------|
| Employment effects | Phased implementation, indexing |
| Small business burden | Exemptions, tax credits |
| Regional variation | Floor approach, local supplements |
| Inflation | Modest increases relative to economy |
