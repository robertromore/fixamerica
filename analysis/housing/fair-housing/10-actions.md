# Fair Housing: Actions

## What Citizens Can Do

### If You Experience Discrimination

**Recognize Discrimination**

| Sign | Example |
|------|---------|
| Different treatment | Shown fewer units than white testers |
| Discouragement | Told neighborhood "might not be right for you" |
| Steering | Directed to certain areas based on race |
| False information | Told unit unavailable when it's not |
| Different terms | Higher deposit, stricter requirements |
| Harassment | Hostile treatment based on protected class |
| Refusal | Denied housing without legitimate reason |

**Document Everything**

| What to Record | How |
|----------------|-----|
| Names and dates | Written notes immediately after |
| What was said | Exact quotes if possible |
| What you observed | Physical spaces, available units |
| Witnesses | Names and contact info |
| Communications | Save all emails, texts, ads |
| Comparable treatment | What others were offered |

**File a Complaint**

| Where | How | Deadline |
|-------|-----|----------|
| HUD | Online, phone, mail | 1 year |
| State agency | Varies by state | Varies (often 1 year) |
| Local agency | If available | Varies |
| Private lawsuit | Hire attorney | 2 years |

**HUD Complaint Process**

| Step | What Happens |
|------|--------------|
| 1. File complaint | HUD intake |
| 2. Investigation | HUD gathers evidence |
| 3. Determination | HUD finds cause or no cause |
| 4. Conciliation | Attempt to settle |
| 5. Charge or dismissal | If no settlement |
| 6. Administrative hearing or federal court | Adjudication |

**What You Can Recover**

| Remedy | Description |
|--------|-------------|
| Actual damages | Out-of-pocket costs |
| Emotional distress | Compensated |
| Punitive damages | To punish discriminator |
| Attorney fees | If you prevail |
| Injunctive relief | Stop discrimination |

### Become a Fair Housing Tester

**What Testers Do**

Testers pose as housing seekers to detect discrimination:

| Role | Activity |
|------|----------|
| Contact landlords/agents | Seek housing as matched pair |
| Document treatment | Record information provided |
| Report findings | Provide evidence for cases |
| Testify if needed | Support litigation |

**How to Become a Tester**

| Step | Action |
|------|--------|
| 1. Find organization | Contact local fair housing group |
| 2. Apply | Complete application |
| 3. Training | Receive testing training |
| 4. Certification | Complete requirements |
| 5. Testing | Conduct supervised tests |

**Organizations That Use Testers**

- National Fair Housing Alliance members
- Local fair housing councils
- Some legal aid organizations
- Private fair housing attorneys

**Why Testing Matters**

| Benefit | Impact |
|---------|--------|
| Detects hidden discrimination | Only reliable method |
| Provides evidence | Supports litigation |
| Deters discriminators | Risk of detection |
| Documents patterns | Shows systemic problems |

## Impact of Citizen Action

### What Individual Action Can Achieve

| Action | Potential Impact |
|--------|------------------|
| Filing complaint | Stop individual discriminator, create record |
| Testing | Detect discrimination, support litigation |
| Advocacy | Change policy, increase enforcement |
| Education | Change attitudes, build support |
| Voting | Elect fair housing supporters |

### What Collective Action Can Achieve

| Collective Effort | Historical Impact |
|-------------------|-------------------|
| Civil rights movement | Fair Housing Act passage |
| Fair housing litigation | Major court victories |
| Policy campaigns | AFFH implementation |
| Organizing | Local protections |

### The Long View

Fair housing progress requires sustained action:

| Timeframe | Focus |
|-----------|-------|
| Immediate | Individual cases, awareness |
| Short-term | Policy campaigns, enforcement |
| Medium-term | Systemic reform, institutional change |
| Long-term | Cultural change, desegregation |

Every action contributes to the long-term goal of truly fair housing.
