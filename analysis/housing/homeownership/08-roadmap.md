# Homeownership: Implementation Roadmap

## Strategic Overview

Expanding homeownership access requires coordinated action across federal, state, and local levels over multiple time horizons. This roadmap sequences reforms based on political feasibility, implementation capacity, and interdependencies.

### Implementation Principles

| Principle | Application |
|-----------|-------------|
| Start with executive action | Use existing authority while building legislative support |
| Demonstrate success | Pilot programs build support for expansion |
| Build coalitions | Identify allies across political spectrum |
| Sequence strategically | Earlier wins enable later reforms |
| Address opposition | Anticipate and counter resistance |
| Measure impact | Track outcomes to refine approach |

### Managing Opposition

**Preemptive Strategies**

| Opposition | Counter-Strategy |
|------------|------------------|
| "Risk of 2008 repeat" | Emphasize sustainable terms, show VA/FHA data |
| "Government overreach" | Frame as leveling playing field |
| "Market distortion" | Note existing subsidies already distort |
| "Unqualified borrowers" | Present data on qualified excluded buyers |

## Implementation Challenges

### Capacity Constraints

| Challenge | Mitigation |
|-----------|------------|
| HFA staffing | Federal technical assistance grants |
| Counselor shortage | Training program expansion |
| CDFI capacity | Capitalization support |
| Data systems | Technology investment |

### Political Risks

| Risk | Mitigation |
|------|------------|
| Administration change | Embed in statute where possible |
| Industry lobbying | Build grassroots counter-pressure |
| NIMBY resistance | Separate from zoning (initially) |
| Fiscal constraints | Revenue-neutral proposals |

### Economic Risks

| Risk | Mitigation |
|------|------------|
| Price inflation | Pair with supply expansion |
| Interest rate volatility | Program design for sustainability |
| Recession | Foreclosure prevention readiness |
| Market disruption | Gradual implementation |
