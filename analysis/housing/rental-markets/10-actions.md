# Rental Markets: Actions

## Overview

Transforming rental markets requires action at every level, from individual tenants protecting their rights to collective organizing that shifts political power. This guide provides practical actions for renters, advocates, and citizens.

## For Individual Renters

### Know Your Rights

**Essential Knowledge**:

| Topic | What to Know |
|-------|--------------|
| Lease terms | Read and understand your lease |
| Notice requirements | How much notice landlord must give |
| Habitability standards | What conditions landlord must maintain |
| Security deposit rules | Limits, return deadlines, deductions |
| Eviction process | Steps, timelines, defenses |
| Discrimination protections | Fair Housing Act, state/local laws |

**Where to Find Information**:

- State Attorney General website
- Local tenant rights organizations
- Legal aid hotlines
- City housing department
- HUD regional office

### Document Everything

**What to Document**:

| Item | How |
|------|-----|
| Lease and amendments | Keep copies, photograph |
| Rent payments | Receipts, bank records |
| Maintenance requests | Written, dated, copy kept |
| Landlord communications | Save emails, texts |
| Property conditions | Photos/video with timestamps |
| Witnesses | Names, contact info |

**Documentation Best Practices**:

1. Request maintenance in writing (email or text)
2. Follow up verbal requests with written confirmation
3. Photograph unit at move-in and move-out
4. Keep a housing log of issues and interactions
5. Save all notices received
6. Record conversations if legal in your state (check laws)

### Respond to Eviction Notices

**Immediate Steps**:

| Step | Timeline |
|------|----------|
| Read notice carefully | Immediately |
| Note response deadline | Check local law |
| Contact legal aid | Within 48 hours |
| Gather documentation | Before deadline |
| File response if required | Before deadline |
| Appear in court | On scheduled date |

**Never Ignore an Eviction Notice**:

- Default judgment means automatic loss
- Many defenses exist
- Legal aid may be available
- Negotiated settlements often possible
- Even if you must leave, timing matters

### Access Rental Assistance

**Types of Assistance**:

| Program | Where to Apply |
|---------|----------------|
| Emergency rental assistance (ERAP) | State/local housing agency |
| Section 8 vouchers | Local housing authority |
| Utility assistance (LIHEAP) | State energy office |
| Nonprofit emergency funds | United Way, Catholic Charities, etc. |
| Legal aid | Local legal aid office |

**Tips for Applications**:

1. Apply early, before crisis deepens
2. Gather required documents (income, lease, ID)
3. Follow up on application status
4. Ask about expedited processing
5. Apply to multiple programs if eligible

## For Tenant Organizing

### Building-Level Organizing

**Getting Started**:

| Step | Actions |
|------|---------|
| 1. Identify issues | What problems affect multiple tenants? |
| 2. Talk to neighbors | One-on-one conversations |
| 3. Build committee | Core group of committed tenants |
| 4. Research landlord | Ownership, other properties, violations |
| 5. Document conditions | Photos, complaints, health impacts |
| 6. Develop demands | Specific, achievable, collective |

**Tenant Association Formation**:

| Stage | Activities |
|-------|------------|
| Outreach | Door knocking, flyering |
| Meetings | Building-wide gatherings |
| Structure | Elect leaders, establish bylaws |
| Communication | Group text, email list |
| Escalation | Letters to landlord, media, city |

### Collective Action Tactics

**Escalation Ladder**:

| Level | Tactic |
|-------|--------|
| 1 | Letter to landlord with tenant signatures |
| 2 | Meeting request with landlord |
| 3 | Media attention, social media campaign |
| 4 | City council testimony, agency complaints |
| 5 | Rent withholding (with legal support) |
| 6 | Rent strike (highest risk, requires strong organization) |

**Rent Strike Considerations**:

| Factor | Requirement |
|--------|-------------|
| Legal protection | State law on rent escrow, habitability |
| Tenant unity | Near-unanimous participation |
| Organization | Clear leadership, communication |
| Legal support | Attorney guidance |
| Public support | Media, community backing |
| Clear demands | Achievable goals |
| Exit strategy | How to end, what constitutes victory |

## Immediate Action Checklist

### If Facing Eviction

- [ ] Read notice carefully, note deadlines
- [ ] Contact legal aid immediately
- [ ] Gather lease, rent records, communications
- [ ] Document property conditions
- [ ] File response to eviction if required
- [ ] Apply for rental assistance
- [ ] Appear at all court dates
- [ ] Do not move out until required by law

### If Organizing Your Building

- [ ] Talk to neighbors one-on-one
- [ ] Document shared issues
- [ ] Research landlord ownership
- [ ] Form organizing committee
- [ ] Connect with local tenant organization
- [ ] Develop specific demands
- [ ] Send letter to landlord
- [ ] Plan escalation if needed

### If Advocating for Policy

- [ ] Identify priority policy (local, state, or federal)
- [ ] Join existing coalition if available
- [ ] Contact elected representatives
- [ ] Prepare personal testimony
- [ ] Attend hearings, speak up
- [ ] Follow up with lawmakers
- [ ] Build relationships for long-term
