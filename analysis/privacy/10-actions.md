# Privacy: Actions

## For Individuals

### Protect Your Own Privacy

**Digital Hygiene:**

| Action | Difficulty | Impact |
|--------|------------|--------|
| Use privacy-focused browser (Firefox, Brave) | Easy | Moderate |
| Install ad/tracker blocker (uBlock Origin) | Easy | High |
| Use privacy search engine (DuckDuckGo) | Easy | Moderate |
| Review app permissions | Easy | Moderate |
| Use encrypted messaging (Signal) | Easy | High |
| Use password manager | Moderate | High |
| Enable two-factor authentication | Moderate | High |
| Use VPN on public WiFi | Moderate | Moderate |
| Review privacy settings quarterly | Moderate | Moderate |

**Limit Data Collection:**

| Action | Method |
|--------|--------|
| Opt out of data brokers | Use broker-specific opt-outs, state registries (CA), optoutprescreen.com (credit offers only) |
| Request data deletion | Use state law rights (CA, VA, etc.) |
| Limit location sharing | Disable when not needed |
| Review connected accounts | Revoke unnecessary access |
| Use disposable email addresses | For signups |
| Pay with cash when possible | Limits financial tracking |

**Be an Informed Consumer:**

| Action | Purpose |
|--------|---------|
| Read privacy policies (key sections) | Know what you're agreeing to |
| Check app permissions before installing | Understand data access |
| Research company privacy practices | Vote with your wallet |
| Support privacy-respecting alternatives | Create market incentive |
