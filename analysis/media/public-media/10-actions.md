## Getting Started Checklist

### Week One

- [ ] Become a member of local station
- [ ] Contact member of Congress about CPB funding
- [ ] Follow station on social media
- [ ] Learn station's community role

### Month One

- [ ] Set up monthly sustaining gift
- [ ] Contact state legislators
- [ ] Share public media importance with friends
- [ ] Attend station event or program

### Year One

- [ ] Actively support funding advocacy
- [ ] Consider advisory board service
- [ ] Build workplace/community support
- [ ] Include station in planned giving
