# Press Freedom: Solutions

## Legal Protections

### Federal Shield Law

| Component | Description |
|-----------|-------------|
| **Qualified privilege** | Protection from compelled testimony |
| **Covered persons** | Functional definition of journalism |
| **Exceptions** | National security, imminent harm, crime |
| **Judicial oversight** | Court balancing test |
| **Process requirements** | Exhaustion, notification |

**Model Approach**: Combine broad protection with narrow, well-defined exceptions.

### Strengthen State Shield Laws

| Reform | Description |
|--------|-------------|
| **Expand to all states** | 10 states lack shield laws |
| **Strengthen existing** | Reduce exceptions |
| **Uniform coverage** | Consistent protection |
| **Digital inclusion** | Cover online journalism |

### Anti-SLAPP Legislation

| Level | Reform |
|-------|--------|
| **Federal anti-SLAPP** | Protect against strategic suits |
| **Strengthen state laws** | Early dismissal, fee-shifting |
| **Uniform standards** | Consistent protection |
| **International enforcement** | Address foreign suits |

### Espionage Act Reform

| Reform | Purpose |
|--------|---------|
| **Journalist exception** | Publication not prosecution |
| **Public interest defense** | Whistleblowing protection |
| **Intent requirement** | Must intend harm |
| **Narrow application** | Only true espionage |

## Evidence Assessment

### What Works

| Solution | Evidence |
|----------|----------|
| **Shield laws** | States with them have fewer conflicts |
| **Anti-SLAPP laws** | Early dismissal effective |
| **Press identification** | Reduces arrest rates |
| **Legal defense** | Enables resistance |

### What's Needed

| Solution | Status |
|----------|--------|
| **Federal shield law** | Repeatedly proposed, never passed |
| **FOIA reform** | Incremental progress |
| **Classification reform** | Little progress |
| **Surveillance limits** | Post-Snowden reforms limited |

### International Models

| Country | Model |
|---------|-------|
| **Sweden** | Constitutional protection, source protection |
| **Germany** | Strong press freedom laws |
| **Canada** | Federal shield law |
| **Australia** | Recent shield law (2017) |
