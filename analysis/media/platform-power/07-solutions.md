# Platform Power: Solutions

## Section 230 Reform

### Targeted Liability Changes

| Approach | Description |
|----------|-------------|
| **Algorithm exception** | Remove immunity for algorithmic amplification |
| **Paid content exception** | Liability for paid/promoted content |
| **Size thresholds** | Full immunity only for small platforms |
| **Knowledge standard** | Liability when platform has notice |

### Transparency Requirements

| Requirement | Purpose |
|-------------|---------|
| **Algorithm disclosure** | How content is ranked |
| **Content moderation reports** | What's removed and why |
| **Advertising archives** | Political and issue ads |
| **Research access** | Independent study |

### Process Requirements

| Requirement | Purpose |
|-------------|---------|
| **Appeals process** | Recourse for users |
| **Explanation requirement** | Tell users why action taken |
| **Due process standards** | Fair enforcement |

## Antitrust Action

### Structural Remedies

| Action | Target |
|--------|--------|
| **Vertical separation** | Separate ad tech from content |
| **Horizontal divestiture** | Break up acquisitions (Instagram, YouTube) |
| **Line of business restrictions** | Prevent self-preferencing |

### Behavioral Remedies

| Remedy | Purpose |
|--------|---------|
| **Non-discrimination** | Equal treatment of competitors |
| **Interoperability mandates** | Allow competition |
| **Data portability** | Users can take data |
| **Self-preferencing prohibition** | Fair competition |

### Merger Review

| Reform | Application |
|--------|-------------|
| **Presumption against platform mergers** | Shift burden |
| **Acquisition review** | Retrospective examination |
| **Data market analysis** | Consider data effects |
| **Startup protection** | Preserve competition |

## Platform Payment for News

### Mandatory Bargaining

| Model | Description |
|-------|-------------|
| **JCPA approach** | Collective bargaining with arbitration |
| **Australian model** | Designate platforms, mandatory negotiation |
| **Canadian model** | Similar mandatory framework |

### Revenue Sharing

| Approach | Mechanism |
|----------|-----------|
| **Percentage of revenue** | Share of news-related ad revenue |
| **Per-click payment** | Payment for each referral |
| **Licensing fees** | Annual payment for news access |

### Distribution Requirements

| Requirement | Purpose |
|-------------|---------|
| **Non-discrimination** | Can't punish negotiating publishers |
| **Local news promotion** | Algorithmic support |
| **Quality signals** | Prioritize journalism over clickbait |
