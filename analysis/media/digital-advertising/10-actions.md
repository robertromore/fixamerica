# Digital Advertising: Actions

## For Individuals

### Protect Your Privacy

| Action | How | Impact |
|--------|-----|--------|
| **Use privacy tools** | Ad blockers, privacy browsers | Reduce tracking |
| **Opt out of tracking** | Platform settings, opt-out links | Less data collected |
| **Use private search** | DuckDuckGo, etc. | Avoid search tracking |
| **Review permissions** | App and browser settings | Limit data access |
| **Read privacy policies** | Understand data use | Informed choices |

### Support Quality Journalism

| Action | Impact |
|--------|--------|
| **Subscribe directly** | Bypass ad model |
| **Support nonprofit news** | Non-commercial model |
| **Avoid clickbait** | Don't reward bad content |
| **Share quality content** | Amplify good journalism |

### Political Engagement

| Action | Target |
|--------|--------|
| **Contact Congress** | Privacy law, antitrust |
| **Contact state legislators** | State privacy laws |
| **Support privacy organizations** | Advocacy funding |
| **Vote for reformers** | Electoral accountability |

### Consumer Action

| Action | Impact |
|--------|--------|
| **File complaints** | FTC, state AGs |
| **Support litigation** | Class actions, advocacy |
| **Data requests** | Exercise access rights |
| **Deletion requests** | Exercise deletion rights |
