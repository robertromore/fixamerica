# Digital Advertising: Current State

## Market Overview

### Market Size

| Metric | Value | Trend |
|--------|-------|-------|
| **US digital ad spending** | $225B+ (2023) | Growing |
| **Global digital** | $600B+ | Growing |
| **Share of total advertising** | 65%+ | Increasing |
| **Programmatic share** | 90%+ of display | Dominant |

### Market Concentration

| Player | US Digital Share |
|--------|-----------------|
| **Google** | ~28% |
| **Meta (Facebook)** | ~20% |
| **Amazon** | ~14% |
| **Microsoft** | ~4% |
| **All others** | ~34% |

### Revenue by Format

| Format | Share |
|--------|-------|
| **Search** | ~40% |
| **Social** | ~30% |
| **Display/Programmatic** | ~20% |
| **Video** | ~10% |

## Tracking and Privacy

### Current Tracking Methods

| Method | Status |
|--------|--------|
| **Third-party cookies** | Being phased out (slowly) |
| **First-party data** | Growing importance |
| **Device fingerprinting** | Persistent |
| **Email/login** | Identity graphs |
| **Mobile IDs** | IDFA (limited), GAID |

### Privacy Changes

| Development | Impact |
|-------------|--------|
| **GDPR (2018)** | Consent requirements |
| **CCPA/CPRA** | California privacy |
| **Apple ATT** | iOS tracking limits |
| **Chrome cookie deprecation** | Delayed repeatedly |
| **Privacy Sandbox** | Google alternative |

### Industry Response

| Response | Status |
|----------|--------|
| **First-party data** | Increased focus |
| **Contextual targeting** | Revival |
| **Identity solutions** | UID 2.0, others |
| **Cohort-based** | Topics API |
| **Walled gardens** | Platform data |

## Impact on Journalism

### Revenue Shift

| Metric | Change |
|--------|--------|
| **Newspaper ad revenue** | Down 70%+ from peak |
| **Digital ad to publishers** | Platform capture |
| **Local ad spending** | Shifted to Google, Meta |
| **CPMs** | Race to bottom |

### Publisher Challenges

| Challenge | Impact |
|-----------|--------|
| **Platform dependence** | Revenue at their mercy |
| **Low CPMs** | Programmatic devalues content |
| **Ad fraud** | Illegitimate competition |
| **Brand safety** | Blacklists hurt news |
| **Fee extraction** | Multiple intermediaries |

### Quality Journalism Effects

| Effect | Mechanism |
|--------|-----------|
| **Clickbait pressure** | Volume over quality |
| **Layoffs** | Revenue insufficient |
| **Local news collapse** | Cannot compete |
| **Subscription shift** | Ad model failing |

## Misinformation and Ads

### Monetization Problem

| Issue | Status |
|-------|--------|
| **Programmatic funds misinfo** | Automated, no review |
| **Brand safety gaps** | Ads appear anyway |
| **MFA sites** | Made-for-advertising junk |
| **Content farms** | Low-quality arbitrage |

### Industry Response

| Response | Effectiveness |
|----------|---------------|
| **Keyword blocklists** | Crude, harms news |
| **Verification vendors** | Limited impact |
| **Inclusion lists** | Reduces scale |
| **Industry initiatives** | Mixed results |

## Ad Fraud

### Types of Fraud

| Type | Description |
|------|-------------|
| **Invalid traffic** | Bots, not humans |
| **Domain spoofing** | Fake premium inventory |
| **Click fraud** | Fake clicks |
| **Attribution fraud** | Credit stealing |
| **Ad stacking** | Hidden ads |

### Scale

| Metric | Value |
|--------|-------|
| **Global ad fraud** | $35B+ annually |
| **Bot traffic** | 20-40% of traffic |
| **Detection challenges** | Sophisticated fraud |
| **Cost passed to** | Advertisers, consumers |
