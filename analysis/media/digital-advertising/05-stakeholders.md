# Digital Advertising: Stakeholders

## Who Is Affected

### Consumers/Users

| Impact | Description |
|--------|-------------|
| **Privacy invasion** | Tracked across internet |
| **Manipulation** | Targeted persuasion |
| **Ad costs** | Passed on in prices |
| **Quality decline** | Clickbait content |
| **Filter bubbles** | Algorithmic silos |

### Publishers

| Impact | Description |
|--------|-------------|
| **Revenue collapse** | Platforms capture value |
| **Fee extraction** | Intermediaries take cut |
| **Data disadvantage** | Platforms know more |
| **Quality pressure** | Clickbait incentives |
| **Dependence** | Platform mercy |

### Advertisers

| Impact | Description |
|--------|-------------|
| **Hidden fees** | Opaque costs |
| **Ad fraud** | Wasted spend |
| **Brand safety** | Ads on bad content |
| **Platform dependence** | Limited alternatives |
| **Measurement** | Attribution unclear |

### Society

| Impact | Description |
|--------|-------------|
| **Journalism decline** | Democracy harmed |
| **Misinformation** | Ads fund falsehoods |
| **Surveillance** | Civil liberties |
| **Market concentration** | Economic power |

## Who Has Power

### Dominant Platforms

| Player | Power |
|--------|-------|
| **Google** | Owns ad tech stack, search, Chrome |
| **Meta** | Social data, audience |
| **Amazon** | E-commerce data, growing ads |
| **Apple** | Device control (privacy changes) |

### Ad Tech Intermediaries

| Player | Power |
|--------|-------|
| **DSPs** | Buying decisions |
| **SSPs** | Selling control |
| **Data brokers** | Targeting data |
| **Verification vendors** | Brand safety calls |

### Government

| Actor | Power |
|-------|-------|
| **FTC** | Antitrust, privacy |
| **DOJ** | Antitrust |
| **Congress** | Legislation |
| **State AGs** | Enforcement |
| **EU regulators** | GDPR, DSA, DMA |

## Stakeholder Positions

### On Privacy Regulation

| Stakeholder | Position |
|-------------|----------|
| **Google** | Support some rules (own advantages) |
| **Meta** | Oppose strict rules |
| **Publishers** | Generally support |
| **Advertisers** | Mixed |
| **Privacy advocates** | Strong support |
| **Ad tech** | Oppose restrictions |

### On Antitrust

| Stakeholder | Position |
|-------------|----------|
| **Google** | Oppose intervention |
| **Competitors** | Support |
| **Publishers** | Support |
| **Advertisers** | Mixed |
| **Ad tech** | Mixed |

### On Publisher Payment

| Stakeholder | Position |
|-------------|----------|
| **Publishers** | Strong support |
| **Platforms** | Oppose mandatory |
| **Advertisers** | Neutral |
| **Journalism advocates** | Support |

## Power Analysis

### Most Powerful Stakeholders

| Rank | Stakeholder | Why |
|------|-------------|-----|
| 1 | **Google** | Market dominance, vertical integration |
| 2 | **Meta** | Social data, audience |
| 3 | **EU regulators** | GDPR enforcement |
| 4 | **FTC/DOJ** | Antitrust potential |
| 5 | **Large advertisers** | Spending power |

### Least Powerful Stakeholders

| Stakeholder | Why Weak |
|-------------|----------|
| **Individual consumers** | No alternatives, no knowledge |
| **Small publishers** | Dependent, fragmented |
| **Small advertisers** | No leverage |
| **Privacy advocates** | Resource-limited |
