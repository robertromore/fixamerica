# Broadcast Regulation: Stakeholders

## Who Is Affected

### General Public

| Interest | Connection |
|----------|------------|
| **Information access** | Broadcast reaches everyone |
| **Local news** | Community information |
| **Emergency alerts** | EAS system |
| **Diverse perspectives** | Ownership affects viewpoint |
| **Children's education** | Educational programming |

### Local Communities

| Interest | Connection |
|----------|------------|
| **Local coverage** | News about community |
| **Cultural representation** | Reflects community |
| **Emergency information** | Disasters, alerts |
| **Political information** | Local races |

### Specific Populations

| Population | Interest |
|------------|----------|
| **Rural areas** | Often primary media source |
| **Low-income** | Free over-the-air access |
| **Non-English speakers** | Ethnic broadcasting |
| **Children** | Educational content |
| **Political candidates** | Access to airtime |

### Civil Society

| Actor | Power |

| **Academic researchers** | Evidence, expertise |
| **Local advocacy** | Community voice |

## Stakeholder Positions

### On Ownership Limits

| Stakeholder | Position |
|-------------|----------|
| **Major broadcast groups** | Want further relaxation |
| **Networks** | Support relaxation |
| **NAB** | Support relaxation |
| **Media reform groups** | Want stronger limits |
| **Small broadcasters** | Mixed |
| **Public interest groups** | Want stronger limits |

### On Public Interest Obligations

| Stakeholder | Position |
|-------------|----------|
| **Broadcasters** | Prefer minimal requirements |
| **NAB** | Oppose new mandates |
| **Media reform groups** | Want stronger obligations |
| **Journalism advocates** | Want local news requirements |
| **Children's advocates** | Want strong CTA |

### On Fairness Doctrine Revival

| Stakeholder | Position |
|-------------|----------|
| **Conservative media** | Strongly oppose |
| **Broadcasters** | Oppose |
| **Some progressives** | Support |
| **Civil liberties groups** | Mixed (First Amendment) |
| **Most advocates** | Focus elsewhere |

## Power Analysis

### Most Powerful Stakeholders

| Rank | Stakeholder | Why |
|------|-------------|-----|
| 1 | **Broadcast industry** | Resources, access, expertise |
| 2 | **FCC** | Regulatory authority |
| 3 | **Congress** | Legislative power |
| 4 | **Networks** | Economic power |
| 5 | **Courts** | Judicial review |

### Least Powerful Stakeholders

| Stakeholder | Why Weak |
|-------------|----------|
| **General public** | Diffuse, unorganized |
| **Local communities** | No direct voice |
| **Small broadcasters** | Limited resources |
| **Media reform groups** | Outspent by industry |
