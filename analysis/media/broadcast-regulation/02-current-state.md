# Broadcast Regulation: Current State

## FCC Structure and Authority

### Commission Composition

| Position | Appointment |
|----------|-------------|
| **Commissioners** | 5 (appointed by President) |
| **Chair** | Designated by President |
| **Party balance** | 3-2 maximum from one party |
| **Terms** | 5 years |
| **Current status** | Often operates with vacancies |

### Media Bureau

| Function | Responsibility |
|----------|---------------|
| **Licensing** | TV and radio licenses |
| **Ownership rules** | Enforce limits |
| **Policy development** | Rulemaking |
| **Enforcement** | Violations, complaints |

## Ownership Regulations

### Current Ownership Limits

| Rule | Current Status |
|------|----------------|
| **National TV cap** | 39% of households |
| **Local TV duopoly** | Allowed in some markets |
| **Radio-TV cross-ownership** | Largely eliminated |
| **Newspaper-broadcast** | Eliminated 2017 |
| **Local radio limits** | Based on market size |

## Public Interest Obligations

### Current Requirements

| Obligation | Status |
|------------|--------|
| **License renewal** | 8-year terms, minimal review |
| **Public file** | Online posting required |
| **Children's programming** | 3 hours/week (flexible) |
| **Political access** | Equal time, lowest unit rate |
| **EAS participation** | Emergency alerts |
| **Sponsorship ID** | Required for paid content |

### What's Not Required

| Former/Proposed | Status |
|-----------------|--------|
| **Fairness Doctrine** | Eliminated 1987 |
| **Local programming quotas** | None |
| **News requirements** | None |
| **Community ascertainment** | Eliminated 1981 |
| **Station presence** | Main studio rule eliminated |

## Enforcement

### Complaint Process

| Step | Process |
|------|---------|
| **Filing** | Anyone can complain |
| **Review** | Staff review |
| **Investigation** | If warranted |
| **Action** | Warning, fine, or dismissal |

### Enforcement Actions

| Type | Frequency |
|------|-----------|
| **Indecency fines** | Rare |
| **Sponsorship ID** | Occasional |
| **EAS violations** | Regular |
| **Ownership violations** | Rare |
| **License revocation** | Extremely rare |

### License Renewal

| Factor | Status |
|--------|--------|

| **Challenges** | Rare |
| **Denial** | Almost never |
| **Public participation** | Limited |

## International Comparison

### Regulatory Models

| Country | Approach |
|---------|----------|
| **UK (Ofcom)** | Strong public service requirements |
| **Canada (CRTC)** | Canadian content rules |
| **Germany** | Strong public broadcasting |
| **Australia** | Australian content rules |
| **U.S.** | Light touch, deregulated |

### Key Differences

| Factor | U.S. vs. Peers |
|--------|----------------|
| **Ownership limits** | Weaker |
| **Content requirements** | Fewer |
| **Public broadcasting** | Smaller |
| **Local requirements** | Weaker |
