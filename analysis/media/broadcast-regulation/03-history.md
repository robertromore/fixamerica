# Broadcast Regulation: History

## Origins (1910s-1930s)

### Radio Act of 1912

| Provision | Purpose |
|-----------|---------|
| **Licensing required** | Government control |
| **Spectrum allocation** | Manage interference |
| **No content regulation** | Technical focus |

### Radio Act of 1927

| Development | Significance |
|-------------|--------------|
| **Federal Radio Commission** | First regulatory body |
| **Public interest standard** | Introduced |
| **Licensing** | Spectrum as public resource |
| **Scarcity rationale** | Limited spectrum justifies regulation |

### Communications Act of 1934

| Provision | Status |
|-----------|--------|
| **Created FCC** | Replaced FRC |
| **Public interest, convenience, and necessity** | Licensing standard |
| **Broadcast regulation** | Core function |
| **Common carrier** | Separate from broadcasting |

## Golden Age of Regulation (1940s-1970s)

### Public Interest Development

| Era | Development |
|-----|-------------|
| **1940s** | Network regulations |
| **1949** | Fairness Doctrine adopted |
| **1960** | Programming inquiry |
| **1960s** | Community ascertainment |
| **1970s** | Prime time access, financial interest rules |

### Fairness Doctrine

| Component | Requirement |
|-----------|-------------|
| **Controversial issues** | Must cover |
| **Balance** | Present opposing views |
| **Personal attack** | Right of reply |
| **Editorial** | Candidate response |

### Ownership Limits

| Rule | Original |
|------|----------|
| **Multiple ownership** | 7-7-7 (AM/FM/TV) |
| **Duopoly** | Prohibited |
| **Cross-ownership** | Prohibited |
| **Network rules** | Restricted |

### Content Requirements

| Requirement | Purpose |
|-------------|---------|
| **Community ascertainment** | Local needs assessment |
| **Programming logs** | Documentation |
| **Local presence** | Main studio requirement |
| **Public file** | Community access |

## Deregulation Era (1980s-2000s)

### Reagan FCC

| Action | Year | Effect |
|--------|------|--------|
| **License renewal simplified** | 1981 | Reduced scrutiny |
| **Community ascertainment eliminated** | 1981 | No needs assessment |
| **Programming guidelines ended** | 1984 | No minimums |
| **Fairness Doctrine eliminated** | 1987 | No balance required |
| **Ownership caps raised** | 1984 | 12-12-12 |

### Telecommunications Act of 1996

| Provision | Effect |
|-----------|--------|
| **National ownership cap** | 35% (raised from 25%) |
| **Radio ownership** | Dramatically deregulated |
| **Cross-ownership** | Relaxed |
| **License renewal** | 8 years |
| **Challenger burden** | Harder to challenge |

### Post-1996 Consolidation

| Sector | Result |
|--------|--------|
| **Radio** | Massive consolidation (Clear Channel, etc.) |
| **TV stations** | Station groups grew |
| **Network ownership** | Allowed station ownership |
| **Cross-ownership** | Pressure to eliminate |

### Bush FCC (2001-2008)

| Action | Year | Status |
|--------|------|--------|
| **2003 ownership order** | 2003 | Largely overturned |
| **Cross-ownership** | 2007 | Partially relaxed |
| **Localism inquiry** | 2004 | No action |

### Obama FCC (2009-2016)

| Action | Year | Status |
|--------|------|--------|
| **Quadrennial review** | 2010-14 | Minimal change |
| **Cross-ownership** | Retained limits |
| **Localism** | No new requirements |
| **Children's TV** | Maintained |

### Trump FCC (2017-2020)

| Action | Year | Effect |
|--------|------|--------|
| **Cross-ownership eliminated** | 2017 | Major deregulation |
| **Main studio rule** | 2017 | Eliminated |
| **UHF discount restored** | 2017 | Enabled consolidation |
| **Newspaper-broadcast** | 2017 | Eliminated ban |
| **Children's TV** | 2019 | Flexibility added |

### Biden FCC (2021-Present)

| Status | Description |
|--------|-------------|
| **Vacancies** | Operated without full commission |
| **Limited action** | Partisan deadlock |
| **Ownership** | No major changes |
| **Net neutrality** | Restored (non-broadcast) |

## Landmark Cases

### Scarcity Doctrine

| Case | Year | Holding |
|------|------|---------|
| **NBC v. United States** | 1943 | Regulation constitutional |
| **Red Lion v. FCC** | 1969 | Fairness Doctrine upheld |
| **FCC v. Pacifica** | 1978 | Indecency regulation upheld |

### Deregulation Era

| Case | Year | Holding |
|------|------|---------|
| **Syracuse Peace Council** | 1987 | Fairness Doctrine repeal upheld |
| **Prometheus Radio v. FCC** | 2004 | Struck down 2003 rules |
| **Prometheus II** | 2011 | More review needed |
| **FCC v. Prometheus** | 2021 | Deregulation upheld |

## Fairness Doctrine History

### Rise

| Year | Development |
|------|-------------|
| **1949** | Adopted by FCC |
| **1959** | Codified in statute (partially) |
| **1967** | Personal attack rules |
| **1969** | Red Lion upheld doctrine |

### Fall

| Year | Development |
|------|-------------|
| **1981** | Reagan FCC signals opposition |
| **1985** | Fairness report criticizes doctrine |
| **1987** | FCC eliminates doctrine |
| **1987, 1991** | Vetoes of congressional restoration |
| **2000s** | Occasional revival attempts |

## Lessons from History

### What Regulation Achieved

| Achievement | Period |
|-------------|--------|
| **Diverse ownership** | Pre-1996 |
| **Local presence** | Pre-2017 |
| **Some balance** | Fairness Doctrine era |
| **Public participation** | Community ascertainment |

### What Deregulation Caused

| Consequence | Result |
|-------------|--------|
| **Consolidation** | Fewer owners |
| **Less localism** | Centralized operations |
| **Partisan media** | No balance requirement |
| **Profit focus** | Public interest diminished |
