# Media Consolidation: Roadmap

## Strategic Framework

### Vision

A diverse media ecosystem with multiple independent owners serving local communities, where ownership structures support quality journalism rather than extraction.

### Key Milestones

| Year | Goal |
|------|------|
| **2025** | Transparency requirements; antitrust enforcement begins |
| **2027** | Nonprofit conversion pathway established |
| **2029** | Ownership limits restored; review process established |
| **2032** | Significant ownership diversification achieved |
| **2035** | Sustainable diverse ownership ecosystem |
