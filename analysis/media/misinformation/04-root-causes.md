# Misinformation: Root Causes

## Psychological Causes

### Cognitive Biases

| Bias | Effect on Misinformation |
|------|-------------------------|
| **Confirmation bias** | Seek information confirming beliefs |
| **Motivated reasoning** | Evaluate evidence based on desired conclusion |
| **Illusory truth effect** | Repetition increases perceived truth |
| **Availability heuristic** | Recent/vivid examples seem more common |
| **Dunning-Kruger** | Overconfidence in own knowledge |
| **Backfire effect** | Corrections sometimes strengthen false beliefs |

### Identity and Tribal Psychology

| Factor | Mechanism |
|--------|-----------|
| **Group identity** | Beliefs signal group membership |
| **In-group bias** | Trust own group's claims |
| **Out-group distrust** | Reject other group's corrections |
| **Identity-protective cognition** | Protect beliefs that define identity |

### Emotional Drivers

| Emotion | Effect |
|---------|--------|
| **Fear** | Threat narratives spread fast |
| **Anger** | Outrage content shared more |
| **Anxiety** | Uncertainty drives information seeking |
| **Disgust** | Moral outrage spreads |

### Information Processing

| Factor | Impact |
|--------|--------|
| **Cognitive load** | Tired people more susceptible |
| **Fast thinking** | Intuition accepts false claims |
| **Source confusion** | Forget where information came from |
| **Fluency** | Easy-to-process claims seem truer |

## Technological Causes

### Algorithmic Amplification

| Mechanism | Effect |
|-----------|--------|
| **Engagement optimization** | Outrage, novelty boosted |
| **Recommendation systems** | Rabbit holes to extremes |
| **Viral dynamics** | False content spreads faster |
| **A/B testing** | Optimize for engagement, not truth |

### Platform Design

| Feature | Misinformation Effect |
|---------|----------------------|
| **Share button** | Frictionless spread |
| **Metrics (likes, shares)** | Social proof for false claims |
| **Algorithmic feeds** | Personalized filter bubbles |
| **Notification systems** | Engagement hooks |

### Information Abundance

| Factor | Impact |
|--------|--------|
| **Content volume** | Too much to verify |
| **Speed** | Spread faster than fact-check |
| **Persistence** | False content stays online |
| **Discoverability** | Search surfaces old misinformation |

### Reduced Gatekeeping

| Traditional | Digital |
|-------------|---------|
| Editors verified | Anyone publishes |
| Reputational cost | Anonymity possible |
| Limited distribution | Unlimited reach |
| Professional standards | No standards |

## Economic Causes

### Attention Economy

| Incentive | Result |
|-----------|--------|
| **Ad revenue** | Sensational content pays |
| **Engagement metrics** | Outrage is engaging |
| **Competition for attention** | Race to bottom |
| **Low-cost production** | Cheap to create misinfo |

### Platform Business Models

| Model Element | Misinformation Effect |
|---------------|----------------------|
| **Ad-supported** | Engagement > accuracy |
| **Growth imperative** | Viral content valued |
| **Low moderation investment** | Cost center, not profit |
| **User-generated content** | No content costs |

### Misinformation Economy

| Actor | Revenue Source |
|-------|----------------|
| **Clickbait sites** | Programmatic advertising |
| **Grifters** | Product sales, donations |
| **Foreign actors** | State funding |
| **Influencers** | Sponsorships, attention |

## Social/Cultural Causes

### Trust Collapse

| Institution | Trust Decline |
|-------------|---------------|
| **Media** | 72% → 32% (1976-2023) |
| **Government** | Similar decline |
| **Science** | Declining, especially partisan |
| **Experts** | Anti-elite sentiment |

### Polarization

| Factor | Effect |
|--------|--------|
| **Partisan sorting** | Different information sources |
| **Affective polarization** | Distrust of other side |
| **Elite polarization** | Different elite cues |
| **Geographic sorting** | Echo chamber communities |

### Knowledge Gaps

| Gap | Effect |
|-----|--------|
| **Media literacy** | Can't evaluate sources |
| **Science literacy** | Can't evaluate claims |
| **Statistical literacy** | Misunderstand data |
| **Digital literacy** | Can't spot manipulation |

## Political Causes

### Elite Misinformation

| Actor | Role |
|-------|------|
| **Politicians** | Spread and legitimize falsehoods |
| **Partisan media** | Amplify false claims |
| **Think tanks** | Manufacture doubt |
| **Foreign actors** | Strategic disinformation |

### Asymmetric Norms

| Factor | Effect |
|--------|--------|
| **Truth as partisan** | Facts become contested |
| **Epistemic tribalism** | Different standards for each side |
| **Accountability gap** | No consequences for lying |
| **Media "balance"** | False equivalence |

### Strategic Disinformation

| Actor | Strategy |
|-------|----------|
| **Russia** | Sow division, undermine democracy |
| **Domestic actors** | Electoral advantage |
| **Industry** | Protect business interests |

## Structural Causes

### Media Ecosystem

| Factor | Effect |
|--------|--------|
| **Local news collapse** | Information vacuum filled by misinfo |
| **Partisan media** | One-sided narratives |
| **Social media dominance** | Algorithms over editors |
| **24-hour news cycle** | Speed over accuracy |

### Regulatory Gaps

| Gap | Effect |
|-----|--------|
| **Section 230** | Platform immunity |
| **First Amendment** | Limits government action |
| **No transparency rules** | Platforms opaque |
| **Weak enforcement** | Limited consequences |

## Interconnected Causes

### The Misinformation System

```
Low Trust + High Polarization
           ↓
    Tribal Information Processing
           ↓
Platform Amplification
           ↓
        Viral Misinformation
           ↓
    Further Trust Erosion
           ↓
       More Polarization
           ↓
        (cycle repeats)
```

| **Economic** | 20% - Incentives |
| **Social/Political** | 25% - Trust, polarization |
