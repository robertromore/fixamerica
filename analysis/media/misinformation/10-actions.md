# Misinformation: Actions

## For Individuals

### Improve Your Own Practices

| Action | How | Impact |
|--------|-----|--------|
| **Pause before sharing** | Count to 10, check source | Stops viral spread |
| **Check sources** | Look up unfamiliar sources | Avoid amplifying misinfo |
| **Use lateral reading** | Open new tabs, verify claims | Better fact-checking |
| **Recognize emotional manipulation** | Pause when outraged/scared | Resist manipulation |
| **Diversify news sources** | Read multiple outlets | Reduce echo chamber |

### Fact-Checking Skills

| Technique | Application |
|-----------|-------------|
| **SIFT method** | Stop, Investigate source, Find better coverage, Trace claims |
| **Reverse image search** | Verify photo authenticity |
| **Check date and context** | Old stories presented as new |
| **Look for original source** | Follow citations back |
| **Check expert consensus** | Scientific claims especially |

### Respond to Misinformation

| Situation | Response |
|-----------|----------|
| **Friend shares misinfo** | Private message with correction |
| **Family member believes false claim** | Empathetic conversation, questions |
| **See it going viral** | Report to platform |
| **Repeated patterns** | Share prebunking resources |

### What NOT to Do

| Action | Why |
|--------|-----|
| **Amplify by debunking publicly** | Increases exposure |
| **Shame people** | Causes defensiveness |
| **Argue endlessly** | Rarely works |
| **Give up** | Silence enables spread |
