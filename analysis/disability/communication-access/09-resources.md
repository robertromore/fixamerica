# Communication Access: Resources

## Overview

This document compiles key resources for understanding, researching, and advocating for communication access improvements for deaf, hard of hearing, deafblind, and speech-disabled Americans. Resources span academic research, government reports, legal authorities, advocacy organizations, and practical guides.
