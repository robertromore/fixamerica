## For Individuals and Families

### Obtaining Coverage

**ACA Marketplace**

| Step | Action |
|------|--------|
| Open Enrollment | November 1 - January 15 annually |
| Check subsidies | Use healthcare.gov calculator |
| Compare plans | Evaluate premiums, deductibles, networks |
| Apply | healthcare.gov or state exchange |
| Get help | Find local navigator (localhelp.healthcare.gov) |

**Special Enrollment Periods**

| Life Event | Enrollment Window |
|------------|-------------------|
| Job loss | 60 days from coverage end |
| Marriage/divorce | 60 days |
| Birth/adoption | 60 days |
| Move to new area | 60 days |
| Loss of other coverage | 60 days |
| Income change (Medicaid) | Any time |

**Medicaid/CHIP**

| Step | Action |
|------|--------|
| Check eligibility | Medicaid.gov or state site |
| Apply any time | No enrollment period |
| Gather documents | Income, residency, citizenship |
| Follow up | States have processing backlogs |
| Renew on time | Watch for renewal notices |

### Navigating the System

**Finding Affordable Care**

| Option | Description | How to Access |
|--------|-------------|---------------|
| FQHCs | Community health centers with sliding fees | findahealthcenter.hrsa.gov |
| Free clinics | Volunteer-staffed clinics | nafcclinics.org |
| Hospital charity care | Required financial assistance | Ask billing department |
| Prescription assistance | Manufacturer programs | needymeds.org, rxassist.org |
| Telehealth | Often lower cost | Many options available |

**Managing Costs**

| Strategy | Approach |
|----------|----------|
| Negotiate bills | Ask for itemized bill, dispute errors, request reduction |
| Payment plans | Most providers offer interest-free plans |
| Compare prices | Use FAIR Health, Healthcare Bluebook |
| Generic drugs | Ask provider about alternatives |
| GoodRx | Compare pharmacy prices |
| 340B pharmacies | Discounted drugs at qualifying pharmacies |

**Appealing Denials**

| Step | Action |
|------|--------|
| Get denial in writing | Request written explanation |
| Understand reason | Coverage, medical necessity, or procedural |
| Internal appeal | File within deadline (usually 180 days) |
| External review | Independent review after internal denial |
| State assistance | Contact insurance commissioner |
| Legal help | National Health Law Program resources |

### Protecting Your Rights

**Know Your Rights**

| Right | Protection |
|-------|------------|
| Emergency care | EMTALA requires stabilization |
| No pre-existing condition exclusions | ACA protection |
| No lifetime limits | ACA protection |
| Preventive care | No cost-sharing for covered services |
| Appeal denials | Right to internal and external review |
| Surprise billing | No Surprises Act protection |
