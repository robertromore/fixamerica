## For Individuals and Families

### Enrolling in Medicaid

**Checking Eligibility**

| Step | Action |
|------|--------|
| Check income limits | Use KFF calculator or state website |
| Gather documents | ID, income proof, residency |
| Apply | healthcare.gov or state Medicaid agency |
| Follow up | Check status, respond to requests |
| Get help | Local navigators, FQHCs, community organizations |

**Eligibility Categories**

| Category | General Requirements |
|----------|----------------------|
| Children | Family income varies by state (typically 200%+ FPL) |
| Pregnant women | Income varies (typically 138%+ FPL) |
| Adults (expansion states) | Income under 138% FPL |
| Elderly/disabled | Asset and income limits; may need SSI/disability status |
| Parents (non-expansion) | Very low income limits (varies widely) |

### Maintaining Coverage

**Renewal Best Practices**

| Step | Action |
|------|--------|
| Update contact information | Keep current with state Medicaid agency |
| Watch for renewal notices | Check mail and state portals regularly |
| Respond promptly | Meet all deadlines |
| Report changes | Income, address, household size |
| Keep copies | Document all communications |

**If Coverage Is Terminated**

| Step | Action |
|------|--------|
| Understand reason | Request written explanation |
| Check for errors | Procedural vs. eligibility issues |
| Appeal if appropriate | Most states allow appeals |
| Reapply | If still eligible, apply immediately |
| Seek help | Navigators, legal aid, advocacy organizations |

### Using Medicaid Benefits

**Accessing Care**

| Challenge | Strategy |
|-----------|----------|
| Finding providers | Use state provider directories, FQHCs |
| Long wait times | Ask about cancellations, urgent appointments |
| Transportation | Use Medicaid non-emergency transportation benefit |
| Specialty care | Get referrals, be persistent |
| Dental care | Check if state covers adult dental |

**Appealing Denials**

| Step | Action |
|------|--------|
| Get denial in writing | Request formal notice |
| Understand basis | Medical necessity, coverage, procedural |
| File internal appeal | Within state deadline |
| Request fair hearing | State hearing process |
| Seek legal help | National Health Law Program resources |

## For Policymakers

### Federal Action

**Coverage Expansion**

| Policy | Mechanism |
|--------|-----------|
| Close coverage gap | Federal fallback coverage |
| Increase FMAP | Higher federal share for all states |
| Mandate expansion | Constitutional limits apply |
| Condition other funds | Use federal leverage |

**Program Improvement**

| Policy | Mechanism |
|--------|-----------|
| 12-month continuous eligibility | Mandate for all states |
| Automatic enrollment | Tax-data linking |
| Payment rate floors | Minimum Medicaid rates |
| HCBS infrastructure | ARPA-style investment |
| Administrative simplification | Reduce paperwork burden |

### State Action

**Coverage Policies**

| Policy | Approach |
|--------|----------|
| Expand Medicaid | Adopt ACA expansion |
| 12-month continuous eligibility | State plan option |
| Express lane eligibility | Use other program data |
| Presumptive eligibility | Immediate temporary coverage |
| Ex parte renewals | Automated renewal process |

**Delivery System**

| Policy | Approach |
|--------|----------|
| Managed care accountability | Strong contracts, oversight |
| Payment adequacy | Rate increases toward Medicare |
| HCBS expansion | Waiver development, state plan |
| Provider participation | Incentives, requirements |
| Quality improvement | Measure, report, reward |

## Staying Informed

### Key Dates and Deadlines

| Event | Timing |
|-------|--------|
| Medicaid enrollment | Year-round (no open enrollment period) |
| Renewal periods | 12-month cycle (varies by state) |
| State budget cycles | Typically annual (January-June in most states) |
| Federal budget | October fiscal year |
| State legislative sessions | Varies by state |
| 1115 waiver renewals | Varies by waiver |
