## For Individuals and Families

### Navigating Coverage

**Getting Covered**

| Situation | Action |
|-----------|--------|
| Uninsured | Check healthcare.gov or state exchange |
| Low income | Apply for Medicaid (income limits vary by state) |
| Recently unemployed | COBRA or special enrollment period |
| Turning 65 | Enroll in Medicare during initial period |
| Young adult | Stay on parent's plan until 26 |

**Choosing Plans**

| Factor | Consider |
|--------|----------|
| Premiums vs. deductibles | Lower premium = higher deductible usually |
| Network | Are your providers in-network? |
| Prescription coverage | Check formulary for your medications |
| Out-of-pocket max | Your maximum annual exposure |
| Subsidies | Calculate with KFF calculator |

**Maintaining Coverage**

| Action | Why |
|--------|-----|
| Update contact info | Don't miss renewal notices |
| Report income changes | Avoid subsidy reconciliation issues |
| Keep documents | Proof of coverage for taxes |
| Watch for notices | Respond to requests promptly |
| Appeal denials | Many denials are reversed |

### Reducing Costs

**Before Care**

| Strategy | How |
|----------|-----|
| Compare prices | Use cost tools, call providers |
| Ask about cash pay | Sometimes cheaper than insurance |
| Use in-network | Avoid out-of-network charges |
| Request generic | Same medication, lower cost |
| Preventive care | Free under ACA plans |

**During Care**

| Strategy | How |
|----------|-----|
| Question necessity | Is this test/procedure needed? |
| Get second opinions | For major decisions |
| Ask about alternatives | Less expensive options |
| Request itemized bills | Know what you're paying for |
| Negotiate | Many providers will reduce charges |

**After Care**

| Strategy | How |
|----------|-----|
| Review bills carefully | Errors are common |
| Dispute incorrect charges | Contact billing department |
| Request payment plans | Interest-free often available |
| Apply for charity care | Hospital financial assistance |
| Negotiate settlements | Offer lump sum for reduction |

### Protecting Your Rights

| Right | Action |
|-------|--------|
| **Appeal denials** | Internal appeal, then external review |
| **Surprise billing protection** | Report violations to state/CMS |
| **Mental health parity** | Document denials, file complaints |
| **Emergency care** | Cannot be denied for inability to pay |
| **Price estimates** | Request good faith estimates |

## For Advocates and Organizations

### State-Level Advocacy

**Priority Campaigns**

| Issue | Strategy |
|-------|----------|
| **Medicaid expansion** | Ballot initiatives in non-expansion states |
| **Public option** | Support state-level public options |
| **Drug pricing** | State prescription drug programs |
| **Surprise billing** | State consumer protections |
| **Mental health parity** | State enforcement, legislation |

**Non-Expansion States (2024)**

| States | Approach |
|--------|----------|
| Texas, Florida, Georgia | Legislative pressure, ballot initiatives |
| Tennessee, Alabama, Mississippi | Hospital coalition, rural focus |
| South Carolina, Kansas | Business case, economic argument |
| Wisconsin*, Wyoming | Technical expansion completion |

*Wisconsin covers to 100% FPL but hasn't formally expanded.

**Effective Tactics**

| Tactic | Application |
|--------|-------------|
| Personal stories | Constituent testimony |
| Economic arguments | Jobs, federal dollars, hospital survival |
| Rural focus | Hospital closures, access deserts |
| Bipartisan messengers | Business, faith, healthcare leaders |
| Media engagement | Local stories, editorials |

### Federal Advocacy

**Current Priorities**

| Issue | Ask |
|-------|-----|
| Enhanced ACA subsidies | Make permanent before 2025 |
| Coverage gap | Federal fallback for non-expansion states |
| Drug pricing | Expand Medicare negotiation |
| Public option | Federal public option legislation |
| Mental health | Parity enforcement funding |

**Engaging Congress**

| Action | How |
|--------|-----|
