# Healthcare: Roadmap

## Overview

Transforming American healthcare requires a sequenced approach that builds political momentum, demonstrates success, and gradually expands reform. This roadmap outlines phases from immediate actions through long-term transformation, recognizing that comprehensive change requires sustained effort over multiple election cycles.

---

## Phase 3: Structural Reform (3-5 Years)

### Public Option Implementation

| Element | Design Choice |
|---------|---------------|
| **Eligibility** | Start with uninsured, expand to all |
| **Payment rates** | Medicare rates + percentage |
| **Administration** | Federal (CMS) or state option |
| **Transition** | Phased enrollment, no mandates initially |

### Payment Reform at Scale

| Element | Implementation |
|---------|----------------|
| **Hospital global budgets** | Expand Maryland model nationally |
| **Primary care capitation** | Move PCPs to per-member-per-month |
| **Episode bundles** | Expand CMS programs |
| **Quality measurement** | Standardized national metrics |

### Market Reform

| Element | Implementation |
|---------|----------------|
| **Antitrust enforcement** | Increased DOJ/FTC resources, revised guidelines |
| **Provider consolidation limits** | Prospective merger review |
| **Insurer competition** | Public option as competitive benchmark |
| **Price transparency** | Mandatory, standardized, enforceable |

## Phase 4: Universal Coverage (5-10 Years)

### Pathways to Universal

| Pathway | Mechanism |
|---------|-----------|
| **Medicare buy-in expansion** | Gradually lower eligibility age |
| **Public option dominance** | Become default coverage |
| **Employer transition** | Employers opt into public system |
| **Medicaid integration** | Merge Medicaid into unified system |

### Financing Transition

| Element | Approach |
|---------|----------|
| **Premium replacement** | Payroll/income tax replaces premiums |
| **Employer contribution** | Maintain employer funding obligation |
| **Subsidy consolidation** | Unify tax exclusion, ACA subsidies, Medicaid |
| **Cost containment** | Global budgets, rate setting fund transition |

### Delivery System Integration

| Element | Approach |
|---------|----------|
| **Care coordination** | Integrated delivery systems |
| **Primary care foundation** | Medical home model universal |
| **Behavioral health integration** | Unified financing, delivery |
| **Long-term care** | LTSS benefit in unified system |

## Risks and Contingencies

### Political Risks

| Risk | Mitigation |
|------|------------|
| **Election losses** | Embed reforms structurally, build constituency |
| **Industry opposition** | Coalition building, public mobilization |
| **Court challenges** | Careful legal drafting, backup provisions |
| **State resistance** | Federal fallbacks, incentives |

### Implementation Risks

| Risk | Mitigation |
|------|------------|
| **Administrative capacity** | Phased implementation, pilot programs |
| **Provider resistance** | Transition support, payment adequacy |
| **Enrollment disruption** | No-wrong-door policies, navigation support |
| **IT failures** | Robust testing, backup systems |

### Economic Risks

| Risk | Mitigation |
|------|------------|
| **Recession** | Automatic stabilizers, countercyclical funding |
| **Cost overruns** | Budget controls, global budgets |
| **Provider shortages** | Workforce investment, scope expansion |
| **Market exits** | Public option fallback |

### Contingency Plans

| If This Fails | Then |
|---------------|------|
| Federal legislation blocked | State-level action |
| Public option defeated | Medicare buy-in |
| Medicaid expansion stalled | Federal coverage gap program |
| Payment reform resisted | Transparency + antitrust |

- National public option operational
- Payment reform at scale
- Administrative costs declining
- Quality metrics improving

- Universal coverage achieved
- Spending below 15% GDP
- Health outcomes match peer nations
- Disparities substantially reduced
