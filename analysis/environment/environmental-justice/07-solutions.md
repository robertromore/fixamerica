# Environmental Justice: Solutions

## Overview

Achieving environmental justice requires comprehensive reforms across permitting, enforcement, investment, participation, and legal frameworks. Solutions must address both preventing new harms and remediating historical injustice. This document outlines evidence-based solutions drawing on successful models and community-identified priorities.

## Cumulative Impact Requirements

### Federal Requirements

| Solution | Description |
|----------|-------------|
| **Cumulative impact assessment** | Require analysis in all permits |
| **Multiple source consideration** | Account for all pollution sources |
| **Health-based standards** | Community health outcomes focus |
| **Denial authority** | Allow permit denial based on cumulative burden |
| **Mandatory buffers** | Distance requirements from sensitive uses |
