# Conservation: Actions for Citizens

## Overview

Citizens can advance conservation through personal choices, volunteer work, advocacy, and political engagement. Individual actions contribute to both direct impact and building momentum for systemic change.

## Personal Actions

### Support Conservation

| Action | Impact |
|--------|--------|
| **Visit public lands** | Economic support |
| **Buy conservation stamps** | Direct funding |
| **Donate** | Support organizations |
| **Volunteer** | Direct work |
| **Learn** | Build knowledge |

| Action | Impact |
|--------|--------|
