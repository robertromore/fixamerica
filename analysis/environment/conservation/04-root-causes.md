# Conservation: Root Causes

## Overview

Conservation challenges stem from economic pressures, political dynamics, institutional limitations, and the inherent difficulty of protecting diffuse values against concentrated interests. Understanding these root causes is essential for effective solutions.

### Externalities

| Factor | Effect |
|--------|--------|
| **Ecosystem services** | Not priced |
| **Biodiversity** | Undervalued |

| **Public goods** | Free rider problem |
