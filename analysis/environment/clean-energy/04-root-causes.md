# Clean Energy: Root Causes

## Overview

While clean energy deployment is accelerating, systemic barriers slow the transition. Understanding the root causes of these barriers is essential for designing effective solutions.

## Market Failures

### Externalities

| Externality | Effect |
|-------------|--------|
| **Carbon emissions** | Not fully priced |
| **Air pollution** | Health costs externalized |
| **Water use** | Below full cost |
| **Land impacts** | Incompletely valued |

### Information Problems

| Problem | Effect |
|---------|--------|
| **Future costs** | Uncertainty discourages investment |
| **Technology risk** | First-mover disadvantage |
| **Performance data** | Incomplete information |
| **Long-term contracts** | Commitment difficulty |

### Coordination Failures

| Failure | Description |
|---------|-------------|
| **Grid planning** | Fragmented across regions |
| **Supply chain** | Uncoordinated development |
| **Workforce** | Training-job mismatch |
| **Infrastructure** | Chicken-and-egg problems |

## Regulatory Barriers

### Permitting

| Factor | Effect |
|--------|--------|
| **Multiple agencies** | Fragmented authority |
| **Environmental review** | Years of process |
| **Litigation risk** | Delays projects |
| **Local authority** | Variable, sometimes hostile |
| **Transmission siting** | State-by-state approval |

### Utility Regulation

| Factor | Effect |
|--------|--------|
| **Rate-of-return** | Favors capital spending |
| **Cost recovery** | Stranded asset concerns |
| **Vertical integration** | Limited competition |
| **Utility monopolies** | Incumbent advantage |

### Market Design

| Problem | Effect |
|---------|--------|
| **Energy-only markets** | Don't value flexibility |
| **Capacity markets** | May favor fossil fuels |
| **Interconnection** | Discriminatory practices |
| **Transmission cost** | Unfair allocation |

### Political Institutions

| Factor | Effect |
|--------|--------|

|--------|--------|

## Infrastructure Constraints

### Transmission

| Problem | Cause |
|---------|-------|
| **Insufficient capacity** | Underinvestment |
| **Planning fragmentation** | Regional boundaries |
| **Cost allocation** | Disputes |
| **Siting** | State-by-state process |
| **Technology** | HVDC underutilized |

### Distribution

| Problem | Cause |
|---------|-------|
| **Aging infrastructure** | Deferred investment |
| **Two-way flows** | Not designed for DERs |
| **Hosting capacity** | Limited in many areas |
| **Interconnection** | Slow processes |

### Grid Operations

| Problem | Cause |
|---------|-------|
| **Regional markets** | Incomplete coverage |
| **Interregional trading** | Limited |
| **Forecasting** | Still improving |
| **Flexibility** | Insufficient in some markets |
