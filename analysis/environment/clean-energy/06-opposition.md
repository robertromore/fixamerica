# Clean Energy: Opposition

## Overview

Opposition to clean energy deployment comes from fossil fuel interests, some utilities, local opposition movements, and ideological opponents of government energy policy. Understanding opposition arguments and tactics enables more effective advocacy.

## Primary Opposition

### Fossil Fuel Industry

| Actor | Activities |
|-------|------------|
| **Coal companies** | Lobbying, litigation |
| **Oil and gas** | Political donations, advocacy |
| **Trade associations** | Policy opposition |
| **Fossil fuel states** | State policy |

#### Industry Arguments

| Argument | Description |
|----------|-------------|
| **Reliability** | Renewables threaten grid stability |
| **Cost** | Subsidies waste taxpayer money |
| **Jobs** | Fossil fuel job losses |
| **Base load** | Need dispatchable power |
| **Energy security** | Domestic fossil fuels |

### Some Utilities

| Concern | Description |
|---------|-------------|
| **Stranded assets** | Unrecovered investments |
| **Rate impacts** | Customer cost concerns |
| **Reliability** | Operational challenges |
| **Business model** | DER disruption |

### Local Opposition

| Group | Concern |
|-------|---------|
| **NIMBY residents** | Visual impact, property values |
| **Wildlife advocates** | Bird/bat mortality |
| **Historic preservation** | Landscape impacts |
| **Agricultural interests** | Land use competition |

### Ideological Opposition

| Source | Argument |
|--------|----------|
| **Free market advocates** | Oppose subsidies |
| **Property rights groups** | Land use concerns |
| **Anti-government** | Federal overreach |
| **Climate skeptics** | Deny need |

## Opposition Arguments

### Reliability

| Claim | Counter-Evidence |
|-------|------------------|
| **Blackouts from renewables** | Fossil failures cause most outages |
| **Can't rely on wind/solar** | Grid integration proven |
| **Need baseload** | Flexible grid works |
| **Storage inadequate** | Rapid deployment |

### Cost

| Claim | Counter-Evidence |
|-------|------------------|
| **Expensive subsidies** | Fossil subsidies larger |
| **Rate increases** | Renewables often cheapest |
| **Hidden costs** | Lower lifecycle costs |
| **Taxpayer burden** | Economic benefits exceed |

### Environment

| Claim | Counter-Evidence |
|-------|------------------|
| **Bird deaths** | Far fewer than fossil, buildings |
| **Mining impacts** | Less than fossil extraction |
| **Land use** | Smaller footprint than coal |
| **Disposal** | Recyclable, manageable |

### Economy

| Claim | Counter-Evidence |
|-------|------------------|
| **Job losses** | Clean energy creates more jobs |
| **Trade deficit** | Domestic manufacturing growing |
| **Rural harm** | Rural areas benefit |
| **Competitiveness** | Clean energy advantage |

| State Type | Opposition |
|------------|------------|

| **Oil/gas states** | Policy resistance |
| **Utility monopoly states** | Slow transition |

### Local

| Context | Opposition |
|---------|------------|
| **Wealthy suburbs** | Aesthetic concerns |
| **Rural areas** | Mixed (some benefit) |
| **Coastal** | Offshore wind opposition |
| **Agricultural** | Land competition |

## Specific Technology Opposition

### Wind

| Concern | Response |
|---------|----------|
| **Noise** | Modern turbines quiet |
| **Visual** | Subjective, siting matters |
| **Birds/bats** | Mitigation effective |
| **Property values** | Studies show minimal impact |

### Solar

| Concern | Response |
|---------|----------|
| **Land use** | Dual use, brownfields |
| **Agricultural** | Agrivoltaics viable |
| **Glare** | Siting addresses |
| **Fire risk** | Manageable with codes |

### Transmission

| Concern | Response |
|---------|----------|
| **Eminent domain** | Compensation, careful routing |
| **EMF** | No health impacts proven |
| **Visual** | Burial option, design |
| **Property** | Easement compensation |

### Market Reality

|---------|---------|

| **Job creation** | Recognized benefit |

### Political Trends

| Trend | Impact |
|-------|--------|

| **Bipartisan support** | Some areas |
