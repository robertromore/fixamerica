# Pollution: History

## Overview

U.S. pollution control evolved from local nuisance laws to comprehensive federal regulation following visible environmental crises in the 1960s and 1970s. Major legislation established frameworks that have achieved significant improvements while leaving persistent challenges.

## Legislative History

### Clean Air Act

| Amendment | Key Changes |
|-----------|-------------|
| **1970** | NAAQS, SIPs, mobile source standards |
| **1977** | PSD, nonattainment, visibility |
| **1990** | Title V permits, acid rain, air toxics |

### Clean Water Act

| Development | Key Changes |
|-------------|-------------|
| **1972** | NPDES permits, technology standards |
| **1977** | Toxic pollutant focus |
| **1987** | Nonpoint source, stormwater |

### Superfund

| Development | Change |
|-------------|--------|
| **1980** | Original enactment |
| **1986** | SARA strengthening |
| **1990s** | Liability refinements |
| **Tax expiration** | 1995, restored 2021 |

## Progress and Setbacks

### Major Improvements

| Area | Progress |
|------|----------|
| **Lead in air** | 99%+ reduction |
| **SO2** | 90%+ reduction |
| **Point source water** | Significant improvement |
| **Acid rain** | Major reduction |
| **Vehicle emissions** | Dramatic per-mile reduction |

### Persistent Challenges

| Area | Challenge |
|------|-----------|
| **Ozone** | Continued non-attainment |
| **PM2.5** | Health impacts continue |
| **Nonpoint water** | Limited progress |
| **Toxics** | Emerging chemicals |
| **Environmental justice** | Persistent disparities |

## Lessons from History

| Lesson | Description |
|--------|-------------|
| **Regulation works** | Standards drive improvement |
| **Technology responds** | Industry adapts |
| **Enforcement matters** | Compliance requires oversight |
| **New challenges emerge** | Continuous vigilance needed |
| **Equity lags** | Justice requires focus |
