# Agriculture Policy: Solutions

## Overview

Transforming American agriculture requires addressing market concentration, environmental degradation, labor exploitation, and barriers to entry simultaneously. The solutions below are organized into comprehensive reforms that build on each other and can be implemented through existing authorities, farm bill reauthorization, and complementary legislation.

---

## Reform Framework

### Guiding Principles

| Principle | Application |
|-----------|-------------|
| **Equity** | Target benefits to those who need them; address historical discrimination |
| **Sustainability** | Link support to environmental stewardship |
| **Competition** | Restore fair markets; break up concentrated power |
| **Dignity** | Ensure fair treatment for workers |
| **Resilience** | Build systems that withstand shocks |
| **Access** | Lower barriers for beginning and socially disadvantaged farmers |
| **Transparency** | Make markets, subsidies, and practices visible |

### Strategic Framework

| Strategy | Components |
|----------|------------|
| **Reform farm programs** | Payment limits, conservation linkage, equity |
| **Restore competition** | Antitrust enforcement, regional processing, transparency |
| **Protect workers** | Labor rights, immigration reform, safety |
| **Conserve resources** | Soil health, water, climate adaptation |
| **Support new farmers** | Land access, credit, training |
| **Build infrastructure** | Broadband, processing, water |

## Proposed Solutions

### Solution 1: Farm Program Equity Reform

**Description:** Restructure commodity programs to target support to small and midsize operations, eliminate loopholes, and link benefits to conservation.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Payment caps** | $250,000 hard cap per entity; close entity splitting loopholes |
| **Means testing** | Phase out payments above $900,000 AGI |
| **Conservation compliance** | Strengthen sodbuster/swampbuster; add soil health metrics |
| **Beginning farmer priority** | 25% set-aside in all programs |
| **Socially disadvantaged farmer priority** | Dedicated funding; address discrimination |

**Evidence:**

| Finding | Source |
|---------|--------|
| Top 10% receive 77% of payments | EWG Farm Subsidy Database |
| Payment limits easily circumvented | GAO reports |
| Conservation compliance works | NRCS evaluations |
| Beginning farmers need support | Census of Agriculture |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Large operation opposition | Build small/midsize coalition |
| Definition of entities | Clear statutory definitions; IRS coordination |
| Political difficulty | Frame as fairness; rural community support |

### Solution 2: Crop Insurance Reform

**Description:** Reform the federal crop insurance program to improve equity, link to conservation, and reduce costs while maintaining essential risk protection.

**Key Components:**

| Component | Details |
|-----------|---------|
| **Graduated premium subsidies** | 80% for first $50K; declining to 40% above $500K |
| **Conservation premium discount** | 5-10% discount for verified practices |
| **Actuarial reform** | Risk-rate marginal lands; discourage repeated losses |
| **Competition** | Allow more providers; reduce administrative costs |
| **Transparency** | Public database of premium subsidies by farm |

**Evidence:**

| Finding | Source |
|---------|--------|
| 62% average premium subsidy | FCIC data |
| Benefits concentrated in large operations | GAO analysis |
| Conservation linkage feasible | Pilot programs |
| Actuarial unsoundness in some areas | Academic research |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Crop insurance industry opposition | Maintain profit opportunity; improve targeting |
| Farmer concerns | Phase in changes; protect small operations |
| Measurement complexity | Use existing conservation verification |

| **Merger review** | Block further consolidation; consider breakups |
| **Regional processing investment** | $10+ billion for regional capacity |
| **Right to repair** | Require equipment repairability |
| **Price transparency** | Mandatory cash market transactions; reporting |
| **Contract fairness** | Ban retaliatory practices; require fair terms |

**Evidence:**

| Finding | Source |

### Solution 5: Farmworker Rights and Protections

**Description:** End the historical exclusion of farmworkers from labor protections, ensure fair treatment, and create workable immigration pathways.

**Key Components:**

| Component | Details |
|-----------|---------|
| **NLRA inclusion** | Extend organizing rights to farmworkers |
| **Overtime coverage** | Phase in overtime requirements |
| **Heat safety standard** | OSHA mandatory heat illness prevention |
| **H-2A reform** | Portable visas, independent housing, enforcement |
| **Immigration pathway** | Earned legalization for agricultural workers |
| **Child labor reform** | Close agricultural hazardous work exemptions |

**Evidence:**

| Finding | Source |
|---------|--------|
| Farmworkers excluded from NLRA since 1935 | 29 U.S.C. 152(3) |
| California ALRA demonstrates viability | State experience |
| Heat deaths increasing | OSHA data |
| H-2A violations widespread | DOL enforcement |

**Challenges:**

| Challenge | Mitigation |
|-----------|------------|
| Grower opposition | Emphasize workforce stability |
| Immigration politics | Bipartisan agricultural labor coalition |
| State variation | Federal floor with state supplements |
