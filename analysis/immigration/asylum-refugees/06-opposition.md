# Asylum and Refugees: Opposition

## Overview

Opposition to asylum and refugee protection comes from those who believe the system is abused, those concerned about security, those opposing immigration generally, and those who prioritize enforcement over protection. Understanding these arguments is essential for developing effective and sustainable policy.

## "Asylum Abuse" Arguments

### Core Claims

| Claim | Assertion |
|-------|-----------|
| **Most claims non-meritorious** | Economic migrants gaming system |
| **Coached claims** | Smugglers teach what to say |
| **Low denial consequences** | Years in US even if denied |
| **Magnet effect** | System encourages migration |
| **Catch and release** | No real enforcement |

### Evidence Cited

| Evidence | Interpretation |
|----------|----------------|
| **Grant rates** | 30-40% means majority denied |
| **Border arrivals** | Asylum as entry method |
| **Credible fear passage** | Too easy to pass |
| **No-show rates** | Absconding (disputed data) |

### Policy Preferences

| Preference | Rationale |
|------------|-----------|
| **Raise credible fear standard** | Screen out weak claims |
| **Mandatory detention** | Ensure appearance |
| **Expedited processing** | Quick decisions |
| **Third country bars** | Process elsewhere |
| **Asylum limits** | Cap annual grants |

## Security Concerns

### Terrorist Threat Arguments

| Concern | Claim |
|---------|-------|
| **Unknown backgrounds** | Can't verify identity |
| **Infiltration risk** | Terrorists pose as refugees |
| **Insufficient vetting** | Not thorough enough |
| **Country conditions** | From conflict zones |

### Actual Security Record

| Fact | Data |
|------|------|
| **Refugee terrorism** | Near zero incidents |
| **Vetting process** | Most thorough of any arrival |
| **Domestic radicalization** | Greater threat |
| **9/11 attackers** | None were refugees/asylum seekers |

### Criminal Concern Arguments

| Concern | Claim |
|---------|-------|
| **Criminal element** | Some asylum seekers have records |
| **Smuggling networks** | Criminal organizations profit |
| **Unscreened arrivals** | Between ports |

## Economic Arguments

### Fiscal Burden Claims

| Claim | Assertion |
|-------|-----------|
| **Welfare use** | Refugees use public benefits |
| **Resettlement costs** | Expensive programs |
| **Labor market** | Compete with low-wage workers |
| **Local impact** | Communities bear costs |

### Counter-Evidence

| Fact | Data |
|------|------|
| **Refugee employment** | High labor force participation |
| **Tax contributions** | Positive fiscal impact long-term |
| **Economic contribution** | Entrepreneurship, innovation |
| **Community development** | Revitalize declining areas |
