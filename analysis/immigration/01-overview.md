# Immigration: Overview

## Executive Summary

American immigration policy is broken across virtually every dimension. Legal immigration backlogs stretch decades, the asylum system is overwhelmed, border enforcement is both ineffective and inhumane, millions of unauthorized immigrants live in legal limbo, and immigration courts face crushing caseloads without basic due process protections. Meanwhile, the economic and demographic case for immigration remains strongâ€”immigration drives innovation, fills labor shortages, and supports fiscal sustainability. Comprehensive reform requires addressing legal pathways, enforcement, unauthorized populations, and integration simultaneously, but partisan polarization has blocked action for decades.

## Scope

This analysis examines the full range of immigration policy:

- **Legal immigration**: Visa categories, backlogs, naturalization
- **Border security**: Enforcement, technology, ports of entry
- **Asylum and refugees**: Protection, processing, resettlement
- **Interior enforcement**: Worksite, deportation, priorities
- **Unauthorized population**: DACA, legalization, status
- **Immigration courts**: Backlog, due process, reform
- **Employment-based**: Skilled workers, labor market needs
- **Family-based**: Reunification, backlogs, reform
- **Detention**: Conditions, alternatives, oversight
- **Integration**: Naturalization, language, economic outcomes

## The Immigration Challenge

### Systemic Dysfunction

| Dimension | Problem |
|-----------|---------|
| **Legal pathways** | Multi-decade backlogs, outdated categories |
| **Asylum** | System designed for thousands processing millions |
| **Border** | Enforcement without legal alternatives |
| **Courts** | No due process, crushing backlog |
| **Unauthorized** | 11 million in permanent limbo |
| **Integration** | Insufficient support, barriers |

### Consequences

| Impact | Description |
|--------|-------------|
| **Human cost** | Family separation, deaths, exploitation |
| **Economic loss** | Talent unable to contribute legally |
| **Rule of law** | System that can't be followed |
| **Political** | Toxic issue blocking broader progress |
| **International** | Damaged relationships, credibility |

## Root Causes

| Factor | Impact |
|--------|--------|
| **1965 system design** | Built for different era |
| **Per-country limits** | Create arbitrary backlogs |
| **Congressional deadlock** | No comprehensive reform since 1996 |
| **Executive whiplash** | Policy changes every administration |
| **Insufficient resources** | Courts, processing chronically underfunded |
| **Geopolitical shifts** | Push factors beyond US control |

| Element | Approach |
|---------|----------|
| **Legal immigration** | Clear backlogs, modernize categories |
| **Border** | Effective processing, regional cooperation |
| **Asylum** | Functional system with fair adjudication |
| **Unauthorized** | Earned legalization pathway |

| **Integration** | Citizenship, language, economic support |

## Why This Matters

Immigration shapes America's economy, demographics, culture, and global standing. Getting it right is essential for:

- **Economic competitiveness**: Innovation, labor force, entrepreneurship
- **Fiscal sustainability**: Working-age population, tax base
- **Family unity**: Core American value
- **Humanitarian obligations**: Refugee protection commitments
- **National security**: Know who's here, orderly system
- **Rule of law**: Laws people can actually follow
