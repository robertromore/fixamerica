# Employment-Based Immigration: Root Causes

## Structural Root Causes

### Outdated Numerical Limits

| Problem | Impact |
|---------|--------|
| **1990 cap** | 140,000 not updated |
| **Per-country limit** | 7% cap per country |
| **H-1B cap** | 65,000 from 1990 |
| **No adjustment** | 35+ years static |
| **Dependents counted** | Against caps |

### Demand-Supply Mismatch

| Reality | Law |
|---------|-----|
| **Tech economy** | Massive skilled need |
| **Global talent** | Available |
| **Legal pathways** | Severely limited |
| **Result** | Lottery, backlog |

### System Design Flaws

| Flaw | Consequence |
|------|-------------|
| **Employer-based** | Worker dependence |
| **Multi-step process** | Years to completion |
| **No self-petition** | Limited to sponsored |
| **Rigid categories** | Poor fit for economy |

## Per-Country Cap Problem

### Design Intent vs. Outcome

| Intent | Outcome |
|--------|---------|
| **Diversity** | Punishes large countries |
| **Prevent dominance** | Creates massive backlogs |
| **Equal opportunity** | Unequal waits |

### Mathematical Problem

| Country | Demand | Supply (7%) |
|---------|--------|-------------|
| **India** | 50%+ of applicants | 9,800 visas |
| **China** | 10% of applicants | 9,800 visas |
| **Others** | 40% of applicants | 120,400 visas |

### Consequences

| Impact | Description |
|--------|-------------|
| **Decades of waiting** | India: 30-50+ years |
| **Brain drain** | Leave for other countries |
| **Career constraints** | Cannot advance |
| **Family separation** | Children age out |

## H-1B System Problems

### Lottery Dysfunction

| Problem | Impact |
|---------|--------|
| **Random selection** | Not merit-based |
| **Multiple filing** | Gaming system |
| **Employer uncertainty** | Cannot plan |
| **Worker precarity** | Status unpredictable |

### Wage Level Issues

| Problem | Impact |
|---------|--------|
| **Level I prevalent** | Below market |
| **Outsourcing model** | Labor arbitrage |
| **Compliance gaps** | Under-enforcement |
| **Prevailing wage** | Outdated methodology |

### Employer Dependence

| Problem | Impact |
|---------|--------|
| **Tied to sponsor** | Limited mobility |
| **Retaliation risk** | Cannot complain |
| **Exploitation** | Wage theft, abuse |
| **Portability limited** | Bureaucratic process |

## Green Card Backlog Causes

### Insufficient Numbers

| Factor | Impact |
|--------|--------|
| **Static cap** | Not adjusted for economy |
| **Dependents** | Count against cap |
| **Unused visas** | Lost, not recaptured |
| **Category spill** | Inefficient allocation |

### Processing Delays

| Stage | Delay |
|-------|-------|
| **PERM** | 6-18+ months |
| **I-140** | 6-12+ months |
| **Priority date** | Years to decades |
| **I-485** | 6-24+ months |

### PERM Process

| Problem | Impact |
|---------|--------|
| **Recruitment fiction** | Market test formality |
| **Documentation** | Burdensome |
| **Audit risk** | Significant delay |
| **Employer reluctance** | Cost, time |

## Systemic Analysis

### Feedback Loops

| Loop | Effect |
|------|--------|
| **Backlog grows** | Workers trapped |
| **Trapped workers** | Cannot complain |
| **Low wages** | Continue |
| **More H-1B demand** | Cycle continues |

### System Incentives

| Incentive | Behavior |
|-----------|----------|
| **Cheap labor** | Employer prefers H-1B |
| **Worker dependence** | Accepts conditions |
| **Lottery gaming** | Multiple filings |
| **Backlog** | Prevents competition |
