# Employment-Based Immigration: Current State

## H-1B Specialty Occupation Visa

### Program Basics

| Element | Specification |
|---------|---------------|
| **Annual cap** | 65,000 regular + 20,000 advanced degree |
| **Duration** | 3 years, renewable to 6 |
| **Requirement** | Bachelor's degree or equivalent |
| **Specialty occupation** | Professional-level work |
| **Employer** | Must petition |

### Application Statistics

| Fiscal Year | Applications | Cap | Ratio |
|-------------|--------------|-----|-------|
| **FY 2024** | 780,000+ | 85,000 | 9:1 |
| **FY 2023** | 483,000 | 85,000 | 5.7:1 |
| **FY 2022** | 308,000 | 85,000 | 3.6:1 |
| **FY 2021** | 275,000 | 85,000 | 3.2:1 |

### Cap-Exempt Employers

| Employer Type | Status |
|---------------|--------|
| **Universities** | No cap |
| **Nonprofit research** | No cap |
| **Government research** | No cap |
| **H-1B affiliated** | No cap |

## L-1 Intracompany Transfer

### Program Basics

| Element | Specification |
|---------|---------------|
| **Cap** | None |
| **L-1A** | Managers, executives |
| **L-1B** | Specialized knowledge |
| **Duration** | L-1A: 7 years, L-1B: 5 years |
| **Requirement** | 1 year with company abroad |

### Annual Statistics

| Metric | Data |
|--------|------|
| **Annual admissions** | ~75,000 |
| **L-1A** | ~40% |
| **L-1B** | ~60% |
| **Denial rate** | 20-30% |

### Criticism

| Issue | Concern |
|-------|---------|
| **Specialized knowledge** | Vague standard |
| **Blanket petitions** | Less scrutiny |
| **Outsourcing** | Replacement workers |
| **Wage rules** | Less stringent than H-1B |

## Labor Certification (PERM)

### Process Overview

| Step | Timeline |
|------|----------|
| **Prevailing wage request** | 2-6 months |
| **Recruitment** | 30-180 days |
| **Application filing** | Preparation |
| **Processing** | 6-18 months |
| **Audit** | +6-12 months if selected |

### Prevailing Wage

| Source | Method |
|--------|--------|
| **DOL OES** | BLS data |
| **Survey** | Employer-provided |
| **Level determination** | I-IV scale |

### Recruitment Requirements

| Element | Requirement |
|---------|-------------|
| **Job posting** | 30 days on job board |
| **Newspaper ads** | 2 Sunday editions |
| **Additional steps** | Professional, campus, etc. |
| **Documentation** | All applications |
