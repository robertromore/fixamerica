# Employment-Based Immigration: Stakeholders

## Primary Stakeholders

### Foreign Workers

| Category | Characteristics |
|----------|-----------------|
| **H-1B workers** | 500,000+ in US |
| **Green card waiting** | 1+ million in backlog |
| **Dependents** | Spouses, children |
| **F-1 students** | OPT pathway |
| **L-1 transfers** | Company employees |

### Worker Interests

| Interest | Description |
|----------|-------------|
| **Status security** | Stable authorization |
| **Mobility** | Job change ability |
| **Path to PR** | Green card access |
| **Family unity** | Spouse work, children status |
| **Fair wages** | Market compensation |

### Worker Segments

| Segment | Primary Concern |
|---------|-----------------|
| **Recent H-1B** | Lottery, status |
| **Green card waiting** | Backlog, per-country |
| **Long-term backlog** | Aging out children |
| **H-4 spouses** | Work authorization |
