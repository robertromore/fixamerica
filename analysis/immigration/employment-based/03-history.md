# Employment-Based Immigration: History

## Early Employment Immigration

### Pre-1965 Era

| Period | Characteristic |
|--------|----------------|
| **Open immigration** | No worker categories |
| **National origins** | 1924 quota system |
| **Labor needs** | Bracero program |
| **Professional** | Limited categories |

### 1952 Immigration and Nationality Act

| Element | Description |
|---------|-------------|
| **Preference system** | Some employment preferences |
| **Quota continuation** | National origins maintained |
| **Professional** | Skilled preference |
| **Limited scope** | Family still dominant |

## Hart-Celler Act (1965)

### Employment Categories

| Change | Impact |
|--------|--------|
| **Preference categories** | Employment-based created |
| **Third preference** | Professionals, scientists |
| **Sixth preference** | Skilled workers |
| **Western Hemisphere** | Caps applied |

### Early Implementation

| Aspect | Result |
|--------|--------|
| **Professional intake** | Gradual increase |
| **Geographic shift** | Asia, Latin America |
| **Skill composition** | Higher education levels |

## Immigration Act of 1990

### Major Reforms

| Change | Impact |
|--------|--------|
| **EB categories** | 5 categories created |
| **140,000 cap** | Significant increase |
| **H-1B created** | Specialty occupation visa |
| **Diversity visa** | New category |

### Employment-Based System Design

| Category | 1990 Allocation |
|----------|-----------------|
| **EB-1** | 40,000 |
| **EB-2** | 40,000 |
| **EB-3** | 40,000 |
| **EB-4** | 10,000 |
| **EB-5** | 10,000 |

### H-1B Program

| Element | 1990 Design |
|---------|-------------|
| **Cap** | 65,000 |
| **Duration** | 6 years |
| **Requirements** | Bachelor's + specialty |
| **Labor condition** | LCA required |

## 1990s: Tech Boom and H-1B

### Growing Demand

| Year | H-1B Cap | Demand |
|------|----------|--------|
| **1990-1997** | 65,000 | Met demand |
| **1998** | 65,000 | Cap hit early |
| **1999** | 115,000 | Temporary increase |
| **2000** | 195,000 | Y2K, dot-com |

### American Competitiveness Act (1998)

| Change | Impact |
|--------|--------|
| **Cap increase** | To 115,000 |
| **Fees** | $500 training fee |
| **Temporary** | 3-year increase |

### AC21 (2000)

| Change | Impact |
|--------|--------|
| **Cap** | Raised to 195,000 |
| **H-1B portability** | Job changes easier |
| **H-1B extension** | Beyond 6 years if backlog |
| **Cap-exempt** | Universities, research |
