# Border Security: Current State

## Overview

The current border security situation is defined by record levels of irregular migration encounters, overwhelmed processing capacity, ongoing debates about barriers versus technology, strained resources, and complex regional dynamics. Ports of entry handle massive legitimate flows while Border Patrol manages unprecedented migration between ports.
