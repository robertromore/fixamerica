# Family-Based Immigration: Root Causes

## Structural Root Causes

### Outdated Numerical Limits

| Problem | Impact |
|---------|--------|
| **1990 numbers** | Not adjusted |
| **226,000 preference** | Insufficient for demand |
| **Population growth** | US grown, numbers frozen |
| **Global demand** | Far exceeds supply |

### Per-Country Cap Design

| Problem | Impact |
|---------|--------|
| **7% limit** | ~25,000 per country |
| **Equal treatment** | Ignores demand differences |
| **Large countries** | Massive backlogs |
| **Small countries** | Often current |

### Arithmetic Problem

| Country | Demand | Supply |
|---------|--------|--------|
| **Mexico** | 1.3 million waiting | 25,000/year |
| **Philippines** | 400,000 waiting | 25,000/year |
| **India** | 300,000 waiting | 25,000/year |

### Category Complexity

| Issue | Impact |
|-------|--------|
| **Multiple categories** | Confusing |
| **Status changes** | Category shifts |
| **Marriage effect** | Changes eligibility |
| **Aging out** | Children lose place |

## Legal Barrier Root Causes

### Unlawful Presence Bars

| Bar | Effect |
|-----|--------|
| **3-year** | Separation from family |
| **10-year** | Decade apart |
| **Catch-22** | Cannot leave to process |
| **Waiver difficulty** | Extreme hardship |

### Entry Without Inspection

| Problem | Impact |
|---------|--------|
| **Cannot adjust** | Must leave US |
| **Leaving triggers bar** | Trapped |
| **Section 245(i)** | Expired |
| **No solution** | For most EWI |

### Mixed-Status Families

| Cause | Effect |
|-------|--------|
| **Different statuses** | Within same family |
| **Legal barriers** | To reunification |
| **Enforcement** | Separation risk |
| **Policy gaps** | No comprehensive solution |

## Processing Root Causes

### USCIS Backlogs

| Problem | Impact |
|---------|--------|
| **I-130 delays** | 12-24 months |
| **Staffing** | Insufficient |
| **Fee structure** | Self-funded model |
| **Premium unavailable** | No expedite option |

### Consular Processing

| Problem | Impact |
|---------|--------|
| **NVC delays** | Additional months |
| **Interview slots** | Limited availability |
| **Country variation** | Inconsistent |
| **Post-COVID** | Still recovering |

### Administrative Burden

| Burden | Impact |
|--------|--------|
| **Documentation** | Extensive requirements |
| **Translation** | Costly |
| **Medical exams** | Expiration issues |
| **Multiple steps** | Process failures |

## Political Root Causes

### "Chain Migration" Framing

| Framing | Impact |
|---------|--------|
| **Pejorative term** | Poisons debate |
| **Extended family** | Targeted |
| **Political attacks** | On family categories |
| **Reform blocked** | Expansion impossible |

### Merit vs. Family Debate

| Position | Argument |
|----------|----------|
| **Merit-based** | Skills over family |
| **Family-based** | Core American value |
| **Compromise** | Both sides lose |
| **Stalemate** | No reform |

### Immigration Politics

| Factor | Impact |
|--------|--------|
| **Polarization** | No bipartisan reform |
| **Nativism** | Opposes all immigration |
| **Border focus** | Diverts attention |
| **Enforcement first** | Delays family reform |

## Systemic Analysis

### Feedback Loops

| Loop | Effect |
|------|--------|
| **Backlog grows** | Wait increases |
| **Families separated** | Pressure to enter illegally |
| **Illegal entry** | Bars apply |
| **Mixed-status** | More families affected |

### Design Flaws

| Flaw | Consequence |
|------|-------------|
| **Fixed numbers** | Cannot respond to demand |
| **Per-country** | Punishes large countries |
| **Category rigidity** | Life changes cause problems |
| **No review** | 35+ years unchanged |

### Competing Interests

| Interest | Position |
|----------|----------|
| **Reunification** | More visas |
| **Restriction** | Fewer visas |
| **Skills focus** | Shift from family |
| **Enforcement** | Address illegal first |
| **Result** | No action |
