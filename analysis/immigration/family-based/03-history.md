# Family-Based Immigration: History

## Early Family Immigration

### Pre-1924 Era

| Period | Characteristic |
|--------|----------------|
| **Open immigration** | No family categories needed |
| **Chain migration** | Natural pattern |
| **Community networks** | Family facilitated |
| **No restrictions** | Until 1882+ |

### National Origins Era (1924-1965)

| Policy | Impact |
|--------|--------|
| **Quota system** | Limited by nationality |
| **Family preference** | Some provisions |
| **Western Hemisphere** | Initially unrestricted |
| **Family pattern** | Continued informally |

## Hart-Celler Act (1965)

### Family Preference Creation

| Change | Description |
|--------|-------------|
| **End national origins** | New system |
| **Family as principle** | Primary category |
| **Preference categories** | 4 family categories |
| **Immediate relatives** | Uncapped |

### 1965 Family Categories

| Category | Description |
|----------|-------------|
| **1st preference** | Unmarried adult children of citizens |
| **2nd preference** | Spouses, children of LPRs |
| **4th preference** | Married children of citizens |
| **5th preference** | Siblings of citizens |

### Unexpected Consequences

| Outcome | Description |
|---------|-------------|
| **Asian immigration** | Dramatic increase |
| **Family chains** | Extended networks |
| **Western Hemisphere** | New caps applied |
| **Backlogs** | Began growing |

## 1976 Amendments

### Western Hemisphere Changes

| Change | Impact |
|--------|--------|
| **Preference system** | Extended to Americas |
| **Per-country caps** | Applied worldwide |
| **20,000 limit** | Per country maximum |
| **Mexico impact** | Immediate backlog |

### Growing Backlogs

| Decade | Development |
|--------|-------------|
| **1970s** | Initial accumulation |
| **1980s** | Significant growth |
| **1990s** | Multi-year waits |

## Immigration Act of 1990

### Family Category Restructuring

| Change | Impact |
|--------|--------|
| **F1** | Unmarried children of citizens |
| **F2A** | Spouses/children of LPRs |
| **F2B** | Unmarried children of LPRs |
| **F3** | Married children of citizens |
| **F4** | Siblings of citizens |

### Numerical Changes

| Category | 1990 Allocation |
|----------|-----------------|
| **Immediate relatives** | Uncapped |
| **Family preference** | 226,000 (floor) |
| **Diversity** | New category |

### Per-Country Cap

| Element | Specification |
|---------|---------------|
| **Limit** | 7% per country |
| **Application** | Preference categories |
| **Effect** | ~25,000 maximum |

## 1996 IIRIRA

### Unlawful Presence Bars

| Bar | Trigger |
|-----|---------|
| **3-year** | 180 days - 1 year unlawful |
| **10-year** | 1+ year unlawful |
| **Permanent** | Reentry after removal |

### Impact on Families

| Effect | Description |
|--------|-------------|
| **Mixed-status** | Trapped in US |
| **Catch-22** | Leave triggers bar |
| **Hardship waiver** | Difficult standard |
| **Family separation** | Increased |

### Affidavit of Support

| Change | Requirement |
|--------|-------------|
| **125% poverty** | Income requirement |
| **Legally enforceable** | Contract obligation |
| **10-year duration** | Until citizenship/40 quarters |

## 2000s: V Visa and CSPA

### LIFE Act (2000)

| Element | Description |
|---------|-------------|
| **V visa** | For F2A waiting 3+ years |
| **Section 245(i)** | Adjustment extension |
| **Limited** | Sunset provisions |

### V Visa Expiration

| Issue | Impact |
|-------|--------|
| **Cutoff** | December 21, 2000 petitions |
| **No renewal** | Program ended |
| **Family separation** | Continued |

### Child Status Protection Act (2002)

| Purpose | Mechanism |
|---------|-----------|
| **Prevent aging out** | Age calculation |
| **Subtract processing** | Petition wait |
| **Seek to acquire** | 1-year requirement |
| **Limitations** | Not all covered |
