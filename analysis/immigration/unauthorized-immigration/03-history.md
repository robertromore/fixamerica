# Unauthorized Immigration: History

## Pre-1965: Open and Restricted Borders

### Early Period (Pre-1882)

| Era | Characteristic |
|-----|----------------|
| **Colonial** | Open immigration |
| **Early republic** | No federal restrictions |
| **Mid-1800s** | Mass European immigration |
| **Concept of "illegal"** | Did not exist |

### First Restrictions (1882-1924)

| Law | Impact |
|-----|--------|
| **Chinese Exclusion Act (1882)** | First race-based restriction |
| **Immigration Act of 1891** | Federal control, deportation |
| **Gentlemen's Agreement (1907)** | Japanese restrictions |
| **Immigration Act of 1917** | Literacy test, Asian bar |

### Quota Era (1924-1965)

| Development | Impact |
|-------------|--------|
| **National Origins Act (1924)** | Strict quotas |
| **Border Patrol (1924)** | Formal enforcement |
| **Mexican exception** | Not subject to quotas |
| **Bracero Program (1942-1964)** | Guest worker program |

## Bracero Program (1942-1964)

### Program Design

| Element | Description |
|---------|-------------|
| **Purpose** | Wartime labor shortage |
| **Scale** | 4.6 million contracts |
| **Duration** | 22 years |
| **Industries** | Agriculture primarily |
| **Legal status** | Temporary, documented |

### Operation Wetback (1954)

| Aspect | Description |
|--------|-------------|
| **Purpose** | Remove unauthorized Mexicans |
| **Scale** | 1+ million deported |
| **Methods** | Mass roundups, deportations |
| **Conditions** | Often abusive |
| **Legacy** | Controversial precedent |

### Program End

| Factor | Impact |
|--------|--------|
| **Labor abuses** | Worker exploitation |
| **Civil rights movement** | Opposition to guest workers |
| **Union opposition** | Wage depression |
| **Program terminated** | 1964 |
| **Consequence** | Continued migration, now unauthorized |

## Hart-Celler Act (1965)

### New System

| Change | Description |
|--------|-------------|
| **End quotas** | Replaced national origins |
| **Hemispheric caps** | New limits imposed |
| **Family preference** | Prioritized reunification |
| **Western Hemisphere cap** | First limits on Americas |

### Unintended Consequences

| Consequence | Impact |
|-------------|--------|
| **Mexican cap** | Previous circular flow disrupted |
| **Legality shift** | Previously legal became unauthorized |
| **Family backlogs** | Multi-decade waits |
| **Unauthorized growth** | From 1970s onward |

## 1970s-1980s: Rising Unauthorized Population

### Population Growth

| Year | Estimated Unauthorized |
|------|----------------------|
| **1965** | Minimal |
| **1970** | 540,000 |
| **1980** | 3 million |
| **1986** | 5 million |

### Contributing Factors

| Factor | Impact |
|--------|--------|
| **Mexican cap** | 20,000 vs. previous unlimited |
| **Economic demand** | US labor market pull |
| **Mexican crisis** | 1980s peso crisis push |
| **Weak enforcement** | Little interior enforcement |
| **Employer demand** | Agriculture, services |

## Immigration Reform and Control Act (IRCA, 1986)

### Legalization Program

| Element | Description |
|---------|-------------|
| **General amnesty** | Pre-1982 residents |
| **SAW program** | Agricultural workers |
| **Total legalized** | 2.7 million |
| **Citizenship** | Many eventually naturalized |

### IRCA Requirements

| Requirement | Standard |
|-------------|----------|
| **Continuous residence** | Since January 1, 1982 |
| **Admissible** | No criminal bars |
| **Minimal English** | Or civics course |
| **Fee** | $185 per person |

### Employer Sanctions

| Element | Impact |
|---------|--------|
| **I-9 requirement** | Employment verification |
| **Penalties** | For knowingly hiring |
| **Effectiveness** | Limited enforcement |
| **Document fraud** | Industry emerged |

### IRCA Assessment

| Aspect | Outcome |
|--------|---------|
| **Legalization** | Successful, 2.7M legalized |
| **Enforcement** | Ineffective |
| **Future flow** | Not addressed |
| **Population** | Returned to growth by 1990s |

## 1990s: Growth and Enforcement

### Population Increase

| Year | Unauthorized |
|------|--------------|
| **1990** | 3.5 million |
| **1995** | 5.7 million |
| **2000** | 8.6 million |

### IIRIRA (1996)

| Provision | Impact |
|-----------|--------|
| **3/10 year bars** | Unlawful presence penalties |
| **287(g)** | Local enforcement authority |
| **Expedited removal** | Faster deportation |
| **Criminal grounds** | Expanded deportability |

### Border Build-Up

| Operation | Description |
|-----------|-------------|
| **Hold the Line (1993)** | El Paso saturation |
| **Gatekeeper (1994)** | San Diego sector |
| **Safeguard (1994)** | Arizona sectors |
| **Strategy** | Push crossings to remote areas |

### Consequences

| Outcome | Impact |
|---------|--------|
| **Crossing deaths** | Increased dramatically |
| **Circular flow ended** | Settlers instead of sojourners |
| **Smuggling** | More organized, expensive |
| **Population growth** | Despite enforcement |

## 2008-2016: Enforcement and Relief

### Great Recession Impact

| Change | Description |
|--------|-------------|
| **Mexican migration** | Net zero/negative |
| **Economic opportunity** | Reduced pull |
| **Enforcement** | Continued increase |
| **Population** | Declined from peak |

### Obama Administration

| Phase | Policy |
|-------|--------|
| **2009-2011** | High deportations |
| **2012** | DACA announcement |
| **2014** | DAPA announced |
| **Priorities** | Focus on criminals |

### DACA (2012)

| Element | Description |
|---------|-------------|
| **Eligibility** | Arrived as children, continuous presence |
| **Benefits** | Work authorization, deferred action |
| **Scope** | ~800,000 approved |
| **Legal basis** | Executive discretion |

### DAPA (2014)

| Element | Description |
|---------|-------------|
| **Target** | Parents of citizens/LPRs |
| **Potential** | 4-5 million |
| **Outcome** | Enjoined by courts |
| **Never implemented** | Texas v. United States |
