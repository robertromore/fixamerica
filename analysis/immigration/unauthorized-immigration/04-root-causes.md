# Unauthorized Immigration: Root Causes

## Structural Causes of Unauthorized Immigration

### Supply-Side Factors (Push)

| Factor | Description |
|--------|-------------|
| **Economic disparity** | Wage differentials of 5-10x |
| **Violence** | Gang violence, political instability |
| **Climate** | Drought, crop failures |
| **Governance** | Corruption, weak institutions |
| **Demographics** | Youth bulge, unemployment |

### Demand-Side Factors (Pull)

| Factor | Description |
|--------|-------------|
| **Labor demand** | US employer needs |
| **Wage premium** | Substantial earnings increase |
| **Opportunity** | Education, safety, stability |
| **Networks** | Family, community ties |
| **Relative welfare** | Better conditions |

### Insufficient Legal Pathways

| Gap | Description |
|-----|-------------|
| **Numerical limits** | Far below demand |
| **Wait times** | Decades for many |
| **No unskilled category** | Legal option eliminated 1965 |
| **Family caps** | Per-country limits |
| **Employment barriers** | Limited visa categories |

## Why Legal Options Fail

### Employment-Based Limitations

| Problem | Impact |
|---------|--------|
| **140,000 cap** | Insufficient for economy |
| **No "unskilled" visa** | No legal path for many jobs |
| **Employer sponsorship** | Required but unavailable |
| **Temporary options** | H-2A/H-2B inadequate |
| **Processing** | Slow, expensive |

### Family-Based Limitations

| Problem | Impact |
|---------|--------|
| **Per-country caps** | Multi-decade waits |
| **Category limits** | Siblings, adult children restricted |
| **Sponsorship requirements** | Income, status barriers |
| **Backlog** | 4+ million waiting |

### Humanitarian Limitations

| Problem | Impact |
|---------|--------|
| **Asylum capacity** | System overwhelmed |
| **Refugee ceiling** | Politically volatile |
| **TPS limitations** | Temporary, uncertain |
| **Protection gaps** | Violence not always qualifying |

## Why Enforcement Alone Fails

### Scale Problem

| Reality | Implication |
|---------|-------------|
| **11 million people** | Cannot deport all |
| **Deep roots** | 16+ years average |
| **Mixed families** | US citizen children |
| **Cost** | $100B+ for mass removal |
| **Labor force** | 5% of workforce |

### Economic Integration

| Factor | Description |
|---------|-------------|
| **Essential industries** | Agriculture, construction |
| **Employer reliance** | Longstanding workforce |
| **Consumer role** | $250B+ spending |
| **Tax payments** | $48B+ annually |
| **GDP contribution** | $1.6 trillion |

### Community Integration

| Factor | Description |
|--------|-------------|
| **Homeownership** | ~3 million |
| **US citizen children** | 4.4 million |
| **Mixed families** | Millions affected |
| **Local roots** | Schools, churches, neighborhoods |
| **Social capital** | Community ties |

### Enforcement Limitations

| Limitation | Reality |
|------------|---------|
| **Resource constraints** | Cannot find/process all |
| **Prioritization** | Must focus limited resources |
| **Constitutional limits** | Due process requirements |
| **Country cooperation** | Cannot always remove |
| **Community relations** | Trust undermines safety |

## Root Causes of Current Population

### 1986 IRCA Failure

| Problem | Outcome |
|---------|---------|
| **Legalization** | 2.7 million succeeded |
| **Enforcement** | Employer sanctions weak |
| **Future flow** | Not addressed |
| **Result** | Population regrew |

### 1990s Border Strategy

| Problem | Outcome |
|---------|---------|
| **Deterrence focus** | Push to remote areas |
| **Circular flow** | Disrupted, settlers instead |
| **Smuggling** | More organized |
| **Deaths** | Increased dramatically |
| **Permanence** | Longer stays |

### 1996 IIRIRA Penalties

| Provision | Impact |
|-----------|--------|
| **3/10 year bars** | Trap people inside US |
| **Unlawful presence** | Triggers bars on departure |
| **Catch-22** | Cannot leave to legalize |
| **Permanence** | Encourages staying |

### Failed Reform Attempts

| Year | What Failed |
|------|-------------|
| **2006** | Comprehensive reform |
| **2007** | Bipartisan effort |
| **2010** | DREAM Act |
| **2013** | Gang of Eight |
| **Result** | Status quo maintained |

## Why Reform Repeatedly Fails

### Political Dynamics

| Factor | Impact |
|--------|--------|
| **Partisan polarization** | Compromise impossible |
| **Primary politics** | Punishes moderation |
| **"Amnesty" framing** | Poisons debate |
| **Border focus** | Diverts from interior population |
| **Electoral calculations** | Neither party prioritizes |

### Coalition Challenges

| Group | Position |
|-------|----------|
| **Business** | Wants legal workers |
| **Labor** | Wants enforcement, worker rights |
| **Restrictionists** | Opposes legalization |
| **Advocates** | Wants protection, path |
| **Result** | No stable majority |

### Structural Barriers

| Barrier | Impact |
|---------|--------|
| **Senate filibuster** | Requires 60 votes |
| **House rules** | Hastert rule |
| **Conference collapse** | Bills die in conference |
| **Poison pills** | Amendments kill bills |
