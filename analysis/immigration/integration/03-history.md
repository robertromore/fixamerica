# Immigrant Integration: History

## Overview

American approaches to immigrant integration have varied dramatically across history, from active "Americanization" campaigns to restrictive exclusion to more recent welcoming community initiatives. This evolution reflects changing immigration patterns and national attitudes.

## Early Republic (1790-1870)

### Naturalization Origins

| Development | Impact |
|-------------|--------|
| **Naturalization Act of 1790** | First citizenship law (white persons only) |
| **5-year residency** | Standard requirement established |
| **State variation** | Different implementation |
| **Party competition** | Political integration important |

### Settlement Patterns

| Era | Integration Approach |
|-----|---------------------|
| **Colonial period** | Community-based settlement |
| **Early 1800s** | Ethnic neighborhoods form |
| **Religious institutions** | Churches aid settlement |
| **Political machines** | Urban immigrant incorporation |

## Mass Immigration Era (1870-1924)

### Scale and Composition

| Period | Annual Arrivals |
|--------|-----------------|
| **1880-1900** | 400,000-700,000 |
| **1900-1914** | 800,000-1.2 million |
| **Origins** | Southern/Eastern Europe |
| **Settlement** | Urban concentration |

### Americanization Movement

| Element | Approach |
|---------|----------|
| **Public schools** | English instruction, civics |
| **Settlement houses** | Hull House model |
| **Night schools** | Adult education |
| **Employers** | Ford English School |
| **Civic groups** | DAR, civic leagues |

### Methods and Goals

| Goal | Method |
|------|--------|
| **English acquisition** | Mandatory instruction |
| **Civic knowledge** | Citizenship classes |
| **Cultural conformity** | "Melting pot" ideal |
| **Loyalty** | Patriotic education |

### Tensions

| Tension | Manifestation |
|---------|---------------|
| **Nativism** | Know-Nothing movement |
| **Religious prejudice** | Anti-Catholic sentiment |
| **Labor competition** | Union exclusion |
| **Racial exclusion** | Asian restrictions |

## Restriction Era (1924-1965)

### Immigration Restriction

| Legislation | Effect |
|-------------|--------|
| **1924 Immigration Act** | National origins quotas |
| **Reduced numbers** | 150,000 annual limit |
| **Composition shift** | Northern/Western European favored |
| **Asian exclusion** | Near total bar |

### Integration Policy

| Aspect | Status |
|--------|--------|
| **Formal programs** | Declined with immigration |
| **Settlement houses** | Shifted focus |
| **Americanization** | Continued in schools |
| **Second generation** | Focus of integration |

### World War II Era

| Development | Impact |
|-------------|--------|
| **Japanese internment** | Integration betrayed |
| **Military service** | Path to belonging |
| **War brides** | New integration challenge |
| **Bracero program** | Temporary workers, limited integration |

## Civil Rights Era (1965-1990)

### Immigration and Nationality Act of 1965

| Change | Effect |
|--------|--------|
| **End national origins** | Diversity of origins |
| **Family reunification** | Chain migration |
| **Rising numbers** | Immigration increases |
| **New origins** | Asia, Latin America |

### Bilingual Education

| Development | Date |
|-------------|------|
| **Bilingual Education Act** | 1968 |
| **Lau v. Nichols** | 1974 (language access) |
| **Expansion** | 1970s-1980s |
| **Controversy** | English-only backlash |

### Refugee Resettlement

| Program | Implementation |
|---------|----------------|
| **Refugee Act of 1980** | Formal resettlement |
| **ORR established** | Integration services |
| **Public-private partnership** | Voluntary agencies |
| **Southeast Asian refugees** | Major integration effort |

## Contemporary Era (1990-Present)

### IRCA and Legalization

| Element | Impact |
|---------|--------|
| **1986 legalization** | 2.7 million legalized |
| **Integration services** | Limited but significant |
| **Path to citizenship** | Many naturalized |
| **Model** | Integration through legalization |

### 1990s Developments

| Development | Effect |
|-------------|--------|
| **1996 welfare reform** | Benefit restrictions |
| **English-only movement** | State initiatives |
| **Immigrant rights** | Organizing response |
| **Growing population** | Integration needs expand |

### Post-9/11

| Change | Impact |
|--------|--------|
| **Security focus** | Enforcement over integration |
| **Citizenship delays** | Processing backlogs |
| **Anti-immigrant sentiment** | Social climate |
| **State/local action** | Filling federal gap |

### State and Local Rise

| Era | Development |
|-----|-------------|
| **2000s** | States create immigrant offices |
| **2010** | Welcoming America founded |
| **2010s** | Municipal ID programs spread |
| **Present** | 150+ welcoming communities |

## Naturalization History

### Evolution of Requirements

| Era | Requirements |
|-----|--------------|
| **1790** | 2 years residency, good character |
| **1795** | 5 years residency, oath |
| **1802** | 5 years, declarations |
| **1906** | English requirement added |
| **1950** | Loyalty requirements |
| **Present** | English, civics, 5 years |

### Citizenship Test Evolution

| Era | Test Format |
|-----|-------------|
| **Pre-1986** | Interview-based |
| **1986** | Standardized questions |
| **2008** | Redesigned civics test |
| **2020** | Expanded then reverted |

### Fee History

| Year | Fee (Adjusted) |
|------|----------------|
| **1990** | $95 |
| **2000** | $225 |
| **2010** | $595 |
| **2020** | $725 |
| **2024** | $760 |

## Lessons from History

### Successful Integration Elements

| Element | Historical Evidence |
|---------|---------------------|
| **Legal status pathway** | IRCA legalization outcomes |
| **English instruction** | Settlement house success |
| **Economic opportunity** | Labor market integration |
| **Political incorporation** | Urban machine model |
| **Community support** | Church and civic groups |

### Failed Approaches

| Approach | Outcome |
|----------|---------|
| **Forced assimilation** | Resistance, resentment |
| **Exclusion** | Marginalization |
| **Benefit restrictions** | Integration barriers |
| **Status limbo** | Permanent underclass |
