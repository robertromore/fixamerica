# Immigrant Integration: Current State

## Overview

The United States has 46+ million foreign-born residents representing 14% of the population. Integration outcomes vary significantly by legal status, country of origin, education level, and geographic location.

## Naturalization

### Current System

| Element | Status |
|---------|--------|
| **Eligibility** | 5 years LPR (3 if married to USC) |
| **Application fee** | $760 (N-400 + biometrics) |
| **Processing time** | 12-18 months average |
| **Pass rate** | 91% of test takers |

### Naturalization Rates

| Group | Naturalization Rate |
|-------|---------------------|
| **Overall eligible** | 70% |
| **Asian immigrants** | 75% |
| **European immigrants** | 73% |
| **Mexican immigrants** | 42% |
| **Central American** | 48% |

### Barriers to Naturalization

| Barrier | Impact |
|---------|--------|
