# Immigrant Integration: Root Causes of Challenges

## Overview

Integration challenges stem from structural policy gaps, resource limitations, legal status barriers, and social factors. Understanding these root causes is essential for designing effective interventions.

## Absence of Federal Integration Policy

### No Dedicated Infrastructure

| Gap | Consequence |
|-----|-------------|
| **No integration office** | Fragmented responsibility |
| **No integration strategy** | Ad hoc approaches |
| **No coordinating body** | Duplication, gaps |
| **No comprehensive data** | Unmeasured outcomes |

### Enforcement Dominance

| Factor | Impact |
|--------|--------|
| **Budget allocation** | $25B enforcement vs. $10M citizenship grants |
| **Agency priorities** | DHS focus on removal |
| **Political incentives** | Enforcement more visible |
| **Policy volatility** | Integration deprioritized |

### Comparison to Other Countries

| Country | Integration Infrastructure |
|---------|---------------------------|
| **Canada** | Federal integration ministry |
| **Germany** | Integration courses mandatory |
| **Netherlands** | Civic integration program |
| **United States** | No equivalent |

## Legal Status Barriers

### Status-Based Exclusions

| Status | Exclusions |
|--------|------------|
| **Unauthorized** | No work authorization, benefits, naturalization path |
| **Recent LPRs** | 5-year Medicaid bar |
| **Non-immigrants** | Limited work, no permanence |
| **TPS/DACA** | Temporary, uncertain |

### Citizenship Barriers

| Barrier | Effect |
|---------|--------|
| **Cost** | $760 excludes low-income |
| **English test** | 9 million LEP eligible |
| **Civics test** | Study burden |
| **Processing time** | 12-18 month wait |

### Status Uncertainty

| Issue | Impact |
|-------|--------|
| **Policy volatility** | TPS, DACA uncertainty |
| **Administrative backlogs** | Application delays |
| **Enforcement fear** | Service avoidance |
| **Family separation** | Mixed-status families |
