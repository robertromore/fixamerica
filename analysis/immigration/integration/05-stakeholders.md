# Immigrant Integration: Stakeholders

## Overview

Immigrant integration involves diverse stakeholders with varied interests, capacities, and influence. Understanding these stakeholders is essential for building effective coalitions and designing inclusive policies.

## Primary Stakeholders

### Immigrants and Families

| Group | Interests |
|-------|-----------|
| **LPRs seeking citizenship** | Affordable, accessible naturalization |
| **Unauthorized individuals** | Path to legal status |
| **Recent arrivals** | Settlement support, orientation |
| **Second generation** | Educational opportunity, belonging |
| **Mixed-status families** | Family unity, stability |

### Concerns and Needs

| Need | Priority |
|------|----------|
| **Legal status security** | Foundation for integration |
| **Economic opportunity** | Employment, advancement |
| **Education access** | Children and adult learning |
| **Healthcare** | Coverage and access |
| **Community belonging** | Social inclusion |
