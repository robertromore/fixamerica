# Immigration Courts: History

## Origins of Immigration Adjudication

### Early Period (Pre-1940)

| Era | Characteristic |
|-----|----------------|
| **Pre-1891** | State-administered |
| **1891** | Federal immigration officers |
| **1903** | Bureau of Immigration, Commerce |
| **1906** | Naturalization courts |
| **1917** | Expanded exclusion hearings |

### Immigration and Naturalization Service Era

| Development | Impact |
|-------------|--------|
| **1933 INS created** | Consolidated functions |
| **Labor to Justice (1940)** | DOJ control begins |
| **Combined agency** | Enforcement + adjudication |
| **Special inquiry officers** | Hearing examiners |

## Establishment of Immigration Courts (1983)

### Creation of EOIR

| Element | Development |
|---------|-------------|
| **Year** | 1983 |
| **Authority** | DOJ reorganization |
| **Purpose** | Separate from INS |
| **Components** | IJs, BIA, OCIJ |
| **Location** | Within DOJ |

### Rationale for Separation

| Concern | Response |
|---------|----------|
| **Combined functions** | Enforcement influenced adjudication |
| **Impartiality** | Separate decision-makers |
| **Independence** | From INS enforcement |
| **But still DOJ** | Attorney General authority |

### Initial Structure

| Element | 1983 Design |
|---------|-------------|
| **Judges** | ~60 |
| **BIA members** | 5 |
| **Courts** | Fewer than 30 |
| **Caseload** | Manageable |

## 1990s: Growing Caseloads

### IIRIRA (1996) Impact

| Change | Effect |
|--------|--------|
| **Expedited removal** | New removal track |
| **Criminal bars** | Expanded removability |
| **Detention mandates** | More detained cases |
| **Streamlined appeals** | BIA reform pressure |

### Caseload Growth

| Year | Pending Cases |
|------|---------------|
| **1990** | ~150,000 |
| **1995** | ~250,000 |
| **2000** | ~180,000 |

### BIA Expansion and Contraction

| Period | Members |
|--------|---------|
| **1994** | 12 |
| **1995** | 15 |
| **1999** | 23 |
| **Post-2002** | Reduced |

## 2002 Reorganization

### Ashcroft Reforms

| Change | Impact |
|--------|--------|
| **Streamlining** | Single-member review |
| **Affirmance without opinion** | AWO introduced |
| **Board reduction** | From 23 to 11 |
| **Member removal** | Perceived political |
| **Efficiency focus** | Speed over quality |

### Creation of DHS

| Change | Impact |
|--------|--------|
| **INS abolished** | 2003 |
| **USCIS created** | Benefits |
| **ICE created** | Enforcement |
| **CBP created** | Border |
| **EOIR remained** | In DOJ |

### Consequences

| Outcome | Description |
|---------|-------------|
| **Federal court appeals** | Increased dramatically |
| **Quality concerns** | Judicial criticism |
| **Backlog shift** | To federal courts |
| **BIA reputation** | Damaged |

## 2008-2016: Obama Era

### Early Period

| Development | Impact |
|-------------|--------|
| **Judge hiring** | Increased |
| **Backlog** | Initially managed |
| **Priorities** | Criminal focus |
| **Docket control** | Some reforms |

### Surge in Cases

| Year | Development |
|------|-------------|
| **2014** | Unaccompanied children surge |
| **2015-16** | Central American migration |
| **Backlog growth** | From 300K to 500K |
| **Resources** | Not matched |

### Priorities and Discretion

| Policy | Impact |
|--------|--------|
| **Administrative closure** | Docket management tool |
| **Prosecutorial discretion** | Case selection |
| **Priority dockets** | Family, UAC cases |

## 2017-2020: Trump Era

### Sessions Reforms

| Change | Impact |
|--------|--------|
| **Quotas** | 700 case completion |
| **Administrative closure** | Eliminated |
| **AG certification** | Matter of A-B-, others |
| **Hiring priorities** | Enforcement background |

### Zero Tolerance Impact

| Effect | Description |
|--------|-------------|
| **Family separation** | Court challenges |
| **Case surge** | MPP returns |
| **Backlog growth** | 500K to 1.3M |
| **Due process** | Concerns increased |

### Judicial Independence Concerns

| Issue | Impact |
|-------|--------|
| **Quotas** | Pressure on judges |
| **Union decertification** | NAIJ battle |
| **AG review** | Policy override |
| **Hiring** | Ideological concerns |

## Backlog History

### Growth Pattern

| Period | Cause |
|--------|-------|
| **2014-2016** | Central American migration |
| **2017-2020** | Policy changes, MPP |
| **2021-present** | Border surge, prior delays |

### Failed Solutions

| Approach | Why Failed |
|----------|------------|
| **Judge hiring** | Never enough |
| **Streamlining** | Quality concerns |
| **Docket management** | Temporary effect |
| **Priorities** | Shifts backlog |
