# Immigration Courts: Root Causes

## Structural Root Causes

### Executive Branch Placement

| Problem | Impact |
|---------|--------|
| **DOJ control** | Political influence |
| **AG authority** | Policy override |
| **No judicial independence** | Compromised impartiality |
| **Attorney classification** | Not judges |
| **Removal power** | Chilling effect |

### Design Flaws

| Flaw | Consequence |
|------|-------------|
| **Not Article I or III** | No constitutional protection |
| **Single agency** | Prosecutors = boss's colleagues |
| **No tenure** | Job insecurity |
| **Performance metrics** | Quantity over quality |
| **Political appointments** | Ideological shifts |

### System Fragmentation

| Problem | Impact |
|---------|--------|
| **EOIR vs. USCIS asylum** | Parallel tracks |
| **Immigration vs. federal courts** | Multiple forums |
| **BIA vs. circuit** | Split review |
| **No unified system** | Inconsistency |

## Resource Root Causes

### Chronic Underfunding

| Resource | Gap |
|----------|-----|
| **Judges** | Need 2-3x current |
| **Support staff** | Severely understaffed |
| **Technology** | Decades behind |
| **Infrastructure** | Space constraints |
| **Legal services** | Minimal funding |

### Arithmetic Problem

| Metric | Reality |
|--------|---------|
| **New cases** | 400,000-600,000/year |
| **Completions** | ~300,000/year |
| **Annual deficit** | 100,000-300,000 |
| **Compound growth** | Exponential backlog |

### Budget Process

| Problem | Impact |
|---------|--------|
| **Discretionary funding** | Annual uncertainty |
| **Political battles** | Immigration funding controversial |
| **Enforcement vs. courts** | Imbalanced priorities |
| **No dedicated stream** | Unpredictable resources |
