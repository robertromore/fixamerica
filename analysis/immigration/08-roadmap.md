# Immigration: Implementation Roadmap

## Overview

Implementing immigration reform requires navigating intense political opposition, building coalitions, and sequencing changes to build trust and demonstrate results. This roadmap outlines phases from immediate administrative actions through comprehensive legislative reform, recognizing that different political scenarios enable different approaches.

## Scenario Planning

### Scenario A: Unified Government (Reform-Friendly)

| Phase | Focus | Timeline |
|-------|-------|----------|
| **Immediate** | Administrative relief, executive action | Months 1-6 |
| **Near-term** | Targeted legislation (DACA, farmworkers) | Year 1 |
| **Medium-term** | Comprehensive bill | Years 1-2 |
| **Implementation** | Phase-in programs | Years 2-5 |

### Scenario B: Divided Government

| Phase | Focus | Timeline |
|-------|-------|----------|
| **Immediate** | Administrative actions within authority | Months 1-6 |
| **Near-term** | Bipartisan negotiations | Years 1-2 |
| **Contingent** | Border deal with limited relief | If breakthrough |
| **Incremental** | Piecemeal wins where possible | Ongoing |

### Scenario C: Restrictionist Government

| Phase | Focus | Timeline |
|-------|-------|----------|
| **Defensive** | Litigation, state resistance | Immediate |
| **State action** | Sanctuary policies, state benefits | Ongoing |
| **Coalition building** | Rebuild support | Years 1-4 |
| **Future reform** | Position for next opportunity | Long-term |

## Phase 2: Targeted Legislation (Year 1-2)

### High-Priority Bills

| Bill | Coalition | Prospects |
|------|-----------|-----------|
| **DREAM Act** | Broad bipartisan | High support |
| **Farm Workforce Modernization** | Ag + labor | Passed House previously |
| **Afghan Adjustment** | Bipartisan | Strong support |
| **Employment backlog** | Business + tech | Growing support |
| **Immigration judge independence** | Legal community | Nonpartisan appeal |

### DREAM Act Components

| Element | Description |
|---------|-------------|
| **Eligibility** | Arrived before 18, 4+ years presence |
| **Conditional status** | 10-year period |
| **Requirements** | Education, employment, military, or civics |
| **Path to LPR** | After requirements |
| **Path to citizenship** | Standard naturalization |

### Farm Workforce

| Element | Description |
|---------|-------------|
| **Current worker legalization** | Years of ag work |
| **H-2A reforms** | Streamlined, portable |
| **E-Verify** | Ag sector phase-in |
| **Worker protections** | Wage, housing standards |

## Phase 3: Comprehensive Reform (Years 2-4)

### Legislative Framework

| Title | Content |
|-------|---------|
| **Title I: Border Security** | Technology, personnel, processing |
| **Title II: Legal Immigration** | Backlog, categories, numbers |
| **Title III: Employment Verification** | E-Verify mandate, phase-in |
| **Title IV: Legalization** | Earned pathway, requirements |
| **Title V: Asylum Reform** | Processing, protections |
| **Title VI: Immigration Courts** | Independence, resources |
| **Title VII: Integration** | Citizenship, services |

### Coalition Building

| Stakeholder | Offer |
|-------------|-------|

| **Labor** | Worker protections, path to citizenship |
| **Restrictionists** | Enforcement, future flow controls |
| **Advocates** | Legalization, family reunification |
| **Faith groups** | Family unity, humanitarian provisions |

| Chamber | Approach |
