# Immigration Enforcement: Root Causes

## Overview

The enforcement challenges stem from fundamental capacity constraints, the impossibility of enforcing against 11 million people, legal and practical barriers to removal, political dynamics that prevent resolution, and the tension between enforcement and community trust. These root causes interact to create a system that satisfies no one.

## Capacity Constraints

### Scale Mismatch

| Factor | Reality |
|--------|---------|
| **Unauthorized population** | 11 million |
| **ERO officers** | 6,000 |
| **Annual arrests** | 100,000-170,000 |
| **Annual removals** | 100,000-400,000 |
| **Mathematical reality** | Cannot remove all |

### Resource Limitations

| Resource | Constraint |
|----------|------------|
| **Personnel** | Insufficient for scale |
| **Detention** | Limited capacity |
| **Courts** | Massive backlog |
| **Transportation** | Aircraft, buses |
| **Processing** | Documentation, logistics |

### Priority Necessity

| Reality | Implication |
|---------|-------------|
| **Cannot enforce against all** | Must prioritize |
| **Political choice** | Who to target |
| **Resource allocation** | Where to focus |
| **Discretion inherent** | Officers choose daily |

## Legal and Practical Barriers

### Due Process Requirements

| Requirement | Impact |
|-------------|--------|
| **Hearing right** | For most removable |
| **Appeal rights** | Delays removal |
| **Relief applications** | Asylum, cancellation |
| **Counsel access** | Preparation time |

### Country Cooperation

| Challenge | Impact |
|-----------|--------|
| **Recalcitrant countries** | Won't accept returns |
| **Document requirements** | Slow issuance |
| **Travel document** | Must have for removal |
| **Diplomatic limits** | Can't force |

### Mixed Families

| Reality | Complication |
|---------|--------------|
| **US citizen children** | 4.5+ million |
| **Mixed families** | 16+ million in households |
| **Separation** | Humanitarian concern |
| **Political** | Sympathetic cases |

### Labor Market Dynamics

| Factor | Impact |
|--------|--------|
| **Jobs available** | Attracts workers |
| **Wage differentials** | Strong incentive |
| **Employer need** | Industries depend on |
| **Hiring continues** | Despite enforcement |

### Employer Sanctions Failure

| Factor | Reason |
|--------|--------|
| **Minimal enforcement** | Few audits, penalties |

| **Document fraud** | Easy to obtain |
| **Economic interest** | Business wants workers |

### E-Verify Limitations

| Limitation | Impact |
|------------|--------|
| **Not mandatory** | Most don't use |
| **Identity fraud** | Works with stolen IDs |
| **Error rates** | Some legal workers flagged |
| **Enforcement** | Limited consequences |

### System Design Flaws

| Flaw | Impact |
|------|--------|

| **Courts backlogged** | No swift consequence |

## Feedback Loops

### Enforcement-Avoidance Cycle

```
Enforcement increase → Avoidance behavior
        ↓                    ↓
Community distrust ← Fear spreading ← Arrests in community
```

### Deterrence Failure

```
Deterrence attempt → Smuggling sophistication
       ↓                    ↓
Higher costs ← Continued migration ← Push factors unchanged
```

### Political Cycle

```
Policy change → Adjustment by all parties
      ↓                    ↓
New administration ← Political pressure ← Dissatisfaction
```

## Root Cause Interactions

| Factor A | + Factor B | = Outcome |
|----------|------------|-----------|
| Limited capacity | Large population | Selective enforcement |
| Selective enforcement | Political pressure | Priority shifts |
| Priority shifts | No legislation | Instability |
| Instability | Community fear | Trust erosion |
| Trust erosion | Sanctuary policies | Federal-local conflict |
