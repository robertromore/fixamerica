# Immigration Detention: Opposition to Reform

## Overview

Opposition to detention reform comes from enforcement advocates, private prison interests, and political actors who view detention as essential to immigration enforcement and public safety.

## Enforcement Arguments

### Necessity Claims

| Argument | Description |
|----------|-------------|
| **Flight risk** | Won't appear for hearings |
| **Public safety** | Criminal aliens |
| **Deterrence** | Discourages migration |
| **Removal execution** | Ensures deportation |

### Counter-Evidence

| Claim | Evidence |
|-------|----------|
| **Flight risk** | 85-95% appearance rates with ATD |
| **Criminal** | 60-70% have no criminal record |
| **Deterrence** | Limited evidence |
| **Cost-effective** | ATD much cheaper |

## Industry Opposition

### Private Prison Interests

| Interest | Mechanism |
|----------|-----------|
| **Revenue** | $3B+ market |
| **Contracts** | Long-term agreements |
| **Lobbying** | Campaign contributions |
| **Jobs** | Local employment |

### Arguments

| Argument | Description |
|----------|-------------|
| **Cost savings** | Cheaper than federal |
| **Capacity** | Can build quickly |
| **Expertise** | Management capability |
| **Jobs** | Local economy |

## Political Opposition

### Conservative Position

| Argument | Description |
|----------|-------------|
| **Rule of law** | Consequences for illegal entry |
| **Public safety** | Protect communities |
| **Border security** | Part of enforcement |
| **Accountability** | Cannot disappear |

### Specific Objections

| Reform | Objection |
|--------|-----------|
| **End detention** | Will not appear |
| **Reduce capacity** | Can't enforce |
| **End private** | Too expensive |
| **More ATD** | Not effective |

## Good Faith vs. Bad Faith

### Good Faith

| Concern | Characteristic |
|---------|----------------|
| **Public safety** | Genuine concern |
| **Efficiency** | Resource allocation |
| **Compliance** | Ensure appearance |

### Bad Faith

| Position | Characteristic |
|----------|----------------|
| **Profit motive** | Industry protection |
| **Deterrence myth** | Despite evidence |
| **All detention** | Ignore alternatives |
