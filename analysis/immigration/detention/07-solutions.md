# Immigration Detention: Solutions

## Overview

Detention reform encompasses reducing reliance on detention, improving conditions for those detained, expanding alternatives, protecting vulnerable populations, and ensuring accountability.

## Reduce Detention

### Presumption of Release

| Reform | Implementation |
|--------|----------------|
| **Default release** | Unless specific risk |
| **Parole presumption** | For asylum seekers |
| **Bond reform** | Affordable amounts |
| **Time limits** | Maximum detention periods |

### End Mandatory Detention

| Reform | Target |
|--------|--------|
| **Narrow categories** | Only violent offenses |
| **Judicial review** | For all cases |
| **Discretion** | Allow case-by-case |
| **Legislative change** | INA 236(c) reform |

### Population Caps

| Approach | Mechanism |
|----------|-----------|
| **Appropriations** | Fund fewer beds |
| **Phase-down** | Gradual reduction |
| **ATD ratio** | Required alternatives |
