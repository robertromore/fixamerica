# Immigration Detention: Root Causes

## Structural Root Causes

### Mandatory Detention Statute

| Provision | Impact |
|-----------|--------|
| **INA 236(c)** | Criminal grounds |
| **Automatic** | No discretion |
| **Broad categories** | Many covered |
| **No bond** | For covered cases |

### Civil vs. Criminal Framing

| Issue | Impact |
|-------|--------|
| **Civil designation** | Fewer rights |
| **Not "punishment"** | Legal fiction |
| **Indefinite** | No sentences |
| **Lower standards** | Reduced protections |

### Privatization Model

| Problem | Impact |
|---------|--------|
| **Profit motive** | Fill beds |
| **Bed guarantees** | Pay regardless |
| **Lobbying** | Maintain system |
| **Accountability gaps** | Private vs. government |

## Political Root Causes

### Enforcement Ideology

| Belief | Policy |
|--------|--------|
| **Deterrence** | Detention deters |
| **Flight risk** | Must detain |
| **Dangerous** | Public safety |
| **Rule of law** | Consequences |

### Congressional Mandates

| Mandate | Impact |
|---------|--------|
| **Bed mandate** | 34,000 beds |
| **Appropriations** | Fund detention |
| **Oversight limits** | Private contractor protection |

### Political Incentives

| Incentive | Effect |
|-----------|--------|
| **"Tough" image** | More detention |
| **Industry lobbying** | Maintain contracts |
| **Local politics** | Jobs, revenue |
| **Fear messaging** | Support detention |

## Economic Root Causes

### Private Prison Industry

| Factor | Impact |
|--------|--------|
| **$3B+ market** | Large industry |
| **Campaign contributions** | Political influence |
| **Lobbying** | Policy influence |
| **Job provision** | Rural communities |

### Cost-Shifting

| Issue | Impact |
|-------|--------|
| **Private = "cheaper"** | Perception |
| **Reduced services** | Cut costs |
| **Quality** | Suffers |
| **True costs** | Hidden |

## Legal Root Causes

### Limited Rights

| Gap | Impact |
|-----|--------|
| **No right to counsel** | Unrepresented |
| **Bond discretion** | High amounts |
| **Habeas limits** | Delayed review |
| **Class action barriers** | Hard to challenge |

### Flores Settlement

| Element | Impact |
|---------|--------|
| **20-day limit** | Children |
| **Litigation** | Ongoing |
| **Workarounds** | Family separation |
| **Compliance** | Inconsistent |

## Systemic Analysis

### Feedback Loops

| Loop | Effect |
|------|--------|
| **More beds** | More detention |
| **Private profits** | More lobbying |
| **More lobbying** | More appropriations |
| **More money** | More beds |

### Lack of Alternatives

| Gap | Impact |
|-----|--------|
| **Underfunded ATD** | Default to detention |
| **No presumption** | Against release |
| **Bond amounts** | Unaffordable |
| **Risk aversion** | Detain to be safe |
