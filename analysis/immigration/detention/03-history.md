# Immigration Detention: History

## Early Immigration Detention

### Pre-1980s

| Era | Characteristic |
|-----|----------------|
| **Ellis Island** | Processing, not long-term |
| **Limited detention** | Primarily exclusion |
| **INS custody** | Small numbers |
| **Criminal focus** | Deportation cases |

### 1980s: Mariel and Expansion

| Development | Impact |
|-------------|--------|
| **Mariel boatlift (1980)** | Mass detention |
| **INS expansion** | Capacity increase |
| **Reagan era** | Deterrence focus |
| **Haitian asylum seekers** | Detention policy |

## 1990s: System Growth

### IIRIRA (1996)

| Provision | Impact |
|-----------|--------|
| **Mandatory detention** | Criminal grounds |
| **Expedited removal** | Detention for processing |
| **287(g)** | Local involvement |
| **Expanded categories** | More detained |

### Privatization Begins

| Development | Impact |
|-------------|--------|
| **CCA (now CoreCivic)** | First private contracts |
| **Wackenhut (now GEO)** | Expansion |
| **Cost arguments** | Privatization justification |
| **Bed guarantees** | Contract provisions |

## 2000s: Post-9/11 Expansion

### Homeland Security Era

| Change | Impact |
|--------|--------|
| **DHS creation (2003)** | New structure |
| **ICE formation** | Enforcement focus |
| **Secure Communities** | Local partnerships |
| **Bed mandate** | Congressional requirement |

### Rapid Growth

| Year | Daily Population |
|------|-----------------|
| **2001** | ~20,000 |
| **2005** | ~20,000 |
| **2008** | ~30,000 |
| **2010** | ~33,000 |

### Bed Mandate

| Element | Impact |
|---------|--------|
| **34,000 beds** | Congressional requirement |
| **Guaranteed minimums** | Contract provisions |
| **Capacity pressure** | Fill beds |
| **Years active** | 2009-2017 |

## 2010s: Scrutiny and Expansion

### Obama Administration

| Period | Policy |
|--------|--------|
| **Early years** | High detention |
| **2014 surge** | Family detention expansion |
| **Later years** | Alternatives growth |
| **Priorities** | Some reduction |

### Family Detention Revival

| Development | Impact |
|-------------|--------|
| **2014 border surge** | Family detention expansion |
| **Dilley, Karnes** | New family facilities |
| **Flores litigation** | 20-day limit |
| **Controversy** | Child detention |

### Trump Administration

| Policy | Impact |
|--------|--------|
| **Zero tolerance** | Family separation |
| **Increased detention** | 50,000+ daily |
| **Reduced ATD** | More detention |
| **New facilities** | Capacity expansion |

## Recent History

### COVID-19 Impact

| Effect | Description |
|--------|-------------|
| **Population drop** | To ~15,000 |
| **Outbreaks** | Major facility spread |
| **Deaths** | COVID-related |
| **Litigation** | Release orders |

### Biden Administration

| Policy | Impact |
|--------|--------|
| **Initial reduction** | Population declined |
| **No new private** | Executive order |
| **Alternatives** | Expansion |
| **2022-2023** | Population increased |

### Current Trends

| Trend | Direction |
|-------|-----------|
| **Population** | Increasing |
| **Private contracts** | Continuing |
| **Alternatives** | Expanding |
| **Oversight** | Increased |
