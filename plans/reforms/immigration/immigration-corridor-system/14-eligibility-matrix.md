# Immigration Corridor System: Eligibility Matrix

---

## Overview

This document clarifies what corridor participants are eligible for at each status tier, addressing common questions about healthcare, identification, banking, benefits, civic participation, and rights.

---

| Service | Phase 0 (Intake) | Tier A/B (CLP) | Tier C | Permanent Resident | Citizen |

## Identification Documents

| Document | Phase 0 | Tier A/B (CLP) | Tier C | Permanent Resident |
|----------|---------|----------------|--------|-------------------|
| **ICS credential** | Provisional | Full CLP card | Tier C card | N/A (green card) |
| **State ID** | State option | Yes (most states) | Yes | Yes |
| **Driver's license** | State option | Yes (most states) | Yes | Yes |
| **REAL ID compliant** | No | State option | Yes | Yes |
| **Passport** | Home country only | Home country only | Home country only | Yes (US) at citizenship |
| **Social Security Number** | No | Yes (work-authorized SSN) | Yes | Yes |

### Driver's License Framework

ICS does not mandate state licensing policy but:

- Encourages states to issue licenses to CLP holders
- Provides REAL ID alternative pathway for CLP holders (federal facility access card)
- Supports insurance access for licensed CLP drivers

## Sponsoring Family Members

| Relationship | CLP Ability to Sponsor | Notes |
|--------------|------------------------|-------|
| **Spouse** | No | Requires permanent resident status |
| **Children** | No | Requires permanent resident status |
| **Parents** | No | Requires citizenship |
| **Siblings** | No | Requires citizenship |

### Family Unity Provisions

- CLP holders cannot sponsor family but can be sponsored by US citizen/PR relatives
- ICS does not break up families present together at intake
- Derivative CLP for spouse and minor children present at intake

## Travel

| Travel Type | Phase 0 | Tier A/B (CLP) | Tier C |
|-------------|---------|----------------|--------|
| **Domestic (within US)** | Corridor restricted | Corridor restricted | Unrestricted |
| **International** | Not recommended | Risky (reentry uncertain) | Advance parole available |

### Advance Parole (Tier C)

- Application-based international travel authorization
- Reentry not guaranteed if circumstances change
- Generally approved for family emergencies, business needs

## Key Transitions

### Tier A to Tier B

| Requirement | Standard |
|-------------|----------|
| **Time in status** | 6-12 months |
| **Employment/training** | Active participation |
| **Language** | Progress toward CEFR A2 |
| **Compliance** | No serious violations |
| **Benefits gained** | Corridor city mobility, expanded services |

### Tier B to Tier C

| Requirement | Standard |
|-------------|----------|
| **Time in status** | 12-24 months |
| **Employment/training** | Stable employment or training completion |
| **Language** | CEFR B1 attainment |
| **Civic orientation** | Completion required |
| **Benefits gained** | Full US mobility, advance parole option |

### Tier C to Permanent Residence

| Requirement | Standard |
|-------------|----------|
| **Time in Tier C** | 12-24 months |
| **Continuous presence** | Substantial US presence |
| **No disqualifying conduct** | Criminal and civil standards |
| **Application filed** | Standard PR application |
| **Benefits gained** | Permanent status, family sponsorship, most federal benefits |
