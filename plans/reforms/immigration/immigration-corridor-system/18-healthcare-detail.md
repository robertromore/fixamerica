# Immigration Corridor System: Healthcare Detail

---

## Overview

This document provides detailed specifications for healthcare delivery within ICS, including clinic operations, mental health protocols, public health surveillance, provider networks, and healthcare financing mechanisms.

---

## Healthcare Delivery Model

### Tiered Access by Status

| Service | Phase 0 | Tier A/B | Tier C | Permanent Resident |
|---------|---------|----------|--------|-------------------|
| **Emergency care** | EMTALA | Full | Full | Full |
| **Primary care** | ICS clinics | ICS + community | Full access | Full access |
| **Specialty care** | Urgent only | Referral network | Full access | Full access |
| **Mental health** | Crisis + screening | ICS programs | Full access | Full access |
| **Dental** | Emergency | Basic at ICS clinics | Full | Full |
| **Vision** | Screening | Basic at ICS clinics | Full | Full |
| **Prescription drugs** | Essential list | Expanded formulary | Full | Full |

| **Existing site expansion** | 500-1,000 sites with increased capacity |
| **Mobile units** | 50-100 mobile health units for surge |
| **Telehealth infrastructure** | All FQHCs equipped for virtual visits |

### Staffing Standards

| Position | Ratio |
|----------|-------|
| **Primary care providers** | 1 per 1,500 patients |
| **Nurses** | 2 per provider |
| **Medical assistants** | 2 per provider |
| **Behavioral health** | 1 per 3,000 patients |
| **Care coordinators** | 1 per 500 CLP patients |
| **Interpreters** | Based on language needs |

### Services Required

| Category | Services |
|----------|----------|

| **Preventive** | Vaccinations, screenings, health education |

|---------|-------------------|

| **HIV** | Opt-out testing at intake; linkage to care |
| **Hepatitis B** | Serology at intake; vaccination if susceptible |
| **Hepatitis C** | Serology at intake; treatment if positive |
| **Syphilis** | Serology at intake; treatment if positive |
| **COVID-19** | Per CDC guidance; vaccination offered |
| **Other STIs** | Risk-based screening |

| Vaccine | Requirement |
|---------|-------------|

| **Tdap** | Required |
| **Polio** | Required for children; recommended for adults |

| **Hepatitis B** | Required series |
| **Influenza** | Offered annually |
| **COVID-19** | Per CDC guidance |

|-------|---------|

## Healthcare Financing

### Federal Funding Streams

| Source | Amount | Use |
|--------|--------|-----|
| **ICS healthcare appropriation** | $1.5-4.0B/year | Direct clinic funding |
| **HRSA FQHC expansion** | $500M-1.0B/year | New and expanded sites |
| **SAMHSA grants** | $200-400M/year | Behavioral health |
| **CDC funding** | $100-200M/year | Public health surveillance |
| **Emergency fund** | $500M reserve | Surge and outbreak response |

### Insurance and Coverage

| Coverage Type | CLP Eligibility |
|---------------|-----------------|
| **Medicaid** | State option; 5-year bar waived for ICS |
| **CHIP** | 5-year bar waived for children |
| **ACA marketplace** | Eligible with subsidies at Tier A/B |
| **Medicare** | Work credits accrue; eligible at PR |
| **Emergency Medicaid** | Available regardless of status |

### High-Cost Care and Payer of Last Resort

ICS establishes a Catastrophic Care Fund (CCF) for Tier A/B participants.

| Rule | Standard |
|------|----------|
| **Primary payer** | Medicaid/CHIP/ACA or employer coverage first |
| **Secondary payer** | CCF covers approved inpatient and specialty care |
| **Emergency care** | No prior authorization; CCF reimburses after EMTALA |
| **Non-emergent care** | Prior authorization required; independent medical review available |
| **Reimbursement rate** | Medicare-based rates with published add-ons |
| **Recovery** | Subrogation from liable third parties when applicable |
| **Patient cost share** | Capped and income-based; no denial for inability to pay |

### Cost Sharing

| Service | CLP Cost Share |
|---------|----------------|
| **FQHC visits** | Sliding scale (0-200% FPL) |
| **Prescriptions** | $0-10 copay on essential list |
| **Emergency care** | No cost share |
| **Specialty referrals** | Sliding scale |
| **Dental/vision** | Nominal copay |
