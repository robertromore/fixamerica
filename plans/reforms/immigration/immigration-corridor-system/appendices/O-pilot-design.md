# Appendix O: Pilot Design

---

## Overview

This appendix provides detailed specifications for ICS pilot programs, including site selection criteria, phased rollout, success metrics, and decision points for scaling. The pilot phase is designed to test core assumptions and refine operations before national implementation.

---

## Pilot Objectives

### Primary Objectives

| Objective | Success Indicator |
|-----------|-------------------|
| **Test placement algorithm** | Participants matched to appropriate cities; low transfer requests |
| **Validate tier advancement** | 70%+ advance on schedule; clear benchmarks work |
| **Assess community impact** | No service degradation; neutral/positive community attitudes |
| **Confirm fiscal assumptions** | Actual costs within 20% of projections |
| **Identify operational gaps** | Problems identified and fixed before scaling |

### Secondary Objectives

| Objective | Success Indicator |
|-----------|-------------------|
| **Build institutional capacity** | Trained staff; functioning systems |
| **Develop best practices** | Documented procedures; training materials |
| **Establish partnerships** | Employer networks; service provider relationships |
| **Generate political support** | Bipartisan endorsements; local champions |
| **Create demonstration effects** | Media coverage; other cities interested |

## Site Selection Criteria

### Hub City Requirements

| Criterion | Weight | Measure |
|-----------|--------|---------|
| **Labor market demand** | 25% | JOLTS data; employer interest |
| **Housing availability** | 20% | Vacancy rates; rental costs |
| **Political support** | 15% | Mayor/council endorsement |
| **Service capacity** | 15% | Healthcare, schools, transit |
| **Community receptivity** | 10% | Polling data; civic engagement |
| **Geographic diversity** | 10% | Regional representation |
| **Population decline history** | 5% | Census trends |

### Hub City Exclusions

| Exclusion | Rationale |
|-----------|-----------|
| **Major coastal metros** | Already high-immigration; test revitalization thesis |
| **Cities under 100,000** | Insufficient service capacity |
| **Cities over 500,000** | Start with mid-size before scaling |
| **State capitals** | Avoid political symbolism complications |
| **Border cities** | Separate operational challenges |

### Candidate Hub Cities (Illustrative)

| City | State | Population | Rationale |
|------|-------|------------|-----------|
| **Buffalo** | NY | 280,000 | Population loss; revitalization efforts; diverse economy |
| **Detroit** | MI | 640,000 | Significant capacity; strong civic infrastructure |
| **Des Moines** | IA | 215,000 | Tight labor market; welcoming history |
| **Dayton** | OH | 140,000 | Population decline; emerging refugee resettlement |
| **Syracuse** | NY | 150,000 | University presence; healthcare capacity |

### Corridor City Selection

| Criterion | Requirement |
|-----------|-------------|
| **Proximity** | Within 150 miles of hub city |
| **Population** | 30,000-150,000 |
| **Employment base** | Diversified economy |
| **Housing** | Affordable rental stock |
| **Voluntary participation** | City council resolution |

## Decision Points

### Month 10: Soft Launch Review

| Finding | Action |
|---------|--------|
| **Targets met** | Proceed to Phase 3 |
| **Minimum thresholds met** | Proceed with modifications |
| **Below minimum thresholds** | Pause expansion; intensive remediation |
| **Critical failure** | Program redesign or termination |

### Month 18: Mid-Pilot Review

| Finding | Action |
|---------|--------|
| **Targets met** | Accelerate expansion planning |
| **Minimum thresholds met** | Continue; targeted improvements |
| **Below minimum thresholds** | Scale back; major modifications |
| **Critical failure** | Congressional notification; redesign |

### Month 30: Scale Decision

| Finding | Action |
|---------|--------|
| **Sustained success** | National rollout authorization |
| **Mixed results** | Extended pilot; selective expansion |
| **Persistent problems** | Major redesign before expansion |
| **Fundamental issues** | Program termination recommendation |

## Evaluation Framework

### Evaluation Design

| Component | Method |
|-----------|--------|
| **Process evaluation** | Qualitative; administrative data |
| **Outcome evaluation** | Quasi-experimental comparison |
| **Impact evaluation** | Difference-in-differences; synthetic control |
| **Cost-benefit analysis** | Full accounting; long-term projections |

### Comparison Groups

| Group | Purpose |
|-------|---------|
| **Non-ICS arrivals (same period)** | Counterfactual outcomes |
| **Pre-ICS arrivals (same cities)** | Historical comparison |
| **Similar cities without ICS** | Geographic comparison |
| **Legacy track vs. new arrivals** | Pathway comparison |

### Independent Evaluation

| Requirement | Rationale |
|-------------|-----------|
| **External evaluator** | Independence from implementation |
| **Pre-registered analysis** | Prevent cherry-picking |
| **Public reporting** | Transparency |
| **Peer review** | Credibility |

## Risk Management

### Pilot-Specific Risks

| Risk | Mitigation |
|------|------------|
| **City withdrawal** | Multi-city pilot; backup sites identified |
| **Political opposition** | Community engagement; early wins emphasis |
| **Participant concentration** | Strict capacity limits; dispersion monitoring |
| **Negative incident** | Crisis communication plan; rapid response |
| **Cost overruns** | Conservative budgeting; contingency reserves |

### Contingency Plans

| Scenario | Response |
|----------|----------|
| **Hub city exits** | Transfer participants to remaining hubs; activate backup site |
| **Intake surge** | Activate pause triggers; expand corridor cities |
| **Community backlash** | Enhanced RID; community dialogue; pause if needed |
| **Negative media** | Pre-prepared responses; local voices emphasis |
| **Federal funding disruption** | City bridge funding; reduced operations |

## Transition to National Scale

### Criteria for National Rollout

| Criterion | Threshold |
|-----------|-----------|
| **Pilot success metrics** | Minimum thresholds sustained 12+ months |
| **Cost projections validated** | Within 20% of estimates |
| **Community acceptance demonstrated** | Net positive attitudes |
| **Operational systems proven** | 99% uptime; <10 critical issues |
| **Political viability confirmed** | Bipartisan Congressional support |
| **Interstate compact foundation** | 3+ states in compact |

### Scaling Sequence

| Phase | Cities | Timeline |
|-------|--------|----------|
| **Pilot** | 5 hub cities | Years 1-3 |
| **Wave 2** | 10-15 hub cities | Years 3-4 |
| **Wave 3** | 15-20 hub cities | Years 4-5 |
| **Full operation** | All designated hubs | Year 5+ |
