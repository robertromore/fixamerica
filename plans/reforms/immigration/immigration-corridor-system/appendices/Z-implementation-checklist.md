# Appendix Z: Implementation Checklist

---

## Overview

This appendix provides comprehensive checklists for ICS implementation at the federal, state, and local levels. These checklists guide the launch sequence, city onboarding, operational readiness, and ongoing compliance verification.

---

## Federal Implementation Checklist

### Phase 1: Legislative and Regulatory Foundation

#### Legislation

- [ ] **ICS Act enacted** - Core statutory authority
- [ ] **Appropriations secured** - Initial funding for Year 1
- [ ] **IIOO established** - Independent oversight office created
- [ ] **Interstate compact authorized** - Congressional consent

#### Regulations

- [ ] **NPRM published** - Notice of proposed rulemaking (DHS)
- [ ] **Comment period completed** - 60-day public comment
- [ ] **Final rule published** - CLP regulations finalized
- [ ] **DOL regulations** - Employer registration rules
- [ ] **HUD regulations** - RID payment rules
- [ ] **ED regulations** - EL surge funding rules
- [ ] **HHS regulations** - Healthcare access rules

#### Interagency Coordination

- [ ] **MOUs executed** - DHS-DOL-HUD-ED-HHS-SSA-IRS
- [ ] **Data sharing agreements** - Between agencies
- [ ] **Joint operations plan** - Coordinated implementation
- [ ] **Communication protocols** - Interagency coordination

### Phase 2: Infrastructure Development

#### IT Systems

- [ ] **Core database operational** - Participant records system
- [ ] **Biometric system deployed** - Fingerprint and facial capture
- [ ] **E-Verify integration** - Employment verification
- [ ] **FBI/DHS integration** - Background checks
- [ ] **SSA integration** - SSN verification/issuance
- [ ] **Staff portal launched** - Case management system
- [ ] **Participant portal launched** - Self-service applications
- [ ] **Employer portal launched** - Registration and verification
- [ ] **Mobile apps released** - Participant and staff apps
- [ ] **Public dashboard live** - Transparency reporting

#### Physical Infrastructure

- [ ] **National processing center** - Primary intake facility
- [ ] **Regional offices** - 10 regional coordination centers
- [ ] **Call center operational** - Multilingual support
- [ ] **Document production** - CLP card printing capability

#### Staffing

- [ ] **Leadership hired** - Director; deputies; regional directors
- [ ] **Core staff hired** - 50% of Year 1 positions
- [ ] **Training completed** - Initial staff training
- [ ] **Contractor support** - Processing surge capacity

### Phase 3: Program Launch Preparation

#### Policies and Procedures

- [ ] **Operations manual** - Complete standard operating procedures
- [ ] **Adjudication guidance** - Decisional standards
- [ ] **Appeals procedures** - Hearing protocols
- [ ] **Compliance manual** - Violation handling
- [ ] **Emergency protocols** - Crisis response plans

#### Partner Readiness

- [ ] **State compacts signed** - Minimum 10 states
- [ ] **Hub cities designated** - Minimum 5 cities
- [ ] **Service provider contracts** - ESL, housing, employment
- [ ] **Employer registration open** - Pre-launch registration

#### Communication

- [ ] **Public information campaign** - Awareness of ICS
- [ ] **Stakeholder briefings** - Congress, advocates, employers
- [ ] **Media strategy** - Launch communications plan
- [ ] **Multilingual materials** - Forms and guides in 10+ languages

## State Compact Checklist

### Pre-Joining Assessment

- [ ] **Statutory authority review** - State can enter compact
- [ ] **Fiscal impact analysis** - Costs and benefits assessed
- [ ] **Service capacity assessment** - Existing infrastructure
- [ ] **Political feasibility** - Legislative support
- [ ] **Stakeholder consultation** - Local government, advocates, business

### Compact Execution

- [ ] **Governor authorization** - Executive authority to sign
- [ ] **Legislative approval** - If required by state law
- [ ] **Compact signed** - State joins interstate compact
- [ ] **State coordinator designated** - Implementation lead
- [ ] **MOU with federal ICS** - Operational agreement

### State Implementation

- [ ] **State regulations adopted** - Implementing rules
- [ ] **Licensing reciprocity** - Driver's license; professional licenses
- [ ] **Benefit eligibility clarified** - State program access
- [ ] **Law enforcement training** - Police training on CLP
- [ ] **Court notification protocols** - Criminal conviction reporting
- [ ] **State agency coordination** - Health, education, labor agencies

### Ongoing Compliance

- [ ] **Annual report submitted** - To federal ICS and compact
- [ ] **Performance targets met** - State obligations
- [ ] **Dispute resolution** - Issues resolved through compact process
- [ ] **Compact dues paid** - State contributions current

## Hub City Onboarding Checklist

### Phase 1: Application and Designation (Months 1-3)

#### City Application

- [ ] **Interest expressed** - City council resolution
- [ ] **Application submitted** - Form ICS-C1
- [ ] **Capacity assessment completed** - Housing, services, schools
- [ ] **Community input gathered** - Public meetings held
- [ ] **Mayor's commitment letter** - Executive support documented

#### Federal Review

- [ ] **Capacity verified** - ICS site visit completed
- [ ] **Community support confirmed** - Survey results reviewed
- [ ] **Economic analysis** - Labor market and fiscal review
- [ ] **Designation decision** - Federal approval issued
- [ ] **Allocation assigned** - Initial participant capacity set

### Phase 2: Infrastructure Development (Months 4-6)

#### Physical Infrastructure

- [ ] **Intake center identified** - Location secured
- [ ] **Intake center equipped** - Biometrics, IT, furniture
- [ ] **Welcome center established** - Orientation facility
- [ ] **Office space secured** - Case management offices

#### Housing

- [ ] **Housing inventory** - Available units identified
- [ ] **Landlord partnerships** - MOUs with property owners
- [ ] **Quality standards** - Inspection protocols established
- [ ] **Placement algorithm configured** - Matching criteria set

#### Services

- [ ] **ESL providers contracted** - Classroom and online
- [ ] **Healthcare access** - FQHC capacity confirmed; MOUs signed
- [ ] **Employment services** - Workforce development partnership
- [ ] **Legal services** - Pro bono network established
- [ ] **Childcare** - Capacity for participant families
- [ ] **Transportation** - Transit access; driver's license pathway

### Phase 3: Staffing and Training (Months 5-7)

#### Hiring

- [ ] **Hub director hired** - Local leadership in place
- [ ] **Case managers hired** - Caseload capacity met
- [ ] **Intake specialists hired** - Processing capacity met
- [ ] **Support staff hired** - Administrative capacity met
- [ ] **Interpreters available** - Language coverage ensured

#### Training

- [ ] **Core training completed** - All staff Level 1-2
- [ ] **Role-specific training** - Specialized modules
- [ ] **Systems training** - IT proficiency verified
- [ ] **Dry run conducted** - Simulated processing
- [ ] **Certification achieved** - Staff certifications current

### Phase 4: Partner Coordination (Months 6-8)

#### Local Government

- [ ] **Police briefed** - CLP status training
- [ ] **Schools notified** - EL surge preparation
- [ ] **Health department coordinated** - Public health protocols
- [ ] **Social services aligned** - Referral pathways

#### Community Partners

- [ ] **Faith organizations engaged** - Volunteer coordination
- [ ] **Civic organizations briefed** - Community orientation
- [ ] **Employer council formed** - Business engagement
- [ ] **Advocacy groups consulted** - Ongoing communication

### Phase 5: Launch Preparation (Months 8-9)

#### Operational Readiness

- [ ] **Systems tested** - End-to-end processing verified
- [ ] **Workflows documented** - SOPs finalized
- [ ] **Quality assurance** - Audit protocols in place
- [ ] **Escalation procedures** - Issue resolution defined
- [ ] **Backup systems** - Contingency plans tested

#### Communication

- [ ] **Community informed** - Launch announcement
- [ ] **Media briefed** - Press materials prepared
- [ ] **Hotline operational** - Community questions answered
- [ ] **Signage installed** - Wayfinding at facilities

### Phase 6: Soft Launch (Month 10)

- [ ] **Limited intake begins** - 25% of capacity
- [ ] **Daily monitoring** - Issue identification
- [ ] **Rapid iteration** - Process adjustments
- [ ] **Staff support** - Intensive supervision
- [ ] **After-action reviews** - Lessons learned documented

### Phase 7: Full Operations (Month 12+)

- [ ] **Full capacity reached** - 100% allocation
- [ ] **Performance targets met** - KPIs achieved
- [ ] **Continuous improvement** - Ongoing optimization
- [ ] **Annual evaluation** - Comprehensive assessment

## Employer Registration Checklist

### Registration Application

- [ ] **Account created** - Employer portal registration
- [ ] **Company information** - EIN, name, address, industry
- [ ] **Authorized users** - Designated representatives
- [ ] **Attestations signed** - Compliance commitments
- [ ] **Application fee paid** - If applicable

### Verification

- [ ] **EIN verified** - IRS confirmation
- [ ] **Business license verified** - State registration
- [ ] **OSHA history reviewed** - Safety record check
- [ ] **WHD history reviewed** - Wage and hour compliance
- [ ] **Debarment check** - Not debarred from federal programs

### Approval

- [ ] **Registration approved** - Employer ID issued
- [ ] **Employer trained** - Compliance training completed
- [ ] **E-Verify enrollment** - If not already enrolled
- [ ] **Prevailing wage determined** - For job classifications
- [ ] **Posting requirements** - Workplace notices displayed

### Ongoing Compliance

- [ ] **Quarterly verification** - Employment confirmation
- [ ] **Annual renewal** - Registration renewed
- [ ] **Audit cooperation** - Respond to WHD inquiries
- [ ] **Violation correction** - Address any findings

## Launch Go/No-Go Checklist

### Critical Requirements (Must Be Met)

- [ ] **IT systems operational** - Core functions working
- [ ] **Biometrics functional** - Capture and matching
- [ ] **Background check integration** - FBI/DHS connected
- [ ] **Staff trained** - Minimum competency achieved
- [ ] **Housing available** - Placement capacity exists
- [ ] **Legal authority** - Regulations effective
- [ ] **Funding available** - Appropriations released

### Important Requirements (Should Be Met)

- [ ] **All staff hired** - Positions filled
- [ ] **All training complete** - Full curriculum
- [ ] **Partner contracts executed** - All MOUs signed
- [ ] **Community prepared** - Stakeholders informed
- [ ] **Public dashboard ready** - Transparency tool

### Desirable Requirements (Nice to Have)

- [ ] **Mobile apps deployed** - Enhanced functionality
- [ ] **Full automation** - All integrations complete
- [ ] **Volunteer network** - Community support ready

### Decision

| Status | Action |
|--------|--------|
| **All critical met; most important met** | GO |
| **All critical met; some important not met** | GO with mitigation |
| **Any critical not met** | NO-GO; resolve and reassess |
