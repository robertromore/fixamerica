# Appendix C: Comparison Systems

---

## Overview

This appendix compares ICS to existing immigration systems in the United States and internationally, highlighting key differences and lessons learned.

---

## Comparison to Current US System

### Cost Comparison

| Current System Component | Annual Cost | ICS Replacement |
|--------------------------|-------------|-----------------|
| **CBP operations** | $18.5B | Partially retained |
| **ICE enforcement** | $8.5B | Reduced by ~40% |
| **Immigration courts backlog** | $1.2B | Reduced via streamlined process |
| **Emergency shelter (FEMA/HHS)** | $2-4B (variable) | Replaced by planned housing |
| **State/local emergency services** | $5-10B (estimated) | Reduced via compensation |
| **Employer enforcement** | $0.5B | Retained/expanded |
| **Total current system** | ~$35-42B | ICS: $12-34B at scale |

### Structural Comparison

| Feature | Current System | ICS |
|---------|---------------|-----|
| **Legal status** | Binary (legal/unauthorized) | Graduated tiers |
| **Work authorization** | Tied to specific visa | Universal for CLP holders |
| **Geographic distribution** | Unmanaged | Structured placement |
| **Community impact** | Uncompensated | RID payments |
| **Enforcement** | Removal-focused | Integration-focused |
| **Timeline to permanence** | Decades (if ever) | 5-10 years |

## Comparison to Past US Immigration Legislation

### Immigration Reform and Control Act (IRCA) 1986

| Feature | IRCA 1986 | ICS |
|---------|-----------|-----|
| **Legalization** | One-time amnesty | Ongoing system |
| **Work authorization** | Immediate | Graduated |
| **Geographic requirements** | None | Corridor residency |
| **Community compensation** | None | RID payments |
| **Employer sanctions** | New requirement | Enhanced |
| **Long-term outcome** | Large unauthorized population returned | Designed for ongoing management |

**IRCA lessons for ICS:**

- One-time amnesty creates expectation of future amnesty
- Employer sanctions require sustained enforcement
- Interior enforcement must accompany legalization

### Comprehensive Immigration Reform (2007, 2013)

| Feature | 2007/2013 Proposals | ICS |
|---------|---------------------|-----|
| **Point system** | Yes (2007) | No |
| **Path to citizenship** | Long (13+ years) | 5-10 years via tiers |
| **Border security trigger** | Yes | No trigger |
| **Guest worker program** | Yes | Integrated into CLP |
| **Community compensation** | No | RID payments |

**2007/2013 lessons for ICS:**

- Triggers create political uncertainty
- Complexity enables opposition
- Community concerns must be addressed directly

### DREAM Act Variants

| Feature | DREAM Act | ICS |
|---------|-----------|-----|
| **Target population** | Youth arrivals | All eligible |
| **Requirements** | Education/military | Employment/training/language |
| **Geographic restrictions** | None | Corridor residency |
| **Family inclusion** | Limited | Derivative status |

## International Comparison: Canada

### Express Entry System

| Feature | Canada Express Entry | ICS |
|---------|---------------------|-----|
| **Selection mechanism** | Point-based ranking | Capacity-based placement |
| **Processing time** | 6 months | Phased over 3-5 years |
| **Status upon arrival** | Permanent residence | CLP (graduated) |
| **Geographic distribution** | Provincial nominee programs | Hub/corridor placement |
| **Integration support** | Settlement services | Comprehensive (housing, education, RID) |

**Canada lessons for ICS:**

- Provincial programs address geographic distribution
- Settlement services improve outcomes
- Clear timelines reduce uncertainty

### Provincial Nominee Programs (PNP)

| Feature | Canada PNP | ICS Corridors |
|---------|-----------|---------------|
| **Local role** | Province nominates | City consents to designation |
| **Retention mechanism** | Nominee commitment | Corridor residency requirement |
| **Labor market alignment** | Provincial labor needs | Local capacity and demand |
| **Flexibility** | Province sets criteria | Federal standards with local input |

## International Comparison: Germany

### Integration Courses (Integrationskurse)

| Feature | Germany | ICS |
|---------|---------|-----|
| **Language requirement** | 600 hours minimum | CEFR B1 for Tier C |
| **Civic orientation** | 100 hours | Required module |
| **Funding** | Federal (mostly) | Federal |
| **Participation** | Mandatory for many | Required for advancement |
| **Completion rate** | ~60% | Target: 80%+ |

**Germany lessons for ICS:**

- Mandatory language training is feasible
- Completion rates require support services
- Civic orientation aids integration

### Residence Permit System

| Feature | Germany | ICS |
|---------|---------|-----|
| **Temporary status** | Aufenthaltserlaubnis | CLP Tiers A/B |
| **Permanent status** | Niederlassungserlaubnis | CLP Tier C / PR |
| **Time requirement** | 5 years | 3-5 years to Tier C |
| **Language requirement** | B1 for permanent | B1 for Tier C |
| **Employment requirement** | Self-supporting | Employed or in training |

## International Comparison: Australia

### Points-Based System

| Feature | Australia | ICS |
|---------|-----------|-----|
| **Selection** | Points (age, skills, language) | Capacity-based placement |
| **Employer sponsorship** | Key pathway | Sector partnerships |
| **Regional incentives** | Regional visa bonus | Corridor placement requirement |
| **Processing** | Pre-arrival | Phased post-arrival |

**Australia lessons for ICS:**

- Regional incentives can distribute population
- Employer engagement requires incentives
- Skills assessment improves labor market fit

### Regional Migration Programs

| Feature | Australia Regional | ICS Corridors |
|---------|-------------------|---------------|
| **Requirement** | 3 years regional residence | Residency until Tier C |
| **Incentive** | Points bonus / faster PR | Tier advancement |
| **Enforcement** | Visa condition | Status tied to location |
| **Outcomes** | Mixed retention | TBD |

## International Comparison: European Union

### Dublin Regulation (Asylum Distribution)

| Feature | Dublin System | ICS |
|---------|--------------|-----|
| **Distribution mechanism** | First-entry country | Capacity-based placement |
| **Compliance** | Poor (secondary movement) | Residency requirements |
| **Burden sharing** | Minimal | RID and stabilization grants |
| **Outcome** | System strain in entry countries | Distributed capacity |

**Dublin lessons for ICS:**

- Distribution without incentives fails
- Secondary movement requires management
- Community compensation reduces resistance

### EU Blue Card

| Feature | EU Blue Card | ICS |
|---------|-------------|-----|
| **Target** | High-skilled workers | All eligible populations |
| **Mobility** | Cross-EU after 18 months | US-wide at Tier C |
| **Family** | Immediate reunification | Derivative CLP |
| **Status** | Temporary renewable | Graduated to permanent |

## Key Differentiators of ICS

| Feature | ICS Innovation | Rationale |
|---------|---------------|-----------|
| **Community compensation (RID)** | Direct payments to host residents | Addresses local concerns; builds support |
| **Geographic management** | Structured hub/corridor placement | Prevents concentration; matches capacity |
| **Graduated status** | Tier system with automatic advancement | Reduces uncertainty; incentivizes compliance |
| **Fail-graceful design** | System slows rather than collapses | Maintains order under stress |
| **Universal work authorization** | All CLP holders can work | Enables self-sufficiency |
| **Integrated oversight** | Independent office with real authority | Protects civil liberties |

---

## Lessons Applied in ICS Design

| Lesson Source | Lesson | ICS Application |
|---------------|--------|-----------------|
| **IRCA 1986** | One-time amnesty creates expectations | Ongoing system, not one-time |
| **2007/2013** | Triggers create political problems | No enforcement triggers |
| **Canada PNP** | Local involvement improves distribution | Hub/corridor consent |
| **Germany** | Language training is feasible | B1 requirement for advancement |
| **Australia** | Regional incentives can work | Corridor residency with advancement |
| **Dublin** | Distribution needs incentives | RID and stabilization grants |
