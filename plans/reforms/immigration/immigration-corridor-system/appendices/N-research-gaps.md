# Appendix N: Research Gaps

---

## Overview

This appendix identifies areas where additional research, data collection, or pilot testing is needed before ICS implementation. Each gap is prioritized and linked to specific program components.

---

## Research Gap Categories

| Category | Description |
|----------|-------------|
| **Empirical** | Requires data collection or analysis |
| **Behavioral** | Requires understanding human responses |
| **Implementation** | Requires operational testing |
| **Legal** | Requires legal analysis or test cases |
| **Economic** | Requires economic modeling or forecasting |

---

## Priority 1: Critical Path Gaps

### Gap 1.1: Optimal Tier Duration

| Attribute | Detail |
|-----------|--------|
| **Category** | Empirical/Behavioral |
| **Question** | What tier durations maximize integration outcomes while maintaining participant buy-in? |
| **Current assumption** | 6-12 months (Tier A), 12-24 months (Tier B), 12-24 months (Tier C) |
| **Risk if wrong** | Too short: insufficient integration; Too long: attrition, frustration |
| **Research needed** | Longitudinal study of integration outcomes by residency duration |
| **Data sources** | Comparable programs (Canada, Germany); refugee resettlement data |
| **Timeline** | Pre-pilot (Year 0) |

### Gap 1.2: Placement Algorithm Fairness

| Attribute | Detail |
|-----------|--------|
| **Category** | Implementation/Legal |
| **Question** | Does the placement algorithm produce disparate impact by national origin, race, or family status? |
| **Current assumption** | Algorithm is neutral based on stated factors |
| **Risk if wrong** | Legal challenges; concentrated communities by origin |
| **Research needed** | Simulation testing with demographic data; legal review |
| **Data sources** | Historical arrival data; census demographics of hub cities |
| **Timeline** | Pre-pilot (Year 0) |

### Gap 1.3: Community Capacity Thresholds

| Attribute | Detail |
|-----------|--------|
| **Category** | Empirical |
| **Question** | What is the actual capacity limit before community services become strained? |
| **Current assumption** | 2% of population/year; 5% school enrollment |
| **Risk if wrong** | Community backlash if underestimated; underutilization if overestimated |
| **Research needed** | Case studies of rapid population changes; school district analysis |
| **Data sources** | Post-Katrina migration; oil boom towns; refugee resettlement data |
| **Timeline** | Pre-pilot and ongoing |

### Gap 1.4: RID Effectiveness

| Attribute | Detail |
|-----------|--------|
| **Category** | Behavioral/Economic |
| **Question** | Do RID payments actually improve community receptivity? |
| **Current assumption** | Financial compensation reduces opposition |
| **Risk if wrong** | Wasted expenditure; accusations of bribery |
| **Research needed** | Polling/focus groups; comparison with other compensation programs |
| **Data sources** | PILOT payment acceptance; military base community attitudes |
| **Timeline** | Pilot evaluation (Year 1-2) |

### Gap 1.5: Language Acquisition Timelines

| Attribute | Detail |
|-----------|--------|
| **Category** | Empirical |
| **Question** | What ESL intensity is needed to achieve B1 within program timelines? |
| **Current assumption** | 10-15 hours/week reaches B1 in 18-24 months |
| **Risk if wrong** | Bottleneck at tier advancement; unrealistic expectations |
| **Research needed** | Meta-analysis of adult ESL programs; literacy considerations |
| **Data sources** | USCIS citizenship test pass rates; refugee ESL outcomes |
| **Timeline** | Pre-pilot (Year 0) |

## Priority 2: Important Design Gaps

### Gap 2.1: Legacy Track Enrollment Rates

| Attribute | Detail |
|-----------|--------|
| **Category** | Behavioral |
| **Question** | What percentage of unauthorized population will register if offered? |
| **Current assumption** | 70-80% registration rate |
| **Risk if wrong** | Higher: capacity strain; Lower: large remaining unauthorized population |
| **Research needed** | Survey research; comparison with DACA, TPS enrollment |
| **Data sources** | DACA application data; TPS registration patterns |
| **Timeline** | Pre-implementation (Year 0-1) |

### Gap 2.2: Employment Sector Matching

| Attribute | Detail |
|-----------|--------|
| **Category** | Economic |
| **Question** | Can hub/corridor cities actually absorb placed workers in appropriate sectors? |
| **Current assumption** | Labor market analysis identifies demand |
| **Risk if wrong** | Underemployment; skills mismatch; worker frustration |
| **Research needed** | Detailed sector analysis by city; employer willingness surveys |
| **Data sources** | BLS JOLTS data; employer interviews |
| **Timeline** | Pre-pilot and ongoing |

### Gap 2.3: Interstate Compact Viability

| Attribute | Detail |
|-----------|--------|
| **Category** | Legal/Implementation |
| **Question** | Will sufficient states join to create viable corridor regions? |
| **Current assumption** | Voluntary participation with federal incentives |
| **Risk if wrong** | Fragmented implementation; coverage gaps |
| **Research needed** | State-by-state political analysis; governor outreach |
| **Data sources** | Other interstate compact adoption patterns |
| **Timeline** | Pre-enactment (Year -1 to 0) |

### Gap 2.4: Family Separation Prevention

| Attribute | Detail |
|-----------|--------|
| **Category** | Implementation |
| **Question** | Does the placement system effectively keep families together? |
| **Current assumption** | Family unit placement with hardship provisions |
| **Risk if wrong** | Family separation; legal challenges; humanitarian concerns |
| **Research needed** | Process testing; case review protocols |
| **Data sources** | Current family separation data; reunification timelines |
| **Timeline** | Pilot (Year 1) |

### Gap 2.5: Mental Health Service Capacity

| Attribute | Detail |
|-----------|--------|
| **Category** | Empirical |
| **Question** | Is there sufficient culturally competent mental health capacity in hub cities? |
| **Current assumption** | FQHC expansion can meet demand |
| **Risk if wrong** | Untreated trauma; integration failures |
| **Research needed** | Provider inventory; training capacity assessment |
| **Data sources** | SAMHSA treatment locator; refugee mental health data |
| **Timeline** | Pre-pilot (Year 0) |

## Priority 3: Operational Refinement Gaps

### Gap 3.1: Biometric System Performance

| Attribute | Detail |
|-----------|--------|
| **Category** | Implementation |
| **Question** | What are error rates for biometric verification in field conditions? |
| **Current assumption** | Low false positive/negative rates |
| **Risk if wrong** | Wrongful denials; identity fraud |
| **Research needed** | Field testing; error rate analysis |
| **Data sources** | DHS biometric systems; NIST testing data |
| **Timeline** | Pre-pilot (Year 0) |

### Gap 3.2: Housing Market Response

| Attribute | Detail |
|-----------|--------|
| **Category** | Economic |
| **Question** | How do housing markets in hub cities respond to concentrated demand? |
| **Current assumption** | Landlord incentives prevent price spikes |
| **Risk if wrong** | Rent increases harm existing residents; displacement |
| **Research needed** | Housing market modeling; historical case analysis |
| **Data sources** | HUD rental data; college town housing patterns |
| **Timeline** | Pilot monitoring (Year 1-2) |

### Gap 3.3: School Integration Outcomes

| Attribute | Detail |
|-----------|--------|
| **Category** | Empirical |
| **Question** | What educational interventions produce best outcomes for EL students? |
| **Current assumption** | Funded ESL support enables grade-level achievement |
| **Risk if wrong** | Poor educational outcomes; long-term integration failure |
| **Research needed** | Meta-analysis of EL interventions; pilot testing |
| **Data sources** | NAEP data; refugee student outcomes; WIDA research |
| **Timeline** | Ongoing |

### Gap 3.4: Employer Compliance Rates

| Attribute | Detail |
|-----------|--------|
| **Category** | Implementation |
| **Question** | Will employers actually register and follow sector-specific requirements? |
| **Current assumption** | Incentives and enforcement achieve compliance |
| **Risk if wrong** | Exploitation; wage theft; program integrity issues |
| **Research needed** | Pilot employer monitoring; compliance surveys |
| **Data sources** | H-2 program compliance data; WHD enforcement patterns |
| **Timeline** | Pilot (Year 1-2) |

### Gap 3.5: Advance Parole Risk Assessment

| Attribute | Detail |
|-----------|--------|
| **Category** | Behavioral/Legal |
| **Question** | What is optimal advance parole policy to balance travel needs with return compliance? |
| **Current assumption** | Tier C eligible; country-specific risk assessment |
| **Risk if wrong** | High overstay rates; too restrictive for legitimate travel |
| **Research needed** | Historical parole data analysis; comparative policy review |
| **Data sources** | USCIS advance parole statistics; TPS travel patterns |
| **Timeline** | Pre-implementation |

| **Research needed** | Literature review; community input; metric validation |
| **Data sources** | Academic integration research; European integration indices |
| **Timeline** | Pre-pilot (Year 0) |

## Research Methodology Recommendations

### Pre-Pilot Research (Year 0)

| Method | Application |
|--------|-------------|
| **Simulation modeling** | Placement algorithm testing; capacity modeling |
| **Survey research** | Legacy population intentions; employer willingness |
| **Case study analysis** | Comparable programs; rapid population change communities |
| **Legal review** | Constitutional analysis; compact viability |
| **Stakeholder interviews** | City readiness; provider capacity |

### Pilot Research (Years 1-2)

| Method | Application |
|--------|-------------|
| **Randomized controlled trials** | ESL interventions; RID messaging |
| **Process evaluation** | System functioning; bottleneck identification |
| **Outcome tracking** | Employment; housing; tier advancement |
| **Community surveys** | Attitude changes; service satisfaction |
| **Administrative data analysis** | Program efficiency; cost tracking |

### Ongoing Research (Years 3+)

| Method | Application |
|--------|-------------|
| **Longitudinal cohort studies** | Long-term outcomes; naturalization |
| **Impact evaluation** | Comparison with non-ICS populations |
| **Cost-benefit analysis** | Fiscal impact; economic contribution |
| **Policy analysis** | What works; program refinement |

## Research Partnerships

### Recommended Partners

| Partner Type | Examples | Role |
|--------------|----------|------|
| **Federal agencies** | Census, BLS, USCIS ORD | Data access; administrative records |
| **Universities** | Migration Policy Institute; Urban Institute | Independent evaluation |
| **Foundations** | Ford, MacArthur, Carnegie | Funding; convening |
| **International** | OECD, UNHCR, IOM | Comparative analysis |
| **Community organizations** | Ethnic CBOs, faith groups | Ground-level insights |

### Data Sharing Framework

| Data Type | Access Level | Protection |
|-----------|--------------|------------|
| **Aggregate outcomes** | Public | Dashboard publication |
| **De-identified micro-data** | Researchers | IRB approval; data use agreements |
| **Identified data** | Federal only | Privacy Act; limited access |

## Research Governance

### Independent Review

| Component | Function |
|-----------|----------|
| **Research Advisory Board** | Methodology review; priority setting |
| **IRB oversight** | Human subjects protection |
| **Peer review** | Publication of findings |
| **Public comment** | Community input on research priorities |

### Conflict of Interest

| Rule | Rationale |
|------|-----------|
| **No evaluator-implementer overlap** | Independence |
| **Disclosed funding sources** | Transparency |
| **Pre-registered analysis plans** | Prevent cherry-picking |
