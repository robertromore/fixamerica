# Immigration Corridor System: Criminal Justice Interaction

---

## Overview

This document details how CLP holders interact with the criminal justice system, the immigration consequences of criminal conduct, coordination between criminal courts and immigration authorities, and the procedural protections that apply. The system treats CLP holders as subject to the same criminal laws as citizens while imposing additional immigration consequences for serious offenses.

---

## Core Principles

### Equal Treatment Under Criminal Law

| Principle | Application |
|-----------|-------------|
| **Same laws apply** | CLP holders subject to identical criminal statutes |
| **Same rights apply** | Full constitutional protections in criminal proceedings |
| **Same courts** | State/federal criminal courts, not immigration courts |
| **Same penalties** | Fines, probation, incarceration as for citizens |
| **Additional consequences** | Immigration consequences for qualifying offenses |

### Distinction from Immigration Enforcement

| Criminal System | Immigration System |
|-----------------|---------------------|
| **Burden of proof** | Beyond reasonable doubt | Clear and convincing |
| **Right to counsel** | Appointed if indigent | At own expense |
| **Purpose** | Punishment; rehabilitation | Status determination |
| **Proceedings** | Public criminal trial | Administrative hearing |

## Categories of Offenses

### Minor Offenses (No Immigration Consequence)

| Category | Examples | CLP Effect |
|----------|----------|------------|
| **Traffic violations** | Speeding, parking | None |
| **Minor infractions** | Littering, noise | None |
| **First-time minor misdemeanors** | Petty theft, simple trespass | None (unless pattern) |

### Moderate Offenses (Tier Freeze)

| Category | Examples | CLP Effect |
|----------|----------|------------|
| **Repeat misdemeanors** | Multiple minor offenses | Tier advancement frozen |
| **DUI (first offense)** | Operating under influence | Tier freeze; monitoring |
| **Simple assault** | Non-aggravated assault | Tier freeze pending resolution |
| **Drug possession (personal)** | Small quantities | Tier freeze; treatment option |

### Serious Offenses (CLP Review)

| Category | Examples | CLP Effect |
|----------|----------|------------|
| **Felony conviction** | Any felony | CLP status reviewed |
| **Aggravated assault** | Assault with weapon/injury | Likely revocation |
| **Drug trafficking** | Distribution offenses | Revocation |
| **Fraud** | Immigration fraud, financial crimes | Revocation |
| **Domestic violence** | DV convictions | Revocation likely |

### Disqualifying Offenses (Automatic Revocation)

| Category | Examples | CLP Effect |
|----------|----------|------------|
| **Murder** | Any degree | Automatic revocation |
| **Rape/sexual assault** | Sexual violence offenses | Automatic revocation |
| **Child abuse** | Child abuse/exploitation | Automatic revocation |
| **Terrorism** | Terrorism offenses | Automatic revocation |
| **Trafficking** | Human trafficking | Automatic revocation |
| **Aggravated felony** | INA ยง 101(a)(43) offenses | Automatic revocation |

## Procedural Protections

### Criminal Proceedings

| Right | Protection |
|-------|------------|
| **Presumption of innocence** | Until proven guilty |
| **Right to counsel** | Appointed if indigent (Sixth Amendment) |
| **Right to jury** | Jury trial for serious offenses |
| **Right to confront** | Cross-examine witnesses |
| **Right to remain silent** | Fifth Amendment privilege |
| **Speedy trial** | Sixth Amendment right |

### Immigration Consequences Notice

| Requirement | Standard |
|-------------|----------|
| **Padilla warning** | Defense counsel must advise of immigration consequences |
| **ICS-specific notice** | Court must provide ICS consequence notice |
| **Interpreter** | Translation of consequences |
| **Documentation** | Written acknowledgment |

### Post-Conviction Immigration Review

| Step | Timeline |
|------|----------|
| **Conviction notification** | Court notifies ICS within 30 days |
| **ICS review** | Assessment within 60 days |
| **Notice to participant** | Written notice of proposed action |
| **Hearing opportunity** | Right to contest before IIOO |
| **Decision** | Within 90 days of conviction |

## Coordination Between Systems

### Information Sharing

| From | To | Information |
|------|----|-------------|
| **Courts** | ICS | Conviction records; sentencing |
| **Law enforcement** | ICS | Arrests (limited); warrants |
| **ICS** | Courts | Status verification |
| **Corrections** | ICS | Release dates; detainers |

### Detainers and Holds

| Policy | Standard |
|--------|----------|
| **No civil detainers** | ICS does not issue immigration detainers for arrests |
| **Post-conviction only** | Action only after conviction, not arrest |
| **Warrant requirement** | Judicial warrant for immigration detention |
| **Time limits** | No extended holds beyond sentence |

### Sanctuary Jurisdictions

| Issue | ICS Position |
|-------|--------------|
| **Local cooperation** | ICS does not require local enforcement cooperation |
| **Courthouse access** | No immigration enforcement in courthouses |
| **Sensitive locations** | Schools, hospitals, churches protected |
| **Victim/witness protection** | No enforcement during victim/witness cooperation |

## Specific Offense Categories

### Driving Offenses

| Offense | First Offense | Repeat |
|---------|---------------|--------|
| **DUI** | Tier freeze; treatment | CLP review |
| **Reckless driving** | Warning | Tier freeze |
| **Driving without license** | Assistance to obtain | Tier freeze if repeated |
| **Hit and run** | CLP review if injury | Likely revocation |

### Drug Offenses

| Offense | Effect | Notes |
|---------|--------|-------|
| **Personal possession** | Tier freeze; diversion available | Treatment-focused |
| **Possession with intent** | CLP review | Depends on quantity/circumstances |
| **Distribution** | Revocation likely | Federal nexus possible |
| **Trafficking** | Automatic revocation | Aggravated felony |

### Theft Offenses

| Offense | Effect |
|---------|--------|
| **Petty theft (<$500)** | Warning; tier freeze if repeated |
| **Theft ($500-$10,000)** | Tier freeze; CLP review |
| **Grand theft (>$10,000)** | CLP review; likely revocation |
| **Theft with force** | Revocation (aggravated felony if robbery) |

### Assault Offenses

| Offense | Effect |
|---------|--------|
| **Simple assault** | Tier freeze |
| **Assault with injury** | CLP review |
| **Aggravated assault** | Revocation likely |
| **Assault with weapon** | Revocation |
| **Domestic violence** | Revocation likely |

### Fraud Offenses

| Offense | Effect |
|---------|--------|
| **Minor fraud (<$10,000)** | Tier freeze; restitution |
| **Major fraud (>$10,000)** | CLP review |
| **Immigration fraud** | Revocation |
| **Identity fraud** | CLP review; revocation if for immigration benefit |

## Incarceration and CLP Status

### During Incarceration

| Issue | Rule |
|-------|------|
| **CLP status** | Suspended during incarceration (not revoked pending review) |
| **Tier clock** | Tolled during incarceration |
| **Benefits** | Suspended during incarceration |
| **Family status** | Derivatives retain independent status |

### Post-Release

| Scenario | Outcome |
|----------|---------|
| **Minor offense** | CLP resumes; tier clock resumes |
| **Moderate offense** | CLP resumes with tier freeze; review |
| **Serious offense** | Revocation proceedings; removal possible |
| **Sentence served + revocation** | Removal proceedings after sentence |

### Reentry After Release

| Requirement | Standard |
|-------------|----------|
| **Reporting** | Report to ICS within 72 hours of release |
| **Conditions** | May include additional monitoring |
| **Housing** | Placement assistance if needed |
| **Employment** | Reemployment assistance |

## Juvenile Justice

### Juvenile Adjudications

| Outcome | CLP Effect |
|---------|------------|
| **Diversion** | No CLP consequence |
| **Delinquency finding (minor)** | No CLP consequence |
| **Delinquency finding (serious)** | Review; best interest considered |
| **Adult prosecution** | Treated as adult offense |

### Protections for Juvenile CLP Holders

| Protection | Standard |
|------------|----------|
| **Best interest** | Primary consideration |
| **Family unity** | Avoid separation if possible |
| **Rehabilitation focus** | Favor rehabilitation over removal |
| **Counsel** | Appointed in all proceedings |
| **Confidentiality** | Juvenile records protected |

## Victim and Witness Protections

### CLP Holders as Victims

| Protection | Standard |
|------------|----------|
| **Crime reporting** | No immigration consequence for reporting |
| **Victim services** | Full access to victim services |
| **Protective orders** | Available regardless of status |
| **U-visa eligibility** | Victims of qualifying crimes eligible |
| **T-visa eligibility** | Trafficking victims eligible |

### CLP Holders as Witnesses

| Protection | Standard |
|------------|----------|
| **Witness cooperation** | Protected from enforcement during cooperation |
| **Subpoena response** | Responding to subpoena is protected |
| **Retaliation** | Protected from status-based retaliation |
| **Continued presence** | May be granted for witness cooperation |

## Expungement and Pardons

### Effect of Expungement

| Type | CLP Effect |
|------|------------|
| **Full expungement** | Offense not considered for CLP purposes |
| **Partial/conditional** | Depends on type; generally still considered |
| **Deferred adjudication completion** | Not a conviction for CLP |
| **Diversion completion** | Not a conviction for CLP |

### Effect of Pardons

| Type | CLP Effect |
|------|------------|
| **Full pardon** | Offense not considered for CLP |
| **Commutation** | Sentence reduced; conviction remains |
| **Clemency** | Depends on scope |

### Vacatur

| Type | CLP Effect |
|------|------------|
| **Vacated for legal error** | Conviction eliminated for CLP |
| **Vacated for immigration purposes only** | Generally not recognized |
| **Plea withdrawal** | Case returns to pending status |

## Appeals and Relief

### Challenging CLP Consequences

| Step | Forum |
|------|-------|
| **Initial review** | IIOO case review |
| **Administrative hearing** | IIOO Administrative Judge |
| **Appeals Board** | IIOO Appeals Board |
| **Federal court** | Petition for review |

### Grounds for Relief

| Ground | Standard |
|--------|----------|
| **Ineffective assistance** | Counsel failed to advise of consequences |
| **Factual innocence** | Evidence of innocence |
| **Procedural error** | Due process violation |
| **Disproportionality** | Consequence disproportionate to offense |
| **Rehabilitation** | Demonstrated rehabilitation |

### Waivers

| Waiver Type | Eligibility |
|-------------|-------------|
| **Hardship waiver** | Extreme hardship to USC/LPR family |
| **Rehabilitation waiver** | Extended rehabilitation; community ties |
| **Minor offense waiver** | Offense minor; long residence |
| **Youthful offender** | Offense committed as juvenile/young adult |
