# Immigration Corridor System: Language Services

---

## Overview

English language acquisition is a core ICS benchmark for tier advancement. This document details ESL program standards, interpreter services, CEFR benchmarks, curriculum requirements, and language access policies throughout the corridor system.

---

## Language Benchmarks

### CEFR Standards

| Level | Description | ICS Application |
|-------|-------------|-----------------|
| **A1** | Breakthrough | Basic survival English |
| **A2** | Waystage | Elementary proficiency |
| **B1** | Threshold | **Required for Tier C advancement** |
| **B2** | Vantage | Professional working proficiency |
| **C1** | Effective operational | Advanced fluency |
| **C2** | Mastery | Near-native proficiency |

### Tier Requirements

| Tier | Language Requirement | Timeline |
|------|---------------------|----------|
| **Tier A** | Enrollment in ESL; A1 progress | 6-12 months |
| **Tier B** | A2 achievement; B1 progress | 12-24 months |
| **Tier C** | B1 achievement | Required for advancement |
| **Citizenship** | B1+ (civics literacy) | Additional requirement |

### Assessment Methods

| Assessment | Use |
|------------|-----|
| **Initial placement** | Determine starting level at intake |
| **Progress monitoring** | Quarterly assessments |
| **B1 certification** | Third-party validated test for Tier C |
| **Alternative demonstration** | Portfolio, interview for special cases |

## ESL Program Structure

### Program Models

| Model | Setting | Hours/Week | Target Population |
|-------|---------|------------|-------------------|
| **Intensive** | Classroom | 20-30 | Full-time learners |
| **Standard** | Classroom | 10-15 | Working adults |

| **Civic content** | US civics integrated |
| **Workplace English** | Sector-specific modules |

| **Financial literacy** | Banking, taxes, budgeting |

### Instructional Hours

|------------------|---------------|
| **0 → A1** | 100-150 hours |

### Instructor Pipeline and Credentialing

| Pipeline Element | Implementation |
|------------------|----------------|

## Special Populations

|-----------|---------------|
| **No L1 literacy** | Literacy-level ESL classes |
| **Limited schooling** | Extended timeline; additional support |
| **Learning disabilities** | Screening; accommodations |

### Elderly Learners

| Accommodation | Provision |
|---------------|-----------|
| **Pace** | Slower progression acceptable |
| **Modality** | In-person preferred; smaller classes |
| **Exemption** | Case-by-case for advancement |
