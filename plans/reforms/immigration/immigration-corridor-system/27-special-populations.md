# Immigration Corridor System: Special Populations

---

## Overview

ICS interfaces with several pre-existing immigration categories and special populations. This document details how refugees, Special Immigrant Visa holders, crime victims (U/T visas), and other protected populations interact with the ICS framework, ensuring their protections are preserved while providing integration support.

---

## Refugees

### Refugee Status vs. CLP

| Feature | Refugee Status | CLP |
|---------|----------------|-----|
| **Basis** | Persecution finding overseas | Entry via ICS |
| **Processing** | UNHCR/overseas | Domestic |
| **Status** | Refugee (8 U.S.C. § 1157) | CLP (new status) |
| **Benefits** | Full refugee benefits | ICS benefits |
| **Path to PR** | 1 year after admission | Tier progression |
| **Family petition** | Yes (I-730) | After PR only |

### Refugees and ICS Services

| Interaction | Policy |
|-------------|--------|
| **ICS services access** | Refugees can access ICS-funded services |
| **Hub city placement** | Refugees may resettle in hub cities |
| **RID eligibility** | Communities hosting refugees qualify for RID |
| **Secondary migration** | Refugees can move to corridor cities |

### Refugee Resettlement Coordination

| Agency | Role |
|--------|------|
| **ORR** | Initial resettlement; benefits |
| **Resettlement agencies** | Case management; services |
| **ICS** | Supplemental services; community coordination |
| **State refugee coordinators** | State-level coordination |

## U-Visa (Crime Victims)

### U-Visa Eligibility

| Requirement | Standard |
|-------------|----------|
| **Qualifying crime** | Domestic violence, trafficking, sexual assault, etc. |
| **Substantial harm** | Physical or mental abuse |
| **Information possession** | Knows about criminal activity |
| **Helpfulness** | Is/has been/likely to be helpful |
| **Law enforcement certification** | Signed certification |

### U-Visa and CLP Interaction

| Scenario | Treatment |
|----------|-----------|
| **CLP holder becomes crime victim** | Can apply for U-visa; better status |
| **U-visa holder in ICS community** | Full ICS service access |
| **U-visa applicant pending** | CLP can provide interim status |
| **U-visa denial** | CLP status if otherwise eligible |

### U-Visa Pathway Comparison

| Feature | U-Visa | CLP |
|---------|--------|-----|
| **Cap** | 10,000/year | Capacity-based |
| **Processing time** | 4-5+ years | 30-90 days |
| **Derivative family** | Yes (U-2, U-3, etc.) | Yes |
| **Path to PR** | 3 years; no cap | Tier progression |
| **Work authorization** | Upon approval | Immediate |

### ICS Support for U-Visa Applicants

| Service | Availability |
|---------|--------------|
| **Legal assistance** | ICS-funded legal aid |
| **Victim services** | Access to ICS healthcare |
| **Housing** | Priority for safe housing |
| **Confidentiality** | Enhanced privacy protections |

## T-Visa (Trafficking Victims)

### T-Visa Eligibility

| Requirement | Standard |
|-------------|----------|
| **Trafficking victim** | Severe form of trafficking |
| **Physical presence** | Present in US due to trafficking |
| **Compliance** | Comply with reasonable law enforcement requests |
| **Hardship** | Extreme hardship upon removal |

### T-Visa and CLP Interaction

| Scenario | Treatment |
|----------|-----------|
| **Trafficking victim in CLP** | Transition to T-visa if eligible |
| **T-visa holder in ICS area** | Full ICS service access |
| **Certification pending** | CLP provides interim protection |
| **Minor victim** | Enhanced services; no compliance requirement |

### Trafficking Victim Services

| Service | Access |
|---------|--------|
| **ORR trafficking certification** | Enables benefits |
| **Housing** | Safe housing priority |
| **Healthcare** | Trauma-informed care |
| **Legal services** | ICS-funded representation |
| **Employment** | Protected employment pathway |

### VAWA Eligibility

| Category | Eligibility |
|----------|-------------|
| **Abused spouse** | Spouse of USC/LPR |
| **Abused child** | Child of USC/LPR |
| **Abused parent** | Parent of abusive USC (21+) |

### VAWA and CLP Interaction

| Scenario | Treatment |
|----------|-----------|
| **CLP holder as victim** | VAWA provides better path if spouse is USC/LPR |
| **VAWA applicant pending** | CLP provides interim status |
| **VAWA + CLP family** | Mixed-status protections apply |
| **Abuser in CLP** | Enforcement; derivative revocation |

### VAWA-Specific Protections in ICS

| Protection | Standard |
|------------|----------|
| **Confidentiality** | Abuser cannot access victim's ICS records |
| **Separate processing** | Victim processed separately |
| **Safe housing** | Abuser-free placement |
| **Economic independence** | Employment support |

## TPS Holders

### TPS and CLP Comparison

| Feature | TPS | CLP |
|---------|-----|-----|
| **Basis** | Country conditions | Integration system |
| **Duration** | Temporary; renewable | Progression to PR |
| **Geographic limits** | None | Tiered |
| **Work authorization** | Yes | Yes |
| **Path to PR** | None (standalone) | Yes |

### TPS Holder Options

| Scenario | Option |
|----------|--------|
| **TPS designated** | Maintain TPS or transition to CLP |
| **TPS terminated** | Register for CLP within grace period |
| **TPS + long residence** | Legacy track may be faster |
| **TPS → CLP** | Preserves lawful presence history |

### TPS Transition to CLP

| Step | Process |
|------|---------|
| **Registration** | File for CLP during TPS validity |
| **Status** | CLP issued; TPS terminates |
| **Tier placement** | Based on integration factors |
| **Timeline credit** | TPS time may count toward residence |

## DACA Recipients

### DACA and Legacy Track

| Feature | DACA | Legacy Track |
|---------|------|--------------|
| **Eligibility age** | Under 31 as of 2012 | Any age |
| **Arrival date** | Before June 2007 | 5+ years residence |
| **Education/military** | Required | Not required |
| **Criminal bars** | Similar | Similar |

### DACA Transition to CLP

| Scenario | Pathway |
|----------|---------|
| **DACA active** | Register for legacy track |
| **DACA expired** | Register for legacy track |
| **DACA denied** | May qualify for legacy or standard CLP |
| **DACA → CLP** | Tier C placement likely (established) |

### DACA-Specific Considerations

| Factor | Treatment |
|--------|-----------|
| **Work history** | Counts toward integration |
| **Tax records** | Evidence of compliance |
| **Education** | Counts toward benchmarks |
| **Geographic ties** | May receive geographic waiver |

### Parole to CLP Transition

| Step | Process |
|------|---------|

| **Eligibility** | Must meet general CLP requirements |
| **Status** | CLP replaces parole |
| **Tier placement** | Based on integration factors |
