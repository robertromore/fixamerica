# Immigration Corridor System: Family Unity

---

## Overview

Family unity is a core ICS principle. This document details how families are kept together, how derivative status works, how mixed-status families are handled, and how family reunification proceeds within the corridor framework.

---

## Core Principles

| Principle | Implementation |
|-----------|----------------|
| **No family separation** | ICS does not separate families except for safety |
| **Derivative status** | Spouse and minor children receive CLP with principal |
| **Joint placement** | Families placed together in same hub/corridor |
| **Coordinated advancement** | Family advances together through tiers |
| **Mixed-status protection** | Citizens/LPRs in family units protected |

---

## Derivative CLP Status

### Eligible Derivatives

| Relationship | Eligibility | Documentation |
|--------------|-------------|---------------|
| **Spouse** | Yes | Marriage certificate or equivalent |
| **Minor children (under 21)** | Yes | Birth certificate or equivalent |
| **Step-children** | Yes (if marriage before child turned 18) | Marriage + birth certificates |
| **Adopted children** | Yes | Adoption decree |
| **Parents** | No (unless UAC sponsor) | N/A |
| **Siblings** | No | N/A |
| **Extended family** | No | N/A |

### Derivative Status Rules

| Rule | Description |
|------|-------------|
| **Concurrent processing** | Derivatives processed with principal |
| **Same tier** | Derivatives in same tier as principal |
| **Joint advancement** | Family advances together |
| **Independent eligibility** | Derivatives may later qualify independently |
| **Age-out protection** | Children who turn 21 retain derivative status |

## Family Intake Process

### At Entry

| Step | Action |
|------|--------|
| **1. Family unit identification** | Confirm relationships |
| **2. Biometrics** | All family members |
| **3. Principal designation** | Usually primary applicant |
| **4. Derivative documentation** | Issue CLP to all members |
| **5. Joint placement** | Single placement decision for family |

### Relationship Verification

| Method | Use Case |
|--------|----------|
| **Documents** | Primary method; birth/marriage certificates |
| **DNA testing** | When documents unavailable; voluntary |
| **Interviews** | Supplemental verification |
| **Affidavits** | Supporting evidence |

### Fraudulent Family Claims

| Finding | Consequence |
|---------|-------------|
| **Fraudulent marriage** | Principal and claimed spouse reviewed separately |
| **Fraudulent parentage** | Child protection protocols activated |
| **Human trafficking indicators** | Immediate separation; investigation |
| **Smuggling debt coercion** | Victim services; no penalty for victim |

## Mixed-Status Families

### Definition

Mixed-status families include members with different immigration statuses:

- US citizens
- Lawful permanent residents
- CLP holders
- Other visa holders
- Undocumented family members

### Treatment Under ICS

| Family Member | ICS Treatment |
|---------------|---------------|
| **US citizen spouse** | Not in CLP; benefits from family placement |
| **US citizen children** | Not in CLP; remain with family |
| **LPR spouse** | Not in CLP; can sponsor principal for PR |
| **CLP principal** | Standard CLP progression |
| **Undocumented family member** | Eligible for Legacy Track or new CLP |

### Protections for Mixed-Status Families

| Protection | Description |
|------------|-------------|
| **No enforcement targeting** | Families not targeted based on CLP member |
| **Benefit coordination** | Citizens/LPRs retain full benefits |
| **Housing assistance** | Mixed household rules apply |
| **School enrollment** | All children enrolled regardless of status |

## Placement and Housing

### Family Placement Algorithm

| Factor | Weight |
|--------|--------|
| **Family size** | Housing unit matching |
| **School-age children** | School capacity consideration |
| **Special needs** | Medical/disability services |
| **Extended family** | Preference for proximity (not required) |

### Housing Standards for Families

| Standard | Requirement |
|----------|-------------|
| **Bedrooms** | Based on family composition |

| **Stability** | Minimum lease terms for families |

## Family Separation Circumstances

ICS does not separate families except when required for:

### Safety-Based Separation

| Circumstance | Protocol |
|--------------|----------|
| **Domestic violence** | Victim protection; separate placement available |
| **Child abuse/neglect** | Child welfare intervention |
| **Human trafficking** | Victim separation from trafficker |
| **Serious criminal conduct** | Standard justice system processing |

### Voluntary Separation

| Circumstance | Treatment |
|--------------|-----------|
| **Adult child establishing independence** | Separate CLP if eligible |
| **Divorce during CLP** | Separate placements; both retain status |
| **Family member choosing to leave US** | Voluntary departure provisions |

### Reunification Priority

| Situation | Response |
|-----------|----------|
| **Separated at entry** | Expedited reunification processing |
| **Family member detained** | Case review for release |
| **Family member deported** | Humanitarian review possible |

### Pregnant Women in ICS

| Provision | Description |
|-----------|-------------|
| **Prenatal care** | Covered under ICS healthcare |
| **Delivery** | Hospital access; Medicaid emergency coverage |
| **Postpartum care** | Continued coverage |
| **Placement stability** | No transfer during late pregnancy |

### Children Born in US

| Status | Treatment |
|--------|-----------|
| **US citizen at birth** | 14th Amendment citizenship |
| **Benefits** | Full citizen benefits |
| **Family unity** | Remains with CLP parents |
| **Future sponsorship** | Can sponsor parents at age 21 |

## Family-Based Immigration Interaction

### CLP Holders Cannot Sponsor

| Category | CLP Ability |
|----------|-------------|
| **Spouse** | No (requires PR or citizenship) |
| **Children** | No (requires PR or citizenship) |
| **Parents** | No (requires citizenship) |
| **Siblings** | No (requires citizenship) |

### CLP Holders Can Be Sponsored

| Sponsor | Petition Type |
|---------|---------------|
| **US citizen spouse** | I-130 immediate relative |
| **US citizen parent** | I-130 immediate relative |
| **US citizen sibling** | I-130 (long wait) |
| **LPR spouse** | I-130 (preference category) |
| **Employer** | Employment-based (if eligible) |

### Pending Petitions

| Situation | ICS Treatment |
|-----------|---------------|
| **Petition already filed** | CLP does not affect petition |
| **Priority date current** | Can adjust status or consular process |
| **CLP during wait** | Provides lawful status while waiting |

## Tier Advancement for Families

### Joint Advancement Rules

| Rule | Description |
|------|-------------|
| **Family unit** | Advances together |
| **Slowest member** | Family advances when all meet criteria |
| **Hardship exception** | Individual advancement if separation required |
| **Children's benchmarks** | Age-appropriate (school attendance vs. employment) |

### Benchmark Adjustments for Families

| Family Situation | Adjustment |
|------------------|------------|
| **Primary caregiver** | Employment requirement reduced |
| **Single parent** | Additional support services |
| **Large family** | Housing assistance increased |
| **Special needs member** | Individualized benchmarks |

## Elderly Family Members

### Grandparents and Extended Family

| Situation | Treatment |
|-----------|-----------|
| **Grandparent as caregiver** | Eligible for CLP if meeting criteria |
| **Grandparent joining family** | Not derivative; must qualify independently |

| **Housing** | Accessible units when needed |
| **Social services** | Aging services through corridor programs |
