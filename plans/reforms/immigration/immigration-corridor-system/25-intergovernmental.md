# Immigration Corridor System: Intergovernmental Coordination

---

## Overview

ICS requires unprecedented coordination among federal agencies, states, localities, tribal nations, and international partners. This document details the coordination mechanisms, governance structures, funding relationships, and dispute resolution processes that enable effective multi-level implementation.

## Federal-State Relations

### State Role in ICS

| Function | State Responsibility |
|----------|---------------------|
| **Service delivery** | Many ICS services delivered through state agencies |
| **Compact participation** | Voluntary interstate compact membership |
| **Supplementation** | States may add to federal minimums |
| **Licensing** | Professional licensing; driver's licenses |

### Interstate Compact

| Feature | Details |
|---------|---------|
| **Purpose** | Coordinate ICS implementation across state lines |
| **Governance** | Compact commission with state representatives |
| **Obligations** | Benefit portability; transfer coordination |
| **Funding** | Cost-sharing formula |
| **Withdrawal** | 180-day notice; orderly transition |

## Tribal Consultation

### Government-to-Government Relationship

| Principle | Application |
|-----------|-------------|
| **Sovereignty** | Tribes as sovereign nations |
| **Consultation** | Meaningful consultation before decisions affecting tribes |
| **Free prior informed consent** | For actions on tribal lands |
| **Self-determination** | Tribal control over tribal programs |

### Tribal Participation Options

| Option | Description |
|--------|-------------|
| **Full participation** | Tribes operate ICS programs on reservations |
| **Partial participation** | Federal operation with tribal input |
| **Non-participation** | ICS does not operate on reservation |
| **Compact/self-governance** | Tribal administration with federal funding |

### Tribal-Specific Considerations

| Issue | Approach |
|-------|----------|
| **Jurisdiction** | Respect tribal criminal/civil jurisdiction |
| **Cultural preservation** | No impact on tribal cultural practices |
| **Economic development** | Align with tribal economic priorities |
| **Sacred sites** | Avoid impact on sacred sites |
| **Environmental protection** | Comply with tribal environmental law |

### Consultation Process

| Stage | Action |
|-------|--------|
| **Early notification** | 90 days before decisions affecting tribes |
| **Consultation meeting** | Meaningful dialogue with tribal leaders |
| **Response period** | 60 days for tribal input |
| **Accommodation** | Address tribal concerns |
| **Documentation** | Written record of consultation |

## Legal Framework

### Preemption

| Issue | Federal Position |
|-------|------------------|
| **Status determination** | Exclusive federal authority |
| **Enforcement priorities** | Federal priorities prevail |
| **Supplementation** | States may supplement, not contradict |
| **Local immigration laws** | Preempted if conflict |

### Anti-Commandeering

| Limit | Accommodation |
|-------|---------------|
| **Cannot compel state action** | Voluntary participation with incentives |
| **Cannot require legislation** | Federal alternatives if states decline |
| **Conditional spending** | Conditions tied to program participation |

### Federal-State Immunity

| Issue | Resolution |
|-------|------------|
| **State immunity** | States can waive via compact |
| **Federal immunity** | Specific waivers for program |
| **Official immunity** | Qualified immunity for officials |
