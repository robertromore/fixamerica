# Immigration Corridor System: International Operations

---

## Overview

ICS operates primarily within US borders, but effective implementation requires coordination with source countries, international organizations, and foreign governments. This document details bilateral relations, in-country processing options, repatriation protocols, and international legal frameworks that support ICS operations.

## In-Country Processing

### Overseas Registration Centers

| Location | Services | Capacity |
|----------|----------|----------|
| **Guatemala City** | Full CLP application | 500/day |
| **San Pedro Sula** | Full CLP application | 300/day |
| **San Salvador** | Full CLP application | 300/day |
| **Mexico City** | Coordination; referrals | 200/day |
| **Port-au-Prince** | Limited services | 100/day |

### In-Country Application Process

| Step | Timeline | Location |
|------|----------|----------|
| **Pre-registration** | Online | Home country |
| **Documentation** | 1-2 weeks | Local center |
| **Biometrics** | 1 day | Registration center |
| **Interview** | 1 day | Registration center |
| **Background check** | 2-4 weeks | US systems |
| **Conditional approval** | Within 60 days | Notification |
| **Travel authorization** | Upon approval | Electronic |
| **Port of entry processing** | 1 day | US POE |
| **Hub placement** | Within 72 hours | Assigned hub |

### Benefits of In-Country Processing

| Benefit | Description |
|---------|-------------|
| **Reduced irregular migration** | Legal pathway reduces incentive for illegal crossing |
| **Orderly flow** | Scheduled arrivals match hub capacity |
| **Better screening** | More time for background checks |
| **Reduced smuggling** | Legal option undercuts smuggler business model |
| **Safer migration** | No dangerous border crossing |
| **Cost savings** | Cheaper than border enforcement |

## Repatriation Protocols

### Voluntary Departure

| Element | Standard |
|---------|----------|
| **Eligibility** | CLP holders choosing to leave |
| **Travel assistance** | Airfare provided |
| **Departure timeline** | Within 60 days of decision |
| **Re-entry eligibility** | Preserved for future application |
| **Belongings** | Assistance with shipping |

### Ordered Removal

| Step | Timeline | Process |
|------|----------|---------|
| **Final order** | Day 0 | Exhaustion of appeals |
| **Travel document request** | Day 1-30 | Consular coordination |
| **Removal scheduling** | Day 30-60 | ICE coordination |
| **Pre-removal custody** | If flight risk | Alternatives to detention preferred |
| **Removal flight** | Day 60-90 | Commercial or charter |
| **Reception** | Upon arrival | Country coordination |

### Travel Document Issuance

| Scenario | Process | Timeline |
|----------|---------|----------|
| **Valid passport** | Use existing | Immediate |
| **Expired passport** | Consular renewal | 2-4 weeks |
| **No passport** | Emergency travel document | 4-8 weeks |
| **Stateless** | UNHCR coordination | Variable |
| **Refused issuance** | Diplomatic engagement | Variable |

### Reception Protocols

| Element | Standard |
|---------|----------|
| **Airport reception** | Immigration authority coordination |
| **Documentation** | Removal paperwork provided |
| **Belongings** | Personal items returned |
| **Cash assistance** | Reintegration grant ($500-1,000) |
| **Referrals** | IOM reintegration programs |

### Countries Refusing Repatriation

| Situation | Response |
|-----------|----------|
| **Temporary refusal** | Diplomatic engagement |
| **Persistent refusal** | Visa sanctions consideration |
| **Statelessness** | Alternative arrangements |
| **Safety concerns** | Withholding/CAT protection |

## International Legal Framework

### Treaty Obligations

| Treaty | US Status | ICS Compliance |
|--------|-----------|----------------|
| **1951 Refugee Convention** | Party to Protocol | Asylum provisions preserved |
| **Convention Against Torture** | Ratified | CAT protection in removal |
| **ICCPR** | Ratified | Due process standards |
| **Vienna Convention (Consular)** | Ratified | Notification protocols |
| **Trafficking Protocol** | Ratified | T-visa; victim protection |

### Non-Refoulement

| Principle | Application |
|-----------|-------------|
| **Asylum seekers** | Full asylum process before removal |
| **CAT claims** | Independent review before removal |
| **Pattern countries** | Enhanced scrutiny for returns |
| **Individual assessment** | Case-by-case danger evaluation |

### International Human Rights Standards

| Standard | ICS Compliance |
|----------|----------------|
| **Prohibition of arbitrary detention** | Alternatives to detention; judicial review |
| **Family unity** | Derivative status; reunification |
| **Non-discrimination** | Objective criteria; no national origin preferences |
| **Due process** | Administrative hearings; appeals |
| **Humane treatment** | Detention standards; healthcare access |
