## Selection Criteria

Policy applications are prioritized by:

1. **Public salience** - Issues people care deeply about
2. **Constitutional demonstration value** - How well it shows framework features
3. **Current dysfunction severity** - How broken the current system is
4. **Clarity of regional fit** - How obviously regional scale helps
